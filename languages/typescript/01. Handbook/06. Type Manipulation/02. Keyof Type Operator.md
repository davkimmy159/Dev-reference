`Keyof` <sub>(타입 연산자)</sub>
===================

##### 객체 타입 전달
- 키 값 해당 문자열 · 숫자 상수 `union` <sub>(타입)</sub>
```ts
type Point = {
  x: number;
  y: number;
};

// type P: "x" | "y"
type P = keyof Point;
```

##### index signature <sub>(`string` · `number`)</sub>
```ts
type Arrayish = {
  [n: number]: unknown
};

// type A = number
type A = keyof Arrayish;

type Mapish = {
  [k: string]: boolean
};

// type M = string | number
type M = keyof Mapish;
```
