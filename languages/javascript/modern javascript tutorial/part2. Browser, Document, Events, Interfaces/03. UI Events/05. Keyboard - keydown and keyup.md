```javascript
// script.js
kinput.onkeydown = kinput.onkeyup = kinput.onkeypress = handle;

let lastTime = Date.now();

function handle(e) {
  if (form.elements[e.type + 'Ignore'].checked) {
    return;
  }

  let text = e.type +
    ' key=' + e.key +
    ' code=' + e.code +
    (e.shiftKey ? ' shiftKey' : '') +
    (e.ctrlKey ? ' ctrlKey' : '') +
    (e.altKey ? ' altKey' : '') +
    (e.metaKey ? ' metaKey' : '') +
    (e.repeat ? ' (repeat)' : '') +
    "\n";

  if (area.value && Date.now() - lastTime > 250) {
    area.value += new Array(81).join('-') + '\n';
  }
  lastTime = Date.now();

  area.value += text;

  if (form.elements[e.type + 'Stop'].checked) {
    e.preventDefault();
  }
}
```

### `key[down·up]` <sub>(이벤트)</sub>

##### 키보드 동작 관련
- 키 누를 때 · 뗄 때 발생

#### `key · code` <sub>(이벤트 객체 프로퍼티)</sub>

##### `key`
- 눌린 키 문자
- 동일 키
  - 상이 값 가능

##### `code`
- 눌린 키 코드
- 동일 키
  - 동일 값

|키|`key`|`code`|
|:---:|---|---|
|z|`z` <sub>(소문자)</sub>|`KeyZ`|
|shift + z|`Z` <sub>(대문자)</sub>|`KeyZ`|

##### 언어 변경 <sub>(영어 → 한국어 등)</sub> 시 차이

|언어|키|`key`|`code`|
|:---:|:---:|---|---|
|영어|z|`z` <sub>(소문자)</sub>|`KeyZ`|
|한글|z|`ㅋ` <sub>(한글)</sub>|`KeyZ`|

##### 특수 키
- 보통 `key` == `code`

|키|`key`|`code`|
|:---:|---|---|
|F1|`"F1"`|`"F1"`|
|backspace|`"Backspace"`|`"Backspace"`|
|shift|`"Shift"`|`"Shift[Right·Left]"`|

### `key` vs `code`

#### `key`

##### 키 의미
- ex\) `"Shift"`

##### 동일 키
- 언어 체계 따라 값 상이

#### `code`

##### 정확히 눌린 키
- ex\) `"Shift[Right·Left]"`

##### 동일 키
- 키보드 레이아웃 따라 값 상이

#### `code` 특징

##### 값 항상 동일
- `keydown` 핸들러 내 사용 용이
- `key`
  - 언어 체계 따라 동일 키 값 상이
```javascript
document.addEventListener('keydown', function(event) {
  if (event.code == 'KeyZ' && (event.ctrlKey || event.metaKey)) {
    alert('Undo!')
  }
});
```

##### 다양한 키보드 레이아웃
- 동일 키 → 타 문자

#### 미국 <sub>("QWERTY")</sub> vs 독일 <sub>("QWERTZ")</sub>

##### 미국 <sub>("QWERTY")</sub>
- Z 키 == `"KeyZ"`

![us-layout](../../images/02/03/05/us-layout.svg)

##### 독일 <sub>("QWERTZ")</sub>
- Y 키 == `"KeyZ"`

![german-layout](../../images/02/03/05/german-layout.svg)

##### `event.code == "KeyZ"`
- 미국 <sub>("QWERTY")</sub>
  - Z 키
- 독일 <sub>("QWERTZ")</sub>
  - Y 키

##### [명세](https://www.w3.org/TR/uievents-code/#table-key-code-alphanumeric-writing-system)
- 키보드 레이아웃 문제 설명
  - 목록 수록
- 일부 키만 해당
  - `"Key[A·Q·Z]"`
- 특수 키 미해당
  - Shift 키
  - 기타 등등

##### `key`
- 키 문자 추적 시 용이
  - 레이아웃 종류 무관

##### `code`
- 항상 동일 <sub>(실제 키보드 상 키 위치 기준)</sub>
  - 핫키 정상 동작 <sub>(사용 언어 무관)</sub>

### 자동 반복

##### 충분히 긴 시간 동안 키 계속 누르면 발생
- `keydown` <sub>(이벤트)</sub>
  - 계속 발생
- `keyup` <sub>(이벤트)</sub>
  - 키 떼면 한 번 발생
- `event.repeat` <sub>(프로퍼티)</sub>
  - `true`

##### 일반적인 키보드 이벤트
- `keydown` <sub>(이벤트)</sub>
  - 다수 발생
- `keyup` <sub>(이벤트)</sub>
  - 한 번 발생

##### 다양한 기본 동작 有
- 화면에 문자 출력 <sub>(가장 기본적인 동작)</sub>
- 문자 삭제
  - Delete 키
- 페이지 스크롤
  - PageDown 키
- '페이지 저장' 대화상자 열기
  - Ctrl + S 키

##### 기본 동작 막기
- 다양한 기본 동작 전부 비활성화
- OS 기반 특수 키 제외
  - ex\) Alt + F4 <sub>(Windows)</sub>
    - 현재 브라우저 창 닫기

##### `<input>` <sub>(필드)</sub> 허용 문자 제한
- 숫자 키
- `'+'`
- `'-'`
- `'('`
- `')'`
```html
<script>
function checkPhoneKey(key) {
  return ('0' <= key && key <= '9')
    || key == '+'
    || key == '-';
    || key == '('
    || key == ')'
}
</script>
<input onkeydown="return checkPhoneKey(event.key)" placeholder="Phone, please" type="tel">
```

### 핫키 확장

##### `runOnKeys(f, c1, c2, …, cn)` <sub>(함수)</sub> 작성
- `c1`, `c2`, …, `cn` 키 동시 누를 때
  - `f` <sub>(인수 함수)</sub> 실행
```javascript
/*
 "Q" · "W" 키 동시 누를 때
 - alert (함수) 실행
   - 언어 무관
   - 대 · 소문자 구분 X
 */
runOnKeys(
  () => alert("Hello!"),
  "KeyQ",
  "KeyW"
);
```
