변화 감지
====

##### `MutationObserver` <sub>(내장 객체)</sub>
- DOM 요소 감시
  - 변화 감지 시 콜백 호출

### 문법

##### 객체 생성
- 콜백 <sub>(인수)</sub> 전달
```javascript
let observer = new MutationObserver(callback);
```

##### DOM 노드 설정
```javascript
observer.observe(node, config);
```

##### `config` <sub>(`boolean` 옵션 객체 · 감시 타입 설정)</sub>
- `childList`
  - 자식 노드
- `subtree`
  - 자손 노드
- `attributes`
  - 속성
- `attributeFilter` <sub>(속성명 배열)</sub>
  - 일부 속성
- `characterData`
  - `data` <sub>(프로퍼티)</sub>
    - 텍스트 내용
- `attributeOldValue`
  - 이전 속성값 콜백 전달 여부
  - `attributes` 설정 필요
- `characterDataOldValue`
  - 이전 `data` <sub>(프로퍼티)</sub> 값 콜백 전달 여부
  - `characterData` 설정 필요

##### 요소 내 변화 발생 시 콜백 <sub>(인수)</sub> 실행
- 1번째 인수
  - [`MutationRecord`](https://dom.spec.whatwg.org/#mutationrecord) <sub>(객체)</sub> 목록
    - 변화 내용
- 2번째 인수
  - `MutationObserver` <sub>(객체)</sub>
    - 자기 자신

#### [`MutationRecord`](https://dom.spec.whatwg.org/#mutationrecord) <sub>(객체)</sub> 프로퍼티

##### `type` <sub>(변화 타입 · 문자열)</sub>
- `'attributes'`
  - 속성 변화
- `'characterData'`
  - `data` <sub>(프로퍼티)</sub> 변화 <sub>(텍스트 노드)</sub>
- `'childList'`
  - 자식 노드 추가 · 제거

##### `target` <sub>(변화 발생 위치 · 문자열)</sub>
- 요소
  - `'attributes'`
  - `'childList'`
- 텍스트 노드
  - `'characterData'`

##### `[added·removed]Nodes`
- 추가 · 제거된 노드

##### `[previous·next]Sibling`
- 이전 · 다음 형제 노드
  - `[added·removed]Nodes` 기준

##### `attributeName[space]`
- 변화 속성 이름 · 네임스페이스 <sub>(XML)</sub>

##### `oldValue` <sub>(이전 값)</sub>
- 속성 <sub>(`attributeOldValue` 설정 시)</sub>
- 텍스트 <sub>(`characterDataOldValue` 설정 시)</sub>

##### `<div contentEditable>` <sub>(요소)</sub>
- 요소 포커스 설정 · 내용 수정 가능
```html
<div contentEditable id="elem">Click and <b>edit</b>, please</div>

<script>
  let observer = new MutationObserver(mutationRecords => {

    mutationRecords; // 변화 출력
  });

  // 속성 외 전부 감시
  observer.observe(elem, {
    childList: true, // 자식 노드
    subtree: true,   // 자손 노드

    // 이전 data (프로퍼티) 값 (텍스트) 콜백 전달
    characterDataOldValue: true
  });
</script>
```

##### 요소 내 텍스트 1개 입력
- 변화 1개 콘솔 출력
```javascript
mutationRecords = [
  {
    type: "characterData",
    oldValue: "edit",
    target: <text node>,
    …나머지 빈 프로퍼티…
  }
];
```

##### 요소 내 복잡한 변경 적용
- 요소 내 텍스트 <sub>(`<b>edit</b>`)</sub> 제거
  - 다수 변화 콘솔 출력
```javascript
mutationRecords = [
  {
    type: "childList",
    target: <div#elem>,
    removedNodes: [<b>],
    nextSibling: <text node>,
    previousSibling: <text node>,
    …나머지 빈 프로퍼티…
  }, {
    type: "characterData"
    target: <text node>
    /* 변화 상세 정보 브라우저별 상이
     ex) 두 근접 텍스트 노드 ("edit" · "please")
     - 결합
     - 분리 유지
     */
  }
];
```

### 아키텍쳐 내 사용

#### 프로그래밍 웹사이트

##### 글 · 기타 구성 요소
- 소스코드 조각 포함 가능

##### HTML 마크업 내 코드 조각
```html
…
<pre class="language-javascript">
  <code>
    // here's the code
    let hello = "world";
  </code>
</pre>
…
```

#### [Prism.js](https://prismjs.com/) <sub>(문법 강조 라이브러리)</sub> 적용 <sub>(가독성 ↑)</sub>

##### `Prism.highlightElem(pre)`
- 동작
  1. `pre` <sub>(요소)</sub> 내용 검사
  2. 특수 태그 · 스타일 추가
     - 문법 강조 <sub>(색상)</sub>
- 권장 실행 시점
  - `DOMContentLoaded` <sub>(이벤트)</sub>
  - 페이지 최하단

##### DOM 준비 완료 시
1. `pre[class*="language"]` <sub>(요소)</sub> 검색
2. `Prism.highlightElem(pre)` 호출
```javascript
// 페이지 내 코드 조각 강조
document
  .querySelectorAll('pre[class*="language"]')
  .forEach(Prism.highlightElem);
```

##### 동적 코드 조각 강조
- 서버 요청 시 내용 <sub>(코드 조각 포함)</sub> 수신
  - 매번 별도 강조 코드 작성 <sub>(비효율적)</sub>
- 서드파티 모듈 내 코드 조각
  - 스크립트 수정 불가능
```javascript
// 코드 조각 포함 내용
// - Prism.highlightElem (문법 강조) 호출 필요
let article = /* 서버 요청 후 수신 */
articleElem.innerHTML = article;

// 별도 강조 코드 작성
articleElem
  .querySelectorAll('pre[class*="language-"]')
  .forEach(Prism.highlightElem);
```

##### `MutationObserver` <sub>(객체)</sub>
- 코드 조각 삽입 감지 후 문법 강조

#### 동적 강조 데모

##### 모든 요소 감시
- 코드 조각 발견 시 문법 강조
```javascript
let observer = new MutationObserver(mutations => {

  for (let mutation of mutations) {

    // 새 노드 추가 시 문법 강조 여부 확인
    for (let node of mutation.addedNodes) {

      // 요소 외 생략
      if (!(node instanceof HTMLElement)) {
        continue;
      }

      // 코드 조각 유무 확인 후 문법 강조
      if (node.matches('pre[class*="language-"]')) {
        Prism.highlightElement(node);
      }

      // 해당 요소 자손 요소
      // - 코드 조각 여부 확인 · 문법 강조
      for (let elem of node.querySelectorAll('pre[class*="language-"]')) {
        Prism.highlightElement(elem);
      }
    }
  }
});

let demoElem = document.getElementById('highlight-demo');

observer.observe(
  demoElem,
  {childList: true, subtree: true}
);

// 내용 (코드 조각 포함) 동적 삽입
// - 문법 강조 동작 테스트
demoElem.innerHTML = `
  A code snippet is below:
  <pre class="language-javascript">
    <code> let hello = "world!"; </code>
  </pre>
  <div>Another one:</div>
  <div>
    <pre class="language-css">
      <code>.class { margin: 5px; } </code>
    </pre>
  </div>
`;
```

### 추가 메서드

##### `disconnect()`
- 감시 중단
  - 미처리 변화 발생 가능

##### `takeRecords()`
- 미처리 변화 목록 반환
  - 변화 발생 후 · 콜백 호출 전
```javascript
// 미처리 변화 목록 처리 (감시 중단 전 필수)
let mutationRecords = observer.takeRecords();

// 감시 중단
observer.disconnect();
```
