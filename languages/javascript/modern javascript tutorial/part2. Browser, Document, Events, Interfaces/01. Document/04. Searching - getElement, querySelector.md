====

### `getElementById(id)` <sub>(메서드)</sub> · `id` <sub>(속성)</sub>

##### 1. `document.getElementById(id)` <sub>(메서드)</sub>
- 요소 내 `id` <sub>(속성)</sub> 존재 시 접근 <sub>(위치 무관)</sub>
```html
<div id="elem">
  <div id="elem-content">Element</div>
</div>

<script>
  // 요소 얻기
  let elem = document.getElementById('elem');

  // 요소 배경색 변경 (붉은색)
  elem.style.background = 'red';
</script>
```

##### 2. 전역 변수 <sub>(자동 선언)</sub>
- 요소 내 `id` <sub>(속성)</sub> 값 사용 <sub>(전역 변수명)</sub>
```html
<div id="elem">
  <div id="elem-content">Element</div>
</div>

<script>
  // elem (전역 변수)
  // - <… id="elem"> (요소) 참조
  elem.style.background = 'red';

  /* elem-content (전역 변수)
   <… id="elem-content"> (요소) 참조
   `-` (하이픈)
   - 변수명 내 사용 불가능
     - `[…]` (대괄호) 이용
   */
  window["elem-content"].style.background = 'red';
</script>
```
- 동일명 변수 선언 시 무용지물
```html
<div id="elem"></div>

<script>
  // elem (변수)
  // - <div id="elem"> (요소) 참조 X
  let elem = 5;

  alert(elem); // 5
</script>
```

### `querySelectorAll(css)` <sub>(메서드)</sub>
- CSS 선택자 대응 자식 요소 전체 반환

##### 마지막 `<li>` <sub>(요소)</sub> 모두 반환
```html
<ul>
  <li>1-1</li>
  <li>1-2</li>
</ul>
<ul>
  <li>2-1</li>
  <li>2-2</li>
</ul>
<script>
  let elements = document.querySelectorAll('ul > li:last-child');

  for (let elem of elements) {

    // 1-2, 2-2
    alert(elem.innerHTML);
  }
</script>
```

##### CSS 선택자 가상 클래스
- `:hover`
  - 마우스 포인터 이동 <sub>(요소 밖 → 요소 위)</sub>
- `:active`
  - 요소 마우스 클릭
- 기타 등등

##### `document.queryselectorAll(':hover')`
- `:hover` 상태 요소 컬렉션 반환
  - 요소 위 마우스 포인터 위치
- 컬렉션 내 순서
  - 최상단 <sub>(`<html>` 요소)</sub> → 최하단

### `querySelector(css)` <sub>(메서드)</sub>
- CSS 선택자 대응 1번째 자식 요소 반환
  - `querySelectorAll(css)[0]` 동일

##### `querySelectorAll(css)[0]`
- 선택자 대응 모든 요소 검색 후 1번째 요소 반환
  - 속도 ↓

##### `querySelector(css)`
- 검색 성공 시 검색 중단
  - 속도 ↑
- 코드 길이 ↓

### `matches(css)` <sub>(메서드)</sub>
- CSS 선택자 일치 여부 판단

##### 요소 배열 순회 시 필요 요소 필터링
```html
<a href="http://example.com/file.zip"> … </a>
<a href="http://ya.ru"> … </a>

<script>
  for (let elem of document.body.children) {
    if (elem.matches('a[href$="zip"]')) {
      alert("CSS 선택자 일치 요소: " + elem.href );
    }
  }
</script>
```

### `closest(css)` <sub>(메서드)</sub>
- CSS 선택자 일치하는 최근접 조상 요소 반환
  - 자기 자신 포함
- DOM 트리 한 단계씩 거슬러 올라가면서 검색
- 요소 검색 성공 시
  - 검색 중단
  - 결과 <sub>(최근접 조상 요소)</sub> 반환

##### 조상 요소
- 해당 요소 상위 위치 요소들
  - 부모 요소
  - 부모 요소 상위 부모 요소
  - 기타 등등
```html
<h1>목차</h1>

<div class="contents">
  <ul class="book">
    <li class="chapter">1장</li>
    <li class="chapter">2장</li>
  </ul>
</div>

<script>
  // <li class="chapter">1장</li> (요소)
  let chapter = document.querySelector('.chapter');

  // <ul class="book"> (요소)
  alert(chapter.closest('.book'));

  // <div class="contents"> (요소)
  alert(chapter.closest('.contents'));

  // null
  // - h1 (조상 요소) 無
  alert(chapter.closest('h1'));
</script>
```

### `getElementsBy[*]` <sub>(메서드들)</sub>
- 태그 · 클래스 등 이용
- 요즘 드물게 사용
- `querySelector[*]` <sub>(메서드들)</sub>
  - 편의성 ↑
  - 문법 길이 ↓

##### `getElementsByTagName(tag)`
- `tag` <sub>(문자열)</sub>
  - 태그명
  - `"*"` <sub>(모든 태그)</sub>
- 반환 값
  - 요소 컬렉션

##### `getElementsByClassName(className)`
- `className` <sub>(문자열)</sub>
  - 클래스명
- 반환 값
  - 요소 컬렉션

##### `getElementsByName(name)`
- `name` <sub>(문자열)</sub>
  - 요소명 <sub>(`name` 속성)</sub>
- 반환 값
  - 요소 컬렉션
- 문서 전체 대상 검색 수행
- 아주 드물게 사용

##### 문서 내 모든 `div` <sub>(요소)</sub> 검색
```javascript
let divs = document.getElementsByTagName('div');
```

##### 테이블 내 모든 `input` <sub>(요소)</sub> 검색
```html
<table id="table">
  <tr>
    <td>나이:</td>
    <td>
      <label>
        <input type="radio" name="age" value="young" checked> 18세 미만
      </label>
      <label>
        <input type="radio" name="age" value="mature"> 18세 이상, 60세 미만
      </label>
      <label>
        <input type="radio" name="age" value="senior"> 60세 이상
      </label>
    </td>
  </tr>
</table>

<script>
  let inputs = table.getElementsByTagName('input');

  for (let input of inputs) {
    alert( input.value + ': ' + input.checked );
  }
</script>
```

##### `<… class="article">` <sub>(요소)</sub> 검색
```html
<form name="my-form">
  <div class="article">글</div>
  <div class="long article">내용이 긴 글</div>
</form>

<script>
  // name (속성) 이용해 검색
  let form = document.getElementsByName('my-form')[0];

  // form 내에서 클래스 이름 이용해 검색
  let articles = form.getElementsByClassName('article');
  alert(articles.length); // 2
</script>
```

### 살아있는 컬렉션

##### `getElementsBy[*]` <sub>(메서드)</sub>
- 살아있는 컬렉션 반환
- 문서 변경 발생 시
  - 컬렉션 '자동 갱신' <sub>(최신 상태 유지)</sub>
```html
<div>첫 번째 div</div>
<script>
  // 1번째 스크립트
  // - <div> (태그) 상응 요소 컬렉션 참조 생성
  // - 컬렉션 길이 : 1 (스크립트 실행 시점)
  let divs = document.getElementsByTagName('div');
  alert(divs.length); // 1
</script>

<div>두 번째 div</div>
<script>
  // 2번째 스크립트
  // - 문서에 <div> (태그) 하나 더 추가된 이후 실행
  // - 컬렉션 길이 : 2
  alert(divs.length); // 2
</script>
```

##### `querySelector[*]` <sub>(메서드)</sub>
- 정적인 컬렉션 반환
- 컬렉션 확정 후 변화 X
```html
<div>첫 번째 div</div>

<script>
  let divs = document.querySelectorAll('div');
  alert(divs.length); // 1
</script>

<div>두 번째 div</div>

<script>
  alert(divs.length); // 1
</script>
```

##### 노드 검색 주요 메서드 <sub>(6가지)</sub>

|메서드|검색 기준|요소 대상 호출|컬렉션 갱신
|---|---|:---:|:---:|
|`querySelector`|CSS 선택자|✔|-|
|`querySelectorAll`|CSS 선택자|✔|-|
|`getElementById`|`id` <sub>(속성)</sub>|-|-|
|`getElementsByName`|`name` <sub>(속성)</sub>|-|✔|
|`getElementsByTagName`|태그명 · `'*'`|✔|✔|
|`getElementsByClassName`|`class` <sub>(속성)</sub>|✔|✔|

#### 실무

##### `querySelector[*]` <sub>(메서드)</sub>
- 자주 사용

##### `getElementBy[*]` <sub>(메서드)</sub>
- 보통 오래된 스크립트 내 존재
- 일부 상황 필요

#### 이 외 메서드

##### `matches(css)`
- CSS 선택자 일치 여부 검사

##### `closest(css)`
- CSS 선택자 일치하는 최근접 조상 요소 탐색
- 자기 자신 검색 대상 포함

#### 부모 · 자식 노드 관계 확인 메서드

##### `a.contains(b)`
- 자신 · 자손 일치 여부 검사
  - `b` == `a`
  - `b` == `a` 자손

## <img src="../../images/commons/icons/circle-check-solid.svg" /> 과제

### 요소 검색하기

##### 특정 테이블 · 폼 문서 찾기 <sub>(해당 조건 충족 요소)</sub>
1. `<table id="age-table">` <sub>(요소)</sub>
2. 테이블 내 `<label>` <sub>(요소)</sub>
    - 총 3개
3. 테이블 내 1번째 `<td>` <sub>(요소)</sub>
    - `<td>Age</td>`
4. `<form name="search">` <sub>(요소)</sub>
5. 폼 내 1번째 `<input>` <sub>(요소)</sub>
6. 폼 내 마지막 `<input>` <sub>(요소)</sub>

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

```javascript
/* 1. <table id="age-table"> (요소) */
let table = document.getElementById('age-table');

/* 2. 테이블 내 <label> (요소) */
// - 총 3개
table.getElementsByTagName('label');
document.querySelectorAll('#age-table label');

/* 3. 테이블 내 1번째 <td> (요소) */
// - <td>Age</td>
table.rows[0].cells[0];
table.getElementsByTagName('td')[0];
table.querySelector('td');

/* 4. <form name="search"> (요소) */
// 단 1개 요소만 존재 시
document.getElementsByName('search')[0];

// 폼 정확히 지정
document.querySelector('form[name="search"]');

/* 5. 폼 내 1번째 <input> (요소) */
form.getElementsByTagName('input')[0];
form.querySelector('input');

/* 6. 폼 내 마지막 <input> (요소) */
//   1. 모든 input (요소) 검색
//   2. 마지막 요소 얻기
let inputs = form.querySelectorAll('input');
inputs[inputs.length-1];
```
