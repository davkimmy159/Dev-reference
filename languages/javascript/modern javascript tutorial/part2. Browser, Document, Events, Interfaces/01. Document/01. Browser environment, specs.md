#### `window` <sub>(최상단 · '루트' 객체)</sub>

##### 1. JS 전역 객체
```javascript
function sayHi() {
  alert("안녕하세요.");
}

// 전역 함수 == window (전역 객체) 메서드
window.sayHi();
```

##### 2. 브라우저 창 대변 <sub>(제어 메서드 제공)</sub>
- 다양한 메서드 · 프로퍼티 구현
```javascript
// 창 내부 높이
alert(window.innerHeight);
```

### 문서 객체 모델 <sub>(DOM)</sub>
- Document Object Model
- 웹 페이지 내 모든 콘텐츠
  - 객체 <sub>(수정 가능)</sub> 표현

##### `document` <sub>(객체)</sub>
- 페이지 기본 '진입점' 역할
- 페이지 내 모든 것 변경 가능
- 원하는 것 생성 가능
```javascript
// 배경색 변경 (붉은색)
document.body.style.background = "red";

// 1초 후 배경색 초기화
setTimeout(() => document.body.style.background = "", 1000);
```

### 브라우저 객체 모델 <sub>(BOM)</sub>
- Browser Object Model
- 브라우저 <sub>(호스트 환경)</sub> 제공 추가 객체
  - 문서 외 모든 것 제어

##### `navigator` <sub>(객체)</sub>
- 브라우저 · 운영체제 정보 제공
- `userAgent` <sub>(프로퍼티)</sub>
  - 현재 사용 중인 브라우저 정보
- `platform` <sub>(프로퍼티)</sub>
  - 브라우저 실행 중인 운영체제 정보
- 기타 등등

##### `location` <sub>(객체)</sub>
- 현재 url 읽기
- 새 url 로 변경 <sub>(redirect)</sub>
```javascript
// 현재 url 표시
alert(location.href);

if (confirm("위키피디아 페이지로 가시겠습니까?")) {

  // 새 페이지 (url) 로 넘어감
  location.href = "https://wikipedia.org";
}
```

- BOM 일부 <sub>(순수 브라우저 메서드)</sub>
  - 문서와 직접 연결 X
  - 사용자 · 브라우저 상호작용 도움
