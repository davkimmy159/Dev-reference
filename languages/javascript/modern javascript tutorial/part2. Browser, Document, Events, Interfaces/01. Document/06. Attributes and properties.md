### DOM 프로퍼티

#### 커스텀 프로퍼티

##### 내장 프로퍼티 불충분 시
- 새 프로퍼티 추가

##### `document.body` <sub>(요소)</sub>
- 새 프로퍼티 추가
```javascript
// 커스텀 프로퍼티 추가
document.body.myData = {
  name: 'Caesar',
  title: 'Imperator'
};

// Imperator
alert(document.body.myData.title);

// 커스텀 메서드 추가
// - this 값 == document.body
document.body.sayTagName = function() {
  alert(this.tagName);
};

document.body.sayTagName(); // BODY
```

#### 내장 프로토타입 수정

##### 수정사항 <sub>(커스터마이징)</sub>
- 특정 노드 전체 적용

##### `Element.prototype` <sub>(내장 프로토타입)</sub> 수정
- 새 메서드 추가
  - 모든 요소 <sub>(노드)</sub> 사용 가능
```javascript
Element.prototype.sayHi = function() {
  alert(`Hello, I'm ${this.tagName}`);
};

// Hello, I'm HTML
document.documentElement.sayHi();

// Hello, I'm BODY
document.body.sayHi();
```

### HTML 속성

##### DOM 객체 생성 시
- HTML 표준 속성 인식
  - DOM 프로퍼티 생성
- HTML 非표준 속성 인식 X
  - DOM 프로퍼티 미생성
```html
<body id="test" something="non-standard">
  <script>
    // test (id 속성)
    alert(document.body.id);

    // undefined (something 속성)
    // - 非표준 속성 프로퍼티 전환 X
    alert(document.body.something);
  </script>
</body>
```

##### 요소별 전용 표준 속성 有 <sub>(명세서 참조)</sub>
- ex\) `type` <sub>(속성)</sub>
  - `<input>` <sub>(`HTMLInputElement`)</sub> 표준
  - `<body>` <sub>(`HTMLBodyElement`)</sub> 표준 X
```html
<body id="body" type="text">
  <input id="input" type="text">

  <script>
    // text (type 속성)
    alert(input.type);

    // undefined (type 속성)
    alert(body.type);
  </script>
</body>
```

##### HTML 명시 속성 <sub>(非표준 속성 등)</sub> 접근 메서드
- `hasAttribute(name)`
  - 속성 <sub>(`name`)</sub> 유무 여부 확인
- `getAttribute(name)`
  - 속성 <sub>(`name`)</sub> 값 가져오기
- `setAttribute(name, value)`
  - 속성 <sub>(`name`)</sub> 값 변경 <sub>(`value`)</sub>
- `removeAttribute(name)`
  - 속성 <sub>(`name`)</sub> 값 제거

##### `attributes` <sub>(프로퍼티)</sub>
- 모든 속성값 가져오기
- `Attr` <sub>(내장 클래스)</sub> 구현 객체 컬렉션
  - `name` <sub>(프로퍼티)</sub>
  - `value` <sub>(프로퍼티)</sub>
```html
<body something="non-standard">
  <script>
    // non-standard (비표준 속성 접근)
    alert(document.body.getAttribute('something'));
  </script>
</body>
```

##### HTML 속성 특징
- 대 · 소문자 구분 X
  - ex\) `id` == `ID`
- 값
  - 항상 문자열
```html
<body>
  <div id="elem" about="Elephant"></div>

  <script>
    /* ①  속성 읽기 (about="Elephant")
     HTML 내 항상 소문자 (about)
     속성 대 · 소문자 구분 X
     - about == About
     */
    alert( elem.getAttribute('About') );

    /* ② 속성 추가 (test="123")
     모든 값 : 속성 대입 가능
     - 최종적으로 문자열 변환
       - 123 <sub>(숫자)</sub> → "123" <sub>(문자열)</sub>
     */
    elem.setAttribute('Test', 123);

    /* ③ 추가된 속성 확인
     3. `outerHTML` <sub>(프로퍼티)</sub>
     - 모든 속성 확인 가능
     */
    alert( elem.outerHTML );

    /* ④ 전체 속성 나열
     4. `attributes` <sub>(프로퍼티)</sub> 반환 컬렉션
    - 열거 가능 <sub>(이터러블 객체)</sub>
    - `Attr` <sub>(내장 클래스)</sub> 객체
      - `name` <sub>(프로퍼티)</sub>
      - `value` <sub>(프로퍼티)</sub>

     */
    for (let attr of elem.attributes) {
      alert( `${attr.name} = ${attr.value}` );
    }
  </script>
</body>
```

### 프로퍼티 · 속성 동기화

##### 표준 속성 수정 시
- 대응 프로퍼티 자동 갱신

##### 대응 프로퍼티 수정 시
- 속성 자동 갱신 <sub>(예외 有)</sub>

##### 표준 속성 ⇆ 대응 프로퍼티
```html
<input>

<script>
  let input = document.querySelector('input');

  // 속성 추가 → 프로퍼티 갱신
  input.setAttribute('id', 'id');
  alert(input.id); // id

  // 프로퍼티 변경 → 속성 갱신
  input.id = 'newId';

  // newId
  alert(input.getAttribute('id'));
</script>
```

##### 예외
- 단방향 <sub>(속성 → 프로퍼티)</sub>
  - ex\) `input.value`
- 유용한 상황 有
  - `value` 값 수정 후
    - '기존' 값 복구 <sub>(속성 저장 값)</sub>
```html
<input>

<script>
  let input = document.querySelector('input');

  // 속성 추가 → 프로퍼티 갱신
  input.setAttribute('value', 'text');

  // text
  alert(input.value);

  // 프로퍼티 변경 → 속성 갱신 X
  input.value = 'newValue';

  // text
  alert(input.getAttribute('value'));
</script>
```

##### `style` <sub>(속성)</sub>
- 속성값
  - 문자열
- 대응 프로퍼티
  - 객체
```html
<div id="div" style="color:red;font-size:120%">Hello</div>

<script>
  /* string */
  // color:red;font-size:120%
  alert(div.getAttribute('style'));

  /* object */
  // [object CSSStyleDeclaration]
  alert(div.style);

  // red
  alert(div.style.color);
</script>
```

##### 대다수 프로퍼티 값
- 문자열

##### 속성과 다른 프로퍼티 문자열 값 예외 有 <sub>(아주 드묾)</sub>
- ex\) `href` <sub>(속성)</sub>

    - 상대 url · `#hash`
  - 프로퍼티
    - 항상 url 전체
```html
<a id="a" href="#hello">link</a>

```

### 비표준 속성 <sub>(`dataset`)</sub>

##### 사용자 지정 데이터
- HTML → JS
- JS 조작 대상 HTML 요소 표시
```html
<!-- 이름 (name) 정보 div -->
<div show-info="name"></div>

<!-- 나이 (age)  정보 div -->
<div show-info="age"></div>

<script>
  let user = {
    name: "Pete",
    age: 25
  };

  for (let div of document.querySelectorAll('[show-info]')) {

    // 원하는 정보
    // - 필드 값 입력
    let field = div.getAttribute('show-info');

    // Pete → <div show-info="name">
    // 25   → <div show-info="age">
    div.innerHTML = user[field];
  }
</script>
```

##### 요소 스타일 적용
- `order-state` <sub>(커스텀 속성)</sub>
  - 주문 상태 따라 스타일 변경
```html
<style>
  /* order-state (커스텀 속성) 따라 스타일 변경 */
  .order[order-state="new"] {
    color: green;
  }

  .order[order-state="pending"] {
    color: blue;
  }

  .order[order-state="canceled"] {
    color: red;
  }
</style>

<div class="order" order-state="new">
  A new order.
</div>

<div class="order" order-state="pending">
  A pending order.
</div>

<div class="order" order-state="canceled">
  A canceled order.
</div>
```

##### 커스텀 속성 선호 이유
- 클래스보다 다루기 편리
  - 속성 상태 쉽게 변경
- 클래스
  - `.order-state-new`
  - `.order-state-pending`
  - `.order-state-canceled`
```javascript
// 새 클래스 추가 · 제거보다 간단
div.setAttribute('order-state', 'canceled');
```

#### 외부 참조 찾기

##### 1. `document.querySelectorAll('a')`
1. 모든 링크 찾기
2. 필요한 것만 걸러내기
```javascript
let links = document.querySelectorAll('a');

for (let link of links) {
  let href = link.getAttribute('href');

  // 속성 부재
  if (!href) continue;

  // 프로토콜 부재
  if (!href.includes('://')) continue;

  // 제외 링크
  // - http://internal.com
  if (href.startsWith('http://internal.com')) continue;

  link.style.color = 'orange';
}
```

##### 2. CSS 선택자 조건 명시
```javascript
// ① href (속성) 내 '://' (프로토콜) 포함 링크
// ② 그 중 'http://internal.com' 시작 X
let selector = 'a[href*="://"]:not([href^="http://internal.com"])';
let links = document.querySelectorAll(selector);

links.forEach(link => link.style.color = 'orange');
```

- HTML 값 필요 시
