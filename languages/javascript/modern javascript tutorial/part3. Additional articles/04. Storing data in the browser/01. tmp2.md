### 부록 1 <sub>(쿠키 함수)</sub>

##### 다양한 쿠키 관련 라이브러리 有
- 참고용 쿠키 함수

#### `getCookie(name)`

##### 쿠키 <sub>(`name`)</sub> 반환
- 정규 표현식 사용
- 쿠키값 디코딩 후 반환
  - `decodeURIComponent` <sub>(함수)</sub> 사용
```javascript
function getCookie(name) {
  let matches = document.cookie.match(new RegExp(
    "(?:^|; )" + name.replace(/([\.$?*|{}\(\)\[\]\\\/\+^])/g, '\\$1') + "=([^;]*)"
  ));

  // 해당 조건 쿠키 無
  // - undefined 반환
  return matches ? decodeURIComponent(matches[1]) : undefined;
}
```

#### `setCookie(name, value, options)`

##### 쿠키 <sub>(`name`)</sub> 설정 <sub>(`value`)</sub>
- `path` <sub>(옵션)</sub> 기본값
  - 현재 경로 <sub>(`path=/`)</sub>
  - 추가 가능
```javascript
function setCookie(name, value, options = {}) {
  options = {
    path: '/',

    // 옵션 기본값 설정
    ...options
  };

  if (options.expires instanceof Date) {
    options.expires = options.expires.toUTCString();
  }

  let updatedCookie = encodeURIComponent(name) + "=" + encodeURIComponent(value);

  for (let optionKey in options) {
    updatedCookie += "; " + optionKey;
    let optionValue = options[optionKey];
    if (optionValue !== true) {
      updatedCookie += "=" + optionValue;
    }
  }

  document.cookie = updatedCookie;
}

// 사용법
setCookie('user', 'John', { secure: true, 'max-age': 3600 });
```

#### `deleteCookie(name)`

##### 쿠키 <sub>(`name`)</sub> 삭제
- 삭제 시 쿠키 만료 기간 음수 설정
```javascript
function deleteCookie(name) {
  setCookie(name, "", {
    'max-age': -1
  })
}
```

### 부록 2 <sub>(서드파티 쿠키)</sub>

##### 서드파티 쿠키
- 타 도메인 설정 쿠키

#### 예시

##### 1. `site.com` 내 특정 페이지
- 이미지 배너 <sub>(타 도메인)</sub> 불러오기
```html
<!-- 타 도메인 이미지 배너 -->
<img src="https://ads.com/banner.png">
```

##### 2. 원격 서버 <sub>(`ads.com` · 이미지 배너 도메인)</sub>
- 배너 전송 시 `Set-Cookie` <sub>(헤더)</sub> 포함
  - 브라우저 쿠키 설정 <sub>(`id=1234` 등)</sub> 유도
- 설정 유도된 브라우저 측 쿠키 <sub>(`id=1234` 등)</sub>
  - `ads.com` <sub>(도메인)</sub> 에서 설정
    - 해당 도메인에서만 보기 가능

![cookie-third-party](../../images/03/04/01/cookie-third-party.svg)

##### 3. 사용자 `site.com` 재접속
- 원격 서버 <sub>(`ads.com`)</sub> 해당 유저 인식
  - 요청 내 쿠키 <sub>(`id=1234 등`)</sub> 이용

![cookie-third-party-2](../../images/03/04/01/cookie-third-party-2.svg)

##### 4. 사용자 이동 <sub>(`site.com`  → `other.com`)</sub>
- `other.com` 내 배너 有
  - 원격 서버 <sub>(`ads.com`)</sub> 쿠키 재수신
    - 해당 쿠키 설정 주체 <sub>(원격 서버)</sub>
    - 사용자 인식 · 이동 추적

##### 광고회사
- 오래전부터 서드파티 쿠키 사용

### 부록 2 <sub>(GDPR)</sub>

##### GDPR 법령
- EU 내 사용자 개인 정보 보호 강제
- 전반적인 보안 이슈 관련 내용
  - 쿠키 등

##### 쿠키 이용한 사용자 추적 · 식별 관련 내용 <sub>(중요 요건 중 하나)</sub>
- 쿠키 추적 시 사용자 명시적 허가 필수

##### 법령 미적용
- 정보 저장 용도 쿠키 설정 · 사용
  - 사용자 추적 · 식별 X

##### 법령 적용 <sub>(사용자 동의 필요)</sub>
- 인증 세션 함께 쿠키 설정

#### GDPR 대응 방법

##### 1. 인증된 사용자 <sub>(전용)</sub> 대상 추적 쿠키 설정
- 가입 양식 내 확인란 표시
  - ex\) "개인 정보 취급 방침 동의" 등
  - 사용자 동의 시 추적 쿠키 설정

##### 2. 모든 사용자 대상 추적 쿠키 설정
1. 최초 방문자 대상 "작은 창" 표시
    - 쿠키 설정 동의 요구
2. 사용자 동의 시 콘텐츠 표시 · 추적 쿠키 설정

요약
====

##### `document.cookie`
- 쿠키 접근 기능
- 쓰기
  - 해당 쿠키 값만 갱신
- 쿠키 이름 · 값
  - 인코딩 필수
- 쿠키 1개 최대 허용 용량
  - 4KB
- 사이트 1개 허용 쿠키 개수
  - 약 20여 개 <sub>(브라우저별 상이)</sub>

#### 쿠키 옵션

##### `path=/`
- 쿠키 접근 페이지 제한
  - 설정 경로 · 하위 경로
- 기본 설정
  - 현재 경로

##### `domain`
- 쿠키 접근 가능 도메인 지정
  - 해당 도메인 · 서브도메인
- 기본 설정
  - 쿠키 설정 도메인

##### `expires` · `max-age`
- 쿠키 만료 시간 지정
- 기본 설정
  - 브라우저 닫힐 때 쿠키 삭제

##### `secure`
- 쿠키 접근 가능 프로토콜 제한
  - HTTPS 연결

##### `samesite`
- 외부 사이트 요청
  - 브라우저 측 쿠키를 전송 방지
- XSRF 공격 방지 유용

#### 추가 사항

  - 서드파티 쿠키 기본 금지

##### EU 국가 거주 사용자 대상

  - 사용자 추적 시 동의 필수
