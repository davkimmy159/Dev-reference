쿠키 · `document.cookie`
====

### 쿠키

##### 소용량 문자열
- 브라우저 내 저장

##### HTTP 프로토콜 일부
- [RFC6265](https://datatracker.ietf.org/doc/html/rfc6265) <sub>(명세)</sub> 정의

##### 주로 웹 서버 의해 생성
1. 서버 HTTP 응답 헤더 전송
    - `Set-Cookie`
2. 브라우저 해당 헤더 내용 자체 저장
    - 쿠키
3. 브라우저 해당 사이트 재접속 시
    - 요청 헤더 전송
      - `Cookie` <sub>(쿠키 내용)</sub>

#### 인증 시 자주 사용 <sub>(클라이언트 식별 등)</sub>

##### 1. 사용자 로그인
- 서버 쿠키 설정
  - `Set-Cookie` <sub>(응답 헤더)</sub> 내
    - '세션 식별자' 정보 이용

##### 2. 사용자 동일 도메인 접속
- 요청 전송 <sub>(브라우저)</sub>
  - 인증 정보 담긴 고윳값 <sub>(세션 식별자)</sub>
    - `Cookie` <sub>(요청 헤더)</sub> 내 명시

##### 3. 서버 사용자 식별
- 요청 헤더 <sub>(브라우저)</sub> 내 세션 식별자 읽기

##### `document.cookie` <sub>(프로퍼티)</sub>
- 브라우저 쿠키 접근 시 사용

### 쿠키 읽기

##### `javascript.info` 관련 쿠키 확인
```javascript
// javascript.info
// - Google Analytics (GA) 사용해 데이터 수집

// cookie1=value1; cookie2=value2; …
alert( document.cookie );
```

##### `document.cookie` <sub>(프로퍼티)</sub> 값
- 키 · 값 쌍 구성
  - `name=value`
  - 독립된 쿠키 표현
- 구분자
  - `;` <sub>(세미콜론)</sub>

##### 쿠키 검색
- `document.cookie` <sub>(프로퍼티)</sub> 값 분리
  - `;` <sub>(세미콜론)</sub> 기준
  - 정규 표현식 · 배열 함수 함께 사용

### 쿠키 쓰기

##### `document.cookie` <sub>(프로퍼티)</sub>
- 직접 값 쓰기 가능
- 접근자 프로퍼티 <sub>(\[g·s\]etter)</sub>
  - 값 할당 시
    - 해당 쿠키 갱신
    - 타 쿠키 미갱신

##### `user` <sub>(쿠키)</sub> 갱신
- `user=John`
```javascript
// user (쿠키) 만 갱신
document.cookie = "user=John";

// 모든 쿠키 출력
alert(document.cookie);
```

##### 쿠키 할당 연산 <sub>(`document.cookie=`)</sub>
- 모든 쿠키 덮어쓰기 X
- 명시 쿠키 값만 수정 <sub>(갱신)</sub>

##### 쿠키 이름 · 값
- 제약 無
  - 모든 글자 허용
- 이스케이프 처리 필수
  - 일관된 형식 유효성 유지
  - `encodeURIComponent` <sub>(함수)</sub> 사용
```javascript
// 특수 값 (공백)
let name = "my name";
let value = "John Smith"

// 인코딩
document.cookie = encodeURIComponent(name) + '=' + encodeURIComponent(value);

// …; my%20name=John%20Smith
alert(document.cookie); 
```

#### 쿠키 제약 사항

##### 인코딩 후 `name=value` <sub>(키 · 값 쌍)</sub> 용량 한정 <sub>(4KB 이하)</sub>
- 용량 초과 <sub>(4KB 이상)</sub> 시 저장 불가능

##### 도메인 1개당 저장 가능 쿠키 개수 한정 <sub>(약 20여 개)</sub>
- 브라우저별 약간 상이

<br />

##### 쿠키 옵션
- 일부 옵션 지정 필수
- `key=value` <sub>(키 · 값 쌍)</sub> 뒤 나열
- 구분자
  - `;` <sub>(세미콜론)</sub>
```javascript
document.cookie = "user=John; path=/; expires=Tue, 19 Jan 2038 03:14:07 GMT"
```

### `path` <sub>(옵션 · url 경로 접두사)</sub>
- `path=/mypath`

##### 쿠키 접근 가능 페이지 제한
- 해당 · 하위 경로

##### 값 설정
- 절대 경로
- 상대 경로 X

##### 기본 설정
- 현재 경로

##### ex\) `path=/admin`
- 쿠키 접근 가능
  - `/admin`
  - `/admin/something`
  - `/admin/…`
- 쿠키 접근 불가능
  - `/home`
  - `/adminpage`
  - 기타 등등

##### 일반적인 옵션 설정 값
- `path=/` <sub>(루트)</sub>
  - 웹사이트 내 모든 페이지 접근 허용

### `domain` <sub>(옵션)</sub>
- `domain=site.com`

##### 쿠키 접근 가능 도메인 제한
- 일부 제약 有
  - 특정 도메인 지정 불가능

##### 기본 설정
- 쿠키 설정 도메인
  - ex\) `site.com` 생성 쿠키
    - `other.com` 접근 불가능

##### 서브도메인 <sub>(제약)</sub>
- 루트 도메인 쿠키 접근 불가능
```javascript
/* site.com (루트 도메인) */
// 쿠키 설정
document.cookie = "user=John"
```
```javascript
/* forum.site.com (서브도메인) */
// user 쿠키 (루트 도메인 설정) 접근 불가능
alert(document.cookie); // 찾을 수 없음
```

##### 도메인 제약사항
- 서브 · 타 도메인 쿠키 접속 불가능
  - ex\) `site.com` 생성 쿠키
    - `other.com` 접근 불가능
- 안정성 ↑
  - 민감한 데이터 저장 쿠키
    - 관련 페이지에서만 접근 가능

##### 서브도메인 쿠키 접근 허용 방법
- `domain` <sub>(옵션)</sub>
  - 루트 도메인 명시적 설정
    - ex\) `domain=site.com`
```javascript
/* site.com (루트 도메인) */
// domain (옵션) 설정
// - 서브도메인 (*.site.com) 내
//   - user (루트 도메인 설정) 쿠키 접속 가능
document.cookie = "user=John; domain=site.com"
```
```javascript
/* forum.site.com (서브도메인) */
alert(document.cookie); // user=John
```

##### `domain=.도메인` <sub>(도메인명 앞 점)</sub>
- 동일 기능
- 하위 호환성 유지
  - 구식 브라우저 지원
- ex\) `domain=.site.com`

### `expires` · `max-age` <sub>(옵션)</sub>

##### 브라우저 종료 후에도 쿠키 유지
- `expires` <sub>(유효 일자)</sub>
- `max-age` <sub>(만료 기간)</sub

##### 미지정
- 브라우저 종료 시 삭제 <sub>(세션 쿠키)</sub>

```javascript
expires=Tue, 19 Jan 2038 03:14:07 GMT
```

##### 브라우저
- 설정 유효 일자까지 쿠키 유지
- 해당 일자 도달 시
  - 해당 쿠키 자동 삭제

##### `expires` <sub>(옵션)</sub>
- 쿠키 유효 일자
  - 반드시 GMT <sub>(포맷)</sub> 설정
    - `toUTCString` <sub>(메서드)</sub> 사용
- 과거 값
  - 즉시 쿠키 삭제

##### 유효 기간 1일 쿠키 생성
```javascript
// 현재 시간 ~ 1일 후
let date = new Date(Date.now() + 86400e3);
date = date.toUTCString();
document.cookie = "user=John; expires=" + date;
```

##### `max-age` <sub>(옵션)</sub>
- `expires` <sub>(옵션)</sub> 대안
- 쿠키 만료 기간 설정
  - 현재 ~ <sub>(범위)</sub>
  - 초 <sub>(단위)</sub>
- `0` · 음수값
  - 즉시 쿠키 삭제
```javascript
max-age=3600
```

```javascript
// 1시간 뒤 삭제
document.cookie = "user=John; max-age=3600";

// 즉시 삭제
document.cookie = "user=John; max-age=0";
```

### `secure` <sub>(보안)</sub>
- `secure`

##### 쿠키 전송 프로토콜 제한
- HTTPS 통신만 허용

##### 기본 설정 <sub>(미지정)</sub>
- HTTP ↔ HTTPS
  - 각자 쿠키 서로 접근 가능
- ex\) `http://site.com` 생성 쿠키
  - `https://site.com` 접근 가능
- ex\) `https://site.com` 생성 쿠키
  - `http://site.com` 접근 가능

##### 기본 확인 사항
- 도메인
- 프로토콜 X

##### `secure` <sub>(옵션)</sub> 설정
- HTTP ↛ HTTPS
- ex\) `https://site.com` 생성 쿠키
  - `http://site.com` 접근 X
- 민감한 데이터 저장 쿠키
  - 非암호화 HTTP 연결 통해 전달 X
```javascript
// HTTPS 통신 시만 'user' 쿠키 접근 가능
document.cookie = "user=John; secure";
```
