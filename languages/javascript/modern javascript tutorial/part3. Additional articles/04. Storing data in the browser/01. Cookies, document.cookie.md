쿠키 · `document.cookie`
====

### 쿠키

##### 소용량 문자열
- 브라우저 내 저장

##### HTTP 프로토콜 일부
- [RFC6265](https://datatracker.ietf.org/doc/html/rfc6265) <sub>(명세)</sub> 정의

##### 주로 웹 서버 의해 생성
1. 서버 HTTP 응답 헤더 전송
    - `Set-Cookie`
2. 브라우저 해당 헤더 내용 자체 저장
    - 쿠키
3. 브라우저 해당 사이트 재접속 시
    - 요청 헤더 전송
      - `Cookie` <sub>(쿠키 내용)</sub>

#### 인증 시 자주 사용 <sub>(클라이언트 식별 등)</sub>

##### 1. 사용자 로그인
- 서버 측 쿠키 설정
  - '세션 식별자' 정보 이용
    - HTTP 응답 헤더 <sub>(`Set-Cookie`)</sub>

##### 2. 사용자 동일 도메인 접속
- 브라우저 요청 전송
  - 인증 정보 담긴 고윳값 <sub>(세션 식별자)</sub>
    - HTTP `Cookie` <sub>(헤더)</sub> 내 저장

##### 3. 서버 측 사용자 식별
- 브라우저 요청 헤더 내 세션 식별자 읽기

##### `document.cookie` <sub>(프로퍼티)</sub>
- 브라우저 내 쿠키 접근

### 쿠키 읽기

##### `javascript.info` 관련 쿠키 확인
```javascript
// javascript.info
// - Google Analytics (GA) 사용해 데이터 수집
alert( document.cookie ); // cookie1=value1; cookie2=value2;...
```

##### `document.cookie` <sub>(프로퍼티)</sub> 값
- 쌍 구성
  - `name=value`
  - 독립된 쿠키 표현
- 구분자
  - `;` <sub>(세미콜론)</sub>

##### 쿠키 찾기
- `document.cookie` 값 분리
  - `;` <sub>(세미콜론)</sub> 기준
- 정규 표현식 · 배열 함수 함께 사용

### 쿠키 쓰기

##### `document.cookie`
- 직접 값 쓰기 가능
- 접근자 프로퍼티
- 값 할당 시
  - 해당 쿠키 갱신 <sub>(브라우저)</sub>
  - 타 쿠키 변경

##### `user` 쿠키 값 갱신
- `John`
```javascript
// 'user' 쿠키 값만 갱신
document.cookie = "user=John";

// 모든 쿠키 출력
alert(document.cookie);
```

##### 쿠키 할당 연산 <sub>(`document.cookie=`)</sub>
- 모든 쿠키 덮어쓰기 X
- 명시 쿠키 값만 갱신

##### 쿠키 이름 · 값
- 특별 제약 X
  - 모든 글자 허용
- 이름 · 값 이스케이프 처리
  - 형식 유효성 유지 필요
  - `encodeURIComponent` <sub>(내장 함수)</sub> 사용
```javascript
// 특수 값 (공백)
// - 인코딩 처리
let name = "my name";
let value = "John Smith"

// 인코딩 처리
// - my%20name=John%20Smith
document.cookie = encodeURIComponent(name) + '=' + encodeURIComponent(value);

alert(document.cookie); // …; my%20name=John%20Smith
```

#### 쿠키 제약 사항

##### 인코딩 후 `name=value` 쌍 용량 한정 <sub>(4KB 이하)</sub>
- 용량 초과 시 저장 X

##### 도메인 1개당 저장 가능 쿠키 개수 한정 <sub>(약 20여 개)</sub>
- 브라우저별 약간 상이

<br />

##### 쿠키 옵션
- 일부 옵션 지정 필수
- 위치
  - `key=value` 쌍 뒤 나열
- 구분자
  - `;` <sub>(세미콜론)</sub>
```javascript
document.cookie = "user=John; path=/; expires=Tue, 19 Jan 2038 03:14:07 GMT"
```

### `path` <sub>(url 경로 접두사)</sub>
- `path=/mypath`

##### 쿠키 접근 가능 페이지 제한
- 해당 경로 · 하위 경로
- 절대 경로

##### 기본 설정 <sub>(미지정)</sub>
- 현재 경로

##### ex\) `path=/admin`
- 쿠키 접근 가능
  - `/admin`
  - `/admin/something`
  - 기타 등등
- 쿠키 접근 X
  - `/home`
  - `/adminpage`
  - 기타 등등

##### 일반적인 옵션 설정
- `path=/` <sub>(루트 설정)</sub>
  - 웹사이트 내 모든 페이지 접근 허용

### `domain`
- `domain=site.com`

##### 쿠키 접근 가능 도메인 지정
- 일부 제약 有
  - 아무 도메인 지정 X

##### 기본 설정 <sub>(미지정)</sub>
- 쿠키 설정 도메인
- ex\) `site.com` 생성 쿠키
  - `other.com` 접근 X

##### 서브도메인
- 루트 도메인 쿠키 접근 X
```javascript
/* site.com (루트 도메인) */
// 쿠키 설정
document.cookie = "user=John"
```
```javascript
/* forum.site.com (서브도메인) */
// 'user' 쿠키 접근
alert(document.cookie); // 찾을 수 없음
```

##### 도메인 제약사항
- 서브 · 타 도메인 쿠키 접속 X
  - ex\) `site.com` 생성 쿠키
    - `other.com` 접근 X
- 안정성 ↑
  - 민감한 데이터 저장 쿠키
    - 관련 페이지에서만 접근 허용

##### 서브도메인 쿠키 접근 허용 방법
- `domain` <sub>(옵션)</sub>
  - 루트 도메인 명시적 설정
    - ex\) `domain=site.com`
```javascript
/* site.com (루트 도메인) */
// 서브 도메인 (*.site.com) 어디서든 'user' 쿠키 접속 가능
document.cookie = "user=John; domain=site.com"
```
```javascript
/* forum.site.com (서브도메인) */
alert(document.cookie); // user=John
```

##### `domain=.도메인` <sub>(루트 도메인명 앞 점)</sub>
- 하위 호환성 유지
  - 구식 브라우저 지원
- ex\) `domain=.site.com`

### `expires` · `max-age`

##### 세션 쿠키
- 브라우저 종료 후에도 존재
- 옵션
  - `expires` <sub>(유효 일자)</sub>
  - `max-age` <sub>(만료 기간)</sub>
- 미지정 시
  - 브라우저 종료 시 쿠키 함께 삭제
```javascript
expires=Tue, 19 Jan 2038 03:14:07 GMT
```

##### 브라우저
- 설정 유효 일자까지 쿠키 유지
- 해당 일자 도달 시
  - 해당 쿠키 자동 삭제

##### `expires` <sub>(옵션)</sub>
- 쿠키 유효 일자
  - 반드시 GMT <sub>(포맷)</sub> 설정
    - `toUTCString` <sub>(메서드)</sub> 사용
- 과거 값
  - 즉시 쿠키 삭제

##### 유효 기간 1일 쿠키 생성
```javascript
// 현재 시간 ~ 1일 후
let date = new Date(Date.now() + 86400e3);
date = date.toUTCString();
document.cookie = "user=John; expires=" + date;
```

##### `max-age` <sub>(옵션)</sub>
- `expires` <sub>(옵션)</sub> 대안
- 쿠키 만료 기간 설정
  - 현재 ~ <sub>(범위)</sub>
  - 초 <sub>(단위)</sub>
- `0` · 음수값
  - 즉시 쿠키 삭제
```javascript
max-age=3600
```

```javascript
// 1시간 뒤 삭제
document.cookie = "user=John; max-age=3600";

// 즉시 삭제
document.cookie = "user=John; max-age=0";
```

### `secure` <sub>(보안)</sub>
- `secure`

##### 쿠키 전송 프로토콜 제한
- HTTPS 통신만 허용

##### 기본 설정 <sub>(미지정)</sub>
- HTTP ↔ HTTPS
  - 각자 쿠키 서로 접근 가능
- ex\) `http://site.com` 생성 쿠키
  - `https://site.com` 접근 가능
- ex\) `https://site.com` 생성 쿠키
  - `http://site.com` 접근 가능

##### 기본 확인 사항
- 도메인
- 프로토콜 X

##### `secure` <sub>(옵션)</sub> 설정
- HTTP ↛ HTTPS
- ex\) `https://site.com` 생성 쿠키
  - `http://site.com` 접근 X
- 민감한 데이터 저장 쿠키
  - 非암호화 HTTP 연결 통해 전달 X
```javascript
// HTTPS 통신 시만 'user' 쿠키 접근 가능
document.cookie = "user=John; secure";
```
