`[local·session]Storage`
====

##### 웹 스토리지 <sub>(storage)</sub> 객체
- 브라우저 내 키 · 값 쌍 저장
- 데이터 유지 기능 제공
  - 페이지 새로 고침 <sub>(`sessionStorage`)</sub>
  - 브라우저 재실행 <sub>(`localStorage`)</sub>

#### 웹 스토리지 객체 특징 <sub>(vs 쿠키)</sub>

##### 네트워크 요청 시 서버 전송 X
- 더 많은 자료 보관 가능
- 대다수 브라우저
  - 최소 2MB · 이상 저장 가능
- 브라우저 내 웹 스토리지 구성 방식 설정 가능

##### 서버 측 HTTP 헤더 통해 스토리지 객체 조작 X
- 모든 웹 스토리지 객체 조작
  - JS 내 수행

##### 오리진 <sub>(도메인 · 프로토콜 · 포트)</sub> 종속
- 타 프로토콜 · 서브 도메인
  - 데이터 접근 X

##### 메서드 · 프로퍼티
- `setItem(key, value)`
  - 키 · 값 쌍 보관
- `getItem(key)`
  - 키 해당 값 받아오기
- `removeItem(key)`
  - 키 해당 값 삭제
- `clear()`
  - 전체 삭제
- `key(index)`
  - 인덱스 해당 키 받아오기
- `length`
  - 저장 항목 개수 얻기

##### `Map` <sub>(내장 객체)</sub> 유사 · 차이 有
- 요소 조작 메서드 지원 <sub>(유사점)</sub>
  - `setItem`
  - `getItem`
  - `removeItem`
- 인덱스 사용해 키 접근 가능 <sub>(차이)</sub>
  - `key(index)`

### `localStorage` <sub>(객체)</sub>

##### `localStorage` 주요 기능
- 동일 오리진 내 모든 탭 · 창
  - 데이터 공유
- 브라우저 · OS 재시작 시
  - 데이터 유지

##### 특정 값 설정
```javascript
localStorage.setItem('test', 1);
```

##### 설정한 값 확인
- 브라우저 재시작
- 타 창에서 해당 페이지 열기
```javascript
// 1
alert( localStorage.getItem('test') );
```

##### 동일 오리진 <sub>(도메인 · 포트 · 프로토콜)</sub> 내 모든 창
- `localStorage` 공유 <sub>(url 경로 무관)</sub>
  - 창 사이 데이터 설정 · 얻기 가능

### 일반 객체처럼 사용하기

##### 일반 객체 방식 키 설정 · 얻기 가능
```javascript
// 키 설정
localStorage.test = 2;

// 키 얻기
alert( localStorage.test ); // 2

// 키 삭제
delete localStorage.test;
```

#### 하위 호환성 유지 목적 · 미권장

##### 1. 객체 내부 프로퍼티 대상 키 설정 가능
- `length` · `toString` · 내장 메서드
  - `[get·set]Item` 정상 작동
  - 일반 객체 취급 시 에러 발생 가능
```javascript
let key = 'length';

// TypeError: Cannot assign to read only property 'length' …
localStorage[key] = 5;
```

##### 2.`storage` <sub>(이벤트)</sub> X
- 데이터 수정 시 발생
- 일반 객체 방식 취급 시 미발생

### 키 순회하기

##### 전체 키 · 값 얻기
- `iterable` <sub>(객체)</sub>
- 배열 취급 동작
```javascript
for(let i = 0; i < localStorage.length; i++) {
  let key = localStorage.key(i);
  alert(`${key}: ${localStorage.getItem(key)}`);
}
```

#### 일반 객체 취급

##### `for…in` <sub>(반복문)</sub> 사용 가능
- 불필요 내장 필드 출력
```javascript
for(let key in localStorage) {
  alert(key); // …, getItem, setItem, …
}
```
- `hasOwnProperty` <sub>(메서드)</sub>
  - 프로토타입 상속 필드 제거
```javascript
for(let key in localStorage) {
  if (!localStorage.hasOwnProperty(key)) {

    // 내장 필드 생략
    // - [get·set]Item
    // - 기타 등등
    continue;
  }

  alert(`${key}: ${localStorage.getItem(key)}`);
}
```

##### `Object.keys`
- 해당 객체 내 정의 키만 반환
- 프로토타입 상속 키 무시
```javascript
let keys = Object.keys(localStorage);
for(let key of keys) {
  alert(`${key}: ${localStorage.getItem(key)}`);
}
```

### 문자열만 사용

#### `localStorage` 키 · 값 타입

##### 반드시 문자열 사용
- 타 자료형 <sub>(숫자 · 객체 등)</sub>
  - `string` 자동 형 변환
```javascript
localStorage.user = {name: "John"};
alert(localStorage.user); // [object Object]
```
- `JSON`
  - 객체 형태 저장 가능
```javascript
localStorage.user = JSON.stringify({name: "John"});
…
let user = JSON.parse( localStorage.user );
alert( user.name ); // John
```
- 스토리지 객체 전체 문자열 변환
  - 디버깅 등 용도
```javascript
alert( JSON.stringify(localStorage, null, 2) );
```

### `sessionStorage` <sub>(객체)</sub>

##### `localStorage` 비해 덜 사용
- 프로퍼티 · 메서드 동일
- 훨씬 제한적인 사용법

##### 현재 탭 내에서만 유지
- 동일 페이지 내 타 탭
  - 데이터 공유 X
- 탭 내 다수 `iframe`
  - 데이터 공유 <sub>(동일 오리진 취급)</sub>

##### 저장 데이터 유지성
- 페이지 새로 고침
  - 데이터 유지
- 탭 닫고 새로 열기
  - 데이터 유실

##### 특정 값 설정
```javascript
sessionStorage.setItem('test', 1);
```

##### 설정한 값 확인
- 페이지 새로 고침
- 타 탭 내 해당 페이지 접속 후 실행
  - `null` 반환
```javascript
// 1
alert( sessionStorage.getItem('test') );
```

##### 오리진 · 브라우저 탭 종속
- 덜 사용 이유

### [`storage`](https://html.spec.whatwg.org/multipage/#the-storage-event) <sub>(이벤트)</sub>

##### 발생 시점
- 웹 스토리지 객체 내 데이터 갱신

##### 프로퍼티
- `key`
  - 변경된 데이터 키
  - `null` <sub>(`.clear()` 호출 시)</sub>
- `oldValue`
  - 이전 값
  - `null` <sub>(새 키 추가 시)
- `newValue`
  - 새 값
  - `null` <sub>(키 삭제 시)</sub>
- `url`
  - 갱신 발생 문서 url
- `storageArea`
  - 갱신 발생 객체
    - `localStorage`
    - `sessionStorage`

##### 이벤트 발생 대상
- 이벤트 발생 스토리지 X
- `window` <sub>(전역 객체)</sub>
  - 해당 스토리지에서 접근 가능 객체

##### 동일 사이트 내 브라우저 창 2개
- `localStorage` 공유
- 각 창 `storage` <sub>(이벤트)</sub> 수신
  - 각 창 내 데이터 갱신 공유
```javascript
// 타 문서
// - 동일 스토리지 대상 갱신 반영
// window.addEventListener('storage', () => {
window.onstorage = event => {
  if (event.key != 'now') return;
  alert(event.key + ':' + event.newValue + " at " + event.url);
};

localStorage.setItem('now', Date.now());
```

##### `event.url` <sub>(프로퍼티)</sub>
- 데이터 갱신 발생 문서 url

##### `event.storageArea` <sub>(프로퍼티)</sub>
- 실제 수정 발생 스토리지 참조
  - 스토리지 종류 무관
- `storage` <sub>(이벤트)</sub>
  - 두 스토리지 갱신 시 모두 발생
    - `localStorage`
    - `sessionStorage`

#### 동일 오리진 내 창 사이 메시지 교환

##### [브로드캐스트 채널 API](https://developer.mozilla.org/en-US/docs/Web/API/Broadcast_Channel_API) <sub>(모던 브라우저)</sub>
- 풍부한 기능
- 대중화 부족

##### `localStorage` 기반 폴리필
- 브라우저 종류 무관

요약
====

##### `[local·session]Storage` <sub>(웹 스토리지 객체)</sub>
- 브라우저 내 키 · 값 쌍 저장

##### 특징
- 키 · 값 타입
  - 반드시 문자열
- 제한 용량
  - 최소 2MB 이상 <sub>(브라우저별 상이)</sub>
- 파기 X
- 오리진 <sub>(도메인 · 포트 · 프로토콜 )</sub> 종속

||공유<br /><sub>(동일 오리진)</sub>|유지|파기|
|---|---|---|---|
|`localStorage`|창 · 탭|브라우저 종료 후 재시작||
|`sessionStorage`|탭 · `iframe`|페이지 새로 고침|브라우저 · 탭 종료|

##### API
- `setItem(key, value)`
  - 키 · 값 쌍 보관
- `getItem(key)`
  - 키 해당 값 받아오기
- `removeItem(key)`
  - 키 · 해당 값 삭제
- `clear()`
  - 전체 삭제
- `key(index)`
  - 인덱스 해당 키 받아오기
- `length`
  - 저장 항목 개수 얻기
- `Object.keys`
  - 전체 키 얻기
- 키 접근 <sub>(≒ 객체 프로퍼티)</sub>
  - `storage` <sub>(이벤트)</sub> 미발생

#### `storage` <sub>(이벤트)</sub>

##### 스토리지 내 데이터 갱신 <sub>(메서드 호출)</sub> 시 발생
- `setItem`
- `removeItem`
- `clear`

##### 내부 보유 정보
- 연산 관련 데이터 전체
  - `key`
  - `oldValue`
  -`newValue`
- `url` <sub>(문서 url)</sub>
- `storageArea` <sub>(스토리지 객체)</sub>

##### 발생 위치
- 이벤트 생성 위치 X
- 스토리지 접근 가능한 모든 `window` <sub>(객체)</sub>
  - `sessionStorage` <sub>(탭 내)</sub>
  - `localStorage` <sub>(전역)</sub>
