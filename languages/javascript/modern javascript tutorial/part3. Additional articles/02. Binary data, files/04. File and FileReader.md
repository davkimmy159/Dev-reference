`File[Reader]` <sub>(객체)</sub>
====

### [`File`](https://www.w3.org/TR/FileAPI/#dfn-file) <sub>(객체)</sub>

##### `Blob` <sub>(객체)</sub> 상속
- 파일 시스템 관련 기능 추가

#### `File` <sub>(객체)</sub> 생성 방법 <sub>(2가지)</sub>

##### 1. 생성자 <sub>(`Blob` 객체 유사)</sub>
```javascript
new File(fileParts, fileName, [options])
```
- `fileParts` <sub>(배열)</sub>
  - `Blob` <sub>(객체)</sub>
  - 문자열
  - `BufferSource` <sub>(가상 이진 처리 객체)</sub>
- `fileName` <sub>(문자열)</sub>
  - 파일명
- `[options]` <sub>(객체)</sub>
  - `lastModified`
    - 마지막 수정 일시 <sub>(타임스탬프)</sub>

##### 2. 사용자 액션 <sub>(OS 정보 받음)</sub>
- `<input type="file">` <sub>(요소)</sub>
- 드래그 앤 드롭
  - `ondragend` <sub>(이벤트)</sub>
- 기타 브라우저 인터페이스

##### 추가 프로퍼티
- `name`
  - 파일명
- `lastModified`
  - 마지막 수정 일시 <sub>(타임스탬프)</sub>

##### `File` <sub>(객체)</sub> 생성 <sub>(사용자 액션)</sub>
- `<input type="file">` <sub>(요소)</sub>
```html
<input type="file" onchange="showFile(this)" />

<script>
function showFile(input) {
  let file = input.files[0];

  // 파일명
  // - ex) my.png
  alert(`File name: ${file.name}`);

  / 마지막 수정 일시 (타임스탬프)
  // - ex) 1552830408824
  alert(`Last modified: ${file.lastModified}`);
}
</script>
```

### [`FileReader`](https://www.w3.org/TR/FileAPI/#dfn-filereader) <sub>(객체)</sub>

##### `Blob` · `File` <sub>(객체)</sub> 데이터 읽기
- 디스크 읽기 시간 ↑
  - 이벤트 통해 데이터 전달

##### 생성자
```javascript
// 인수 X
let reader = new FileReader();
```

#### 주 메서드

##### `readAsArrayBuffer(blob)`
- 데이터 읽은 후 변환
  - 이진 포맷 <sub>(`ArrayBuffer` 객체)</sub>

##### `readAsText(blob, [encoding])`
- 데이터 읽은 후 변환
  - 문자열
- `[encoding]` <sub>(변환 인코딩 · 문자열)</sub>
  - `'utf-8'` <sub>(기본값)</sub>
  - 기타 등등

##### `readAsDataURL(blob)`
- 데이터 읽은 후 변환
  - 데이터 url <sub>(base64 인코딩 문자열)</sub>

##### `abort()`
- 동작 취소

#### 메서드 선택 <sub>(데이터 사용 용도)</sub>

##### `readAsArrayBuffer` <sub>(이진 포멧)</sub>
- 저수준 이진 작업
- 고수준 작업 <sub>(`slice` 등)</sub>
  - 읽기 없이 직접 호출 가능 <sub>(`Blob` 상속)</sub>

##### `readAsText` <sub>(문자열)</sub>
- 문자열 작업

##### `readAsDataURL` <sub>(데이터 url)</sub>
- url 필요 요소 대상 작업
  - `<img>` <sub>(`src` 속성)</sub>
  - `<a>` <sub>(`href` 속성)</sub>
  - 기타 등등
- 대안
  - `URL.createObjectURL`

##### 이벤트
- `loadstart`
  - 로드 시작
- `progress`
  -  읽기 중
- `load` <sub>(자주 사용)</sub>
  - 읽기 완료 <sub>(에러 미발생)</sub>
- `abort`
  - `abort()` <sub>(메서드)</sub> 호출
- `error` <sub>(자주 사용)</sub>
  - 에러 발생
- `loadend`
  - 읽기 완료 <sub>(성공 · 실패)</sub>

##### 결과 <sub>(읽기 완료)</sub> 프로퍼티
- `result`
  - 성공 결과
- `error` <sub>(에러 객체)</sub>
  - 실패 결과

##### 파일 읽기
```html
<input type="file" onchange="readFile(this)">

<script>
function readFile(input) {
  let file = input.files[0];
  let reader = new FileReader();

  // 읽은 후 문자열 변환
  reader.readAsText(file);

  reader.onload = function() {
    console.log(reader.result);
  };

  reader.onerror = function() {
    console.log(reader.error);
  };

}
</script>
```

요약

##### `File` <sub>(객체 · `Blob` 상속)</sub>
- `Blob` <sub>(객체)</sub> 프로퍼티 · 메서드 사용 가능

##### 추가 프로퍼티
- `name`
- `lastModified`

##### 추가 기능
- 파일 시스템 내 파일 읽기

##### `File` <sub>(객체)</sub> 얻기 <sub>(사용자 액션)</sub>
- `<input type="file">` <sub>(요소)</sub>
- 드래그 앤 드롭
  - `ondragend` <sub>(이벤트)</sub>

##### 읽기 메서드 <sub>(`File` · `Blob` 읽은 후 변환)</sub>
- `readAsArrayBuffer`
  - `ArrayBuffer` <sub>(객체)</sub>
- `readAsText`
  - 문자열
- `readAsDataURL`
  - data url <sub>(base64 인코딩)</sub>

##### 실무 대다수 경우
- 파일 내용 읽기 X
- 짧은 url 생성
  - `URL.createObjectURL`
  - 요소에 할당 <sub>(`<a>` · `<img>` 등)</sub>
    - 파일 다운로드
    - 이미지 표시 <sub>(`<canvas>` 등)</sub>
