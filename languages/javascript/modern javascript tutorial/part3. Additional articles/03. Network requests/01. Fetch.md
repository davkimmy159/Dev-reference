`fetch` <sub>(메서드)</sub>
====

##### 네트워크 요청 작업
- 주문 전송
- 사용자 정보 읽기
- 서버 내 최신 변경 정보 가져오기
- 등등

##### AJAX <sub>(Asynchronous JavaScript And XML)</sub>
- 서버 내 추가 정보 비동기적 요청 기술
- 포괄적인 기술 표현 용어

##### `fetch` <sub>(메서드)</sub>
- 모던 · 다재다능
- 대다수 모던 브라우저 지원
- 구식 브라우저 미지원
  - 폴리필 필요

#### 기본 문법
```javascript
let promise = fetch(url, [options]);
```

##### `url`
- 요청 url

##### `[options]` <sub>(객체)</sub>
- HTTP 메서드 · 헤더 등 지정
- 미전달 시 `GET` <sub>(HTTP 메서드)</sub> 요청 진행
  - `url` 내용 다운로드

##### 메서드 호출
- 네트워크 요청 전송
- `Promise` 반환
  - 호출 코드 내 사용

### 응답 <sub>(보통 2단계)</sub>

#### 1. 응답 헤더 도착

#####  `Promise` <sub>(`fetch` 메서드 반환값)</sub> 즉시 이행
- 이행 값
  - [`Response`](https://fetch.spec.whatwg.org/#response-class) <sub>(내장 클래스)</sub> 인스턴스

##### 현재 본문 <sub>(`body`)</sub> 도착 전 상태
- 응답 헤더 확인 가능
  - 요청 처리 성공 · 실패 확인

##### HTTP 요청 전송 불가능 시 <sub>(네트워크 문제 · 부재 사이트 접속 등)</sub>
- `Promise` <sub>(`fetch` 메서드 반환값)</sub> 거부

#### 응답 프로퍼티 <sub>(HTTP 상태)</sub>

##### `status`
- HTTP 상태 코드
  - ex\) `200` 등

##### `ok` <sub>(`boolean`)</sub>
- 요청 처리 성공 여부
- `true`
  - `200` ~ `299` <sub>(HTTP 상태 코드)</sub>
```javascript
let response = await fetch(url);

// 요청 성공
// - 200 ~ 299 (HTTP 상태 코드)
if (response.ok) {

  // 응답 몬문 받기
  let json = await response.json();

// 요청 실패
} else {
  alert("HTTP-Error: " + response.status);
}
```

#### 2. 응답 본문 수신

##### `response` <sub>([`Response`](https://fetch.spec.whatwg.org/#response-class) 내장 클래스 인스턴스)</sub>
- 다양한 본문 읽기 메서드 구현

|메서드|반환|비고|
|---|---|---|
|`text()`|텍스트 형태||
|`json()`|JSON <sub>(객체)</sub> 형태|본문 파싱|
|`formData()`|`FormData` <sub>(객체)</sub> 형태||
|`blob()`|`Blob` <sub>(객체)</sub> 형태|이진 데이터 <sub>(타입 有)</sub>|
|`arrayBuffer()`|`ArrayBuffer` <sub>(객체)</sub> 형태|이진 데이터 <sub>(저수준)</sub>|
|`body` <sub>(프로퍼티)</sub>|응답 본문 읽기 <sub>(청크 단위)</sub>|[`ReadableStream`](https://streams.spec.whatwg.org/#rs-class) <sub>(객체)</sub>|

##### GitHub 마지막 커밋 받기 <sub>(JSON 객체)</sub>
- `await` 사용 코드
```javascript
let url = 'https://api.github.com/repos/javascript-tutorial/ko.javascript.info/commits';
let response = await fetch(url);

// 응답 본문 읽고 파싱 (JSON 객체 형태)
let commits = await response.json();

alert(commits[0].author.login);
```
- `await` 미사용 코드 <sub>(프라미스만 사용)</sub>
```javascript
fetch('https://api.github.com/repos/javascript-tutorial/en.javascript.info/commits')
  .then(response => response.json())
  .then(commits => alert(commits[0].author.login));
```
- 텍스트 응답 받기
  - `.json()` X
  - `.text()`
```javascript
let response = await fetch('https://api.github.com/repos/javascript-tutorial/en.javascript.info/commits');

// 응답 본문 읽기 (텍스트 형태)
let text = await response.text();

alert(text.slice(0, 80) + '...');
```

##### [`fetch` 명세서](https://fetch.spec.whatwg.org/) 우상단 로고 <sub>(이진 데이터)</sub> 가져오기
```javascript
let response = await fetch('/article/fetch/logo-fetch.svg');

// 응답 다운로드 (Blob 객체)
let blob = await response.blob();

// <img> (요소) 생성
// - 다운로드받은 Blob (객체) 설정
let img = document.createElement('img');
img.style = 'position:fixed;top:10px;left:10px;width:100px';
document.body.append(img);

// 이미지 표시
img.src = URL.createObjectURL(blob);

// 3초 후 이미지 제거
setTimeout(() => {
  img.remove();
  URL.revokeObjectURL(img.src);
}, 3000);
```

### 응답 헤더

##### `response.headers` 내 저장
- `Map` X
- `Map` 유사 · 유사 메서드 지원
  - 헤더 일부 추출
  - 헤더 전체 순회
  - 기타 등등
```javascript
let response = await fetch('https://api.github.com/repos/javascript-tutorial/en.javascript.info/commits');

// 헤더 일부 추출
// - application/json; charset=utf-8
alert(response.headers.get('Content-Type'));

// 헤더 전체 순회
for (let [key, value] of response.headers) {
  alert(`${key} = ${value}`);
}
```

### `POST` 요청

##### 요청 전송
- `user` <sub>(객체)</sub> 직렬화 <sub>(JSON 문자열)</sub>
```javascript
let user = {
  name: 'John',
  surname: 'Smith'
};

let response = await fetch('/article/fetch/post/user', {
  method: 'POST',

  // json 헤더 설정
  headers: {
    'Content-Type': 'application/json;charset=utf-8'
  },
  body: JSON.stringify(user)
});

let result = await response.json();
alert(result.message);
```

##### 문자열 <sub>(요청 본문)</sub>
- `Content-Type` <sub>(헤더)</sub> 기본 설정
  - `text/plain;charset=UTF-8`

### 이미지 전송

##### `Blob` <sub>(객체)</sub> · `BufferSource` <sub>(가상 이진 처리 객체)</sub>
- `fetch` <sub>(메서드)</sub> 이진 데이터 전송

##### `<canvas>` <sub>(요소)</sub>
- 요소 상 마우스 이동
  - 이미지 생성
- '전송' 버튼
  - 이미지 서버 전송
```html
<body style="margin:0">
  <canvas id="canvasElem" width="100" height="80" style="border:1px solid"></canvas>

  <input type="button" value="전송" onclick="submit()">

  <script>
    canvasElem.onmousemove = function(e) {
      let ctx = canvasElem.getContext('2d');
      ctx.lineTo(e.clientX, e.clientY);
      ctx.stroke();
    };

    async function submit() {
      let blob = await new Promise(resolve => canvasElem.toBlob(resolve, 'image/png'));
      let response = await fetch('/article/fetch/post/image', {
        method: 'POST',
        body: blob
      });

      // 전송 성공 응답 도착
      // 이미지 사이즈 얼럿창 출력
      let result = await response.json();
      alert(result.message);
    }

  </script>
</body>
```

##### `Blob` <sub>(객체)</sub>
- 내장 타입 보유
  - `Content-Type` <sub>(헤더)</sub> 설정 필요 X
- 객체 타입 → `Content-Type` <sub>(헤더)</sub> 값
  - ex\) `toBlob(… , 'image/png')`

##### `submit()` <sub>(함수)</sub>
- `async` · `await` 미사용 코드
```javascript
function submit() {
  canvasElem.toBlob(function(blob) {
    fetch('/article/fetch/post/image', {
      method: 'POST',
      body: blob
    })
    .then(response => response.json())
    .then(result => alert(JSON.stringify(result, null, 2)))
  )}, 'image/png');
}
```

### Github 사용자 정보 가져오기

##### `getUsers(names)` <sub>(비동기 함수)</sub> 작성
- 인수
  - GitHub 사용자명 배열
- 해당 사용자 정보 가져오기
  - [GitHub API](https://api.github.com/users/사용자명) 요청 전송

##### 임의 함수 작성
- 받아온 사용자 정보 배열 반환

##### 조건
1. 사용자당 `fetch` <sub>(메서드)</sub> 요청 수
    - 1번
2. 각 요청
    - 타 요청 결과 대기 X <sub>(빠른 속도 보장)</sub>
3. 요청 실패 · 부재 사용자 정보 요청
    - `null` 반환 <sub>(결과 배열 내 추가)</sub>

##### 사용자 정보 가져오기
```javascript
fetch('https://api.github.com/users/사용자명');
```
- `200` <sub>(응답 상태 코드)</sub>
  - `json()`<sub>(메서드)</sub> 호출
    - 응답 본문 파싱
- 요청 실패 · 이외 상태 코드
  - `null` 반환 <sub>(결과 배열 내 추가)</sub>
```javascript
async function getUsers(names) {
  let jobs = [];

  for (let name of names) {
    let job = fetch(`https://api.github.com/users/${name}`).then(
      successResponse => {
        if (successResponse.status != 200) {
          return null;
        } else {
          return successResponse.json();
        }
      },
      failResponse => {
        return null;
      }
    );
    jobs.push(job);
  }

  let results = await Promise.all(jobs);

  return results;
}
```

##### `.then` 핸들러
- `fetch` <sub>(메서드)</sub> 실행 직후 호출
- 응답 도착 시
  - 타 `fetch` <sub>(메서드)</sub> 요청 대기 X
  - 즉시 응답 읽기 시작 <sub>(`json()` 메서드)</sub>

##### `Promise.all`
- 모든 `fetch` <sub>(메서드)</sub> 요청 응답 완료 대기
```javascript
await Promise.all(names.map(name => fetch(…)));
```

##### 각 `fetch` <sub>(메서드)</sub> 별 본문 메서드 <sub>(`json()`)</sub> 호출
- 타 `fetch` <sub>(메서드)</sub> 요청 응답 대기 X
- 하위 수준 프라미스 API
  - 여전히 유용
```javascript
let jobs = [];
for (let name of names) {

  // 개별 fetch (메서드) 본문 메서드 호출
  let job = fetch(…).then(…);
  jobs.push(job);
}
let results = await Promise.all(jobs);
```
