유니코드 <sub>(`u` 플래그 · `\p{…}` 클래스)</sub>
====

##### JS 문자열
- [유니코드 인코딩](https://en.wikipedia.org/wiki/Unicode) 사용

##### 대다수 문자
- 2바이트 인코딩 <sub>(65,536개 글자 표현)</sub>
  - 모든 글자 표현 불가능 <sub>(숫자 부족)</sub>

##### 일부 문자
- 4바이트 인코딩
  - `𝒳` <sub>(수학 x)</sub>
  - `😄` <sub>(웃는 표정)</sub>
  - 일부 상형 문자
  - 기타 등등

|문자|유니코드|바이트 수|
|:---:|---|:---:|
|`a`|`0x0061`|2|
|`≈`|`0x2248`|2|
|`𝒳`|`0x1d4b3`|4|
|`𝒴`|`0x1d4b4`|4|
|`😄`|`0x1f604`|4|

##### 과거 JS
- 단순한 유니코드 인코딩
  - 4바이트 문자 無
- 일부 언어 기능 <sub>(`length` 프로퍼티 등)</sub>
  - 현재까지도 적절히 처리 불가능

##### `length` <sub>(프로퍼티)</sub>
- 4바이트 문자 1개 → 2바이트 문자 2개 <sub>(취급)</sub>
  - 잘못된 결과
  - 하나로 묶어서 취급 필요
    - '서로게이트 쌍'
```javascript
alert('😄'.length); // 2
alert('𝒳'.length);  // 2
```

##### 정규표현식 <sub>(기본)</sub>
- 4바이트 문자 1개 → 2바이트 문자 2개 <sub>(취급)</sub>

##### `u` <sub>(플래그)</sub>
- 4바이트 문자 적절히 처리
- 유니코드 프로퍼티 사용해 검색 가능

<br />

##### 모든 유니코드 문자 내 다양한 프로퍼티 有
- 다양한 정보 저장
  - 속한 '범주' 설명
  - 기타 등등
- ex\) `Letter` <sub>(프로퍼티)</sub>
  - 어떤 언어 내 글자 의미
- ex\) `Number` <sub>(프로퍼티)</sub>
  - 숫자 <sub>(아라비아 · 한자 등)</sub>
- 기타 등등

##### `\p{…}`
- `u` <sub>(플래그)</sub> 필수
- 프로퍼티 사용해 문자 검색
  - ex\) `p{Letter}` · `p{L}`
    - 언어 글자 표기
- 대다수 프로퍼티
  - 약자 有
    - ex\) `L` <sub>(`Letter`)</sub>

##### 3종류 글자 검색
- 영문자
- 조지아 문자
- 한글
```javascript
let str = "A ბ ㄱ";

// A,ბ,ㄱ
alert( str.match(/\p{L}/gu) );

// null (불일치)
// - u (플래그) 無
alert( str.match(/\p{L}/g) );
```

#### 파생 범주 <sub>(일부)</sub>

##### `Alphabetic` <sub>(`Alpha`)</sub>
- `L` <sub>(Letter)</sub>
- `Nl` <sub>(문자 형태 숫자)</sub>
  - ex\) `Ⅻ` <sub>(로마 숫자)</sub> 등
- `Other_Alphabetic` <sub>(`OAlpha`)</sub>

##### `Hex_digit`
- 16진수 숫자
  - `0-9`
  - `a-f`

- 다양한 파생 범주 有

#### 16진수 검색

##### 16진수
- `xFF` <sub>(형식)</sub>
  - `F` <sub>(0 ~ 9 · A ~ F)</sub>

##### `\p{Hex_Digit}`
- 16진수 표기
```javascript
let regexp = /x\p{Hex_Digit}\p{Hex_Digit}/u;

// xAF
alert("number: xAF".match(regexp));
```

#### 한자 검색

##### `Script` <sub>(`sc` · 표기 체계)</sub>
- 값 <sub>([전체 목록](https://en.wikipedia.org/wiki/Script_(Unicode)))</sub>
  - `Cyrillic` <sub>(키릴 문자)</sub>
  - `Greek` <sub>(그리스 문자)</sub>
  - `Arabic` <sub>(아라비아 문자)</sub>
  - `Han` <sub>(한자)</sub>
  - 기타 등등

##### 특정 표기 체계 문자 검색
- `Script=<value>` <sub>(`sc=<value>`)</sub>
  - `\p{sc=Cyrillic}` <sub>(키릴 문자)</sub>
  - `\p{sc=Han}` <sub>(한자)</sub>
  - 기타 등등
```javascript
let str = `Hello Привет 你好 123_456`;

// 한자 반환
let regexp = /\p{sc=Han}/gu;

// 你,好
alert( str.match(regexp) );
```

#### 통화 검색

##### `\p{Currency_Symbol}` <sub>(`\p{Sc}` · 통화 단위)</sub>
- `$`
- `€`
- `¥`
- 기타 등등

##### '통화 단위 바로 뒤 숫자' 검색
```javascript
let str = `Prices: $2, €1, ¥9`;
let regexp = /\p{Sc}\d/gu;

// $2,€1,¥9
alert( str.match(regexp) );
```
