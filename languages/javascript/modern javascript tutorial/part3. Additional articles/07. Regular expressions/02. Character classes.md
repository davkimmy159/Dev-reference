문자 클래스
====

- 특정 집합 내 모든 기호 일치 표현

##### 숫자 클래스
- `\d`
  - 아무 숫자 하나

##### 전화번호 내 1번째 숫자 검색
```javascript
let str = "+7(903)-123-45-67";
let regexp = /\d/;

str.match(regexp); // 7
```

##### 전화번호 내 모든 숫자 검색
- `g` <sub>(플래그)</sub>
```javascript
let str = "+7(903)-123-45-67";
let regexp = /\d/g;

str.match(regexp); // 7,9,0,3,1,2,3,4,5,6,7

// 검색 결과 (배열) 이어붙이기
str.match(regexp).join(''); // 79035419441
```

#### 주요 문자 클래스

##### `\d` <sub>(숫자)</sub>
- `0` ~ `9`

##### `\s` <sub>(공백 기호)</sub>
- 스페이스
- `\t` <sub>(탭)</sub>
- `\n` <sub>(줄 바꿈)</sub>
- `\v`
- `\f`
- `\r`

##### `\w` <sub>(단어 구성 문자)</sub>
- 라틴 문자 <sub>(`A` ~ `Z` · `a` ~ `z`)</sub>
- 숫자 <sub>(`0` ~ `9`)</sub>
- `_` <sub>(밑줄)</sub>
- 非라틴 문자 X <sub>(키릴 · 힌디 등)</sub>

##### `\d\s\w`
- 숫자 1개 + 공백 문자 1개 + 단어 구성 문자 1개
  - ex\) `1 a`

##### 일반 기호 · 문자 클래스 조합
- `CSS\d`
  - 'CSS' <sub>(문자열)</sub> + 숫자 1개
```javascript
let str = "Is there CSS4?";
let regexp = /CSS\d/;

str.match(regexp); // CSS4
```

##### 다수 문자 클래스 사용
```javascript
// ' HTML5'
"I love HTML5!".match(/\s\w\w\w\w\d/);
```

### 반대 클래스

##### 해당 문자 외 아무 문자
- 아무 문자 클래스 대응 <sub>(대문자)</sub>

##### `\D` <sub>(숫자 외 · `\d` 반대)</sub>
- 일반 글자
- 기타 등등

##### `\S` <sub>(공백 외 · `\s` 반대)</sub>
- 일반 글자
- 기타 등등

##### `\W` <sub>(단어 구성 문자 외 · `\w` 반대)</sub>
- 非라틴 문자
- 공백
- 기타 등등

##### 전화번호 내 모든 숫자 검색
- `\D` <sub>(숫자 외)</sub>
```javascript
let str = "+7(903)-123-45-67";

// 숫자 외 제거
str.replace(/\D/g, ""); // 79031234567
```

### `.` <sub>(아무 문자)</sub>

##### `\n` <sub>(줄 바꿈 문자)</sub> 외 아무 문자
```javascript
"Z".match(/./); // Z
```
```javascript
let regexp = /CS.4/;

"CSS4".match(regexp); // CSS4
"CS-4".match(regexp); // CS-4
"CS 4".match(regexp); // CS 4 (공백 포함)
```
- 문자 부재 불일치
  - 일치 문자 필수
```javascript
// 일치 문자 無 → 검색 결과 無
"CS4".match(/CS.4/); // null (불일치)
//           CS4?
```

#### 모든 문자 검색 <sub>(`.` 아무 문자 · `s` 플래그)</sub>

##### `.` <sub>(아무 문자)</sub>
- `\n` <sub>(줄 바꿈 문자)</sub> X

##### `/A.B/`
- `A` · `B` 사이 아무 문자 <sub>(`\n` 제외)</sub>
```javascript
"A\nB".match(/A.B/); // null (불일치)
```

##### `.` <sub>(아무 문자)</sub> · `s` <sub>(플래그)</sub>
- `\n` <sub>(줄 바꿈 문자)</sub> 포함 아무 문자
  - 'dotall' 모드 활성화
```javascript
"A\nB".match(/A.B/s); // A\nB
```

##### 범용 대안
- `[\s\S]`
  - 공백 <sub>(`\s`)</sub> · 공백 외 <sub>(`\S`)</sub> 문자
- `[\d\D]`
  - 숫자 <sub>(`\d`)</sub> · 숫자 외 <sub>(`\D`)</sub> 문자
- `[^]`
  - 문자 부재 제외
- 기타 등등
```javascript
"A\nB".match(/A[\s\S]B/); // A\nB
```

##### 하이픈 구분 숫자 검색
- `\d-\d`
```javascript
"1 - 5".match(/\d-\d/); // null (불일치)
```
- `\d - \d` · `\d\s-\s\d`
```javascript
"1 - 5".match(/\d - \d/);   // 1 - 5
"1 - 5".match(/\d\s-\s\d/); // 1 - 5
```

##### 공백 == 문자
- 정규표현식 내 공백 추가 · 제거
  - 검색 동작 상이
