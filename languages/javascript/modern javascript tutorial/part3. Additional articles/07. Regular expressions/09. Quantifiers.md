수량자 <sub>(`{n}` · `?` · `+` · `*`)</sub>
====

### `{n}`

##### 사용법
- 타겟 뒤 추가
- 수량 · 범위 명시

#### `{n}` <sub>(수량)</sub>

##### `\d{5}`
- 숫자 5개
- `\d\d\d\d\d` 동일
```javascript
let regexp = /\d{5}/;

// 12345
alert( "I'm 12345 years old".match(regexp) );
```

##### `\b\d{5}\b`
- 숫자 5개 초과 시 불일치

#### `{n,[m]}` <sub>(범위)</sub>

##### `\d{3,5}`
- 숫자 3 ~ 5개
```javascript
let regexp = /\d{3,5}/;

// 1234
alert( "I'm not 12, but 1234 years old".match(regexp) );
```

##### `\d{3,}`
- 숫자 3개 이상
```javascript
let regexp = /\d{3,}/;

// 345678
alert( "I'm not 12, but 345678 years old".match(regexp) );
```

##### `\d{1,}`
- 숫자 1개 이상
```javascript
let str = "+7(903)-123-45-67";
let numbers = str.match(/\d{1,}/g);

// 7,903,123,45,67
alert(numbers);
```

### 축약형

#### `?` <sub>(`{0,1}`)</sub>
- 0 ~ 1개
- 선택 문법 유사

##### `colou?r`
- `color`
- `colour`
```javascript
let str = "Should I write color or colour?";

// color, colour
alert( str.match(/colou?r/g) );
```

#### `+` <sub>(`{1,}`)</sub>
- 1개 이상

##### `\d+`
- 숫자 1개 이상
```javascript
let str = "+7(903)-123-45-67";

// 7,903,123,45,67
alert( str.match(/\d+/g) );
```

#### `*` <sub>(`{0,}`)</sub>
- 0개 이상

##### `\d0*` vs `\d0+`
- `0` <sub>(숫자)</sub>
  - 0개 이상
  - 1개 이상
```javascript
// 100, 10, 1
alert( "100 10 1".match(/\d0*/g) );

// 100, 10
alert( "100 10 1".match(/\d0+/g) );
```

### 추가 예시

##### 수량자
- 자주 사용
- 복잡한 정규표현식 내 주요 블록 구성

##### `\d+\.\d+`
- 소수점 숫자
```javascript
// 12.345
alert( "0 1 12.345 7890".match(/\d+\.\d+/g) );
```

#### 여는 html 태그 <sub>(속성 無)</sub>

##### 1. `/<[a-z]+>/i` <sub>(단순)</sub>
1. `"<"`
2. 라틴 문자 1개 이상
3. `">"`
```javascript
// <body>
alert( "<body>…</body>".match(/<[a-z]+>/gi) );
```

##### 2. `/<[a-z][a-z0-9]*>/i` <sub>(개선)</sub>
- 태그 내 숫자 명시 가능 <sub>(맨 앞 제외)</sub>
1. `"<"`
2. 라틴 문자 <sub>(소문자)</sub> 1개
3. 라틴 문자 · 숫자 1개 이상
4. `">"`
```javascript
// <h1>
alert( "<h1>Hi!</h1>".match(/<[a-z][a-z0-9]*>/gi) );
```

#### 여는 · 닫는 html 태그 <sub>(속성 無)</sub>

##### `/<\/?[a-z][a-z0-9]*>/i`
- `\/?` <sub>(이스케이프 필요)</sub>
  - `/` <sub>(슬래시)</sub> 0 ~ 1개
```javascript
// <h1>, </h1>
alert( "<h1>Hi!</h1>".match(/<\/?[a-z][a-z0-9]*>/gi) );
```

### `...` <sub>(생략 부호)</sub>

##### 정규표현식 작성
- `...` <sub>(생략 부호)</sub> 검색
  - 연속된 점 3개 이상
```javascript
let regexp = /…/g;

// ..., .....
alert( "Hello!... How goes?.....".match(regexp) );
```

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

##### `/\.{3,}/g`
- `\.` <sub>(특수문자 이스케이프)</sub>
```javascript
let regexp = /\.{3,}/g;

// ..., .....
alert( "Hello!... How goes?.....".match(regexp) );
```

### HTML 색상

##### 정규표현식 작성
- HTML 내 사용 색상 검출
  - ex\) `#ABCDEF`
- `#` + 숫자 <sub>(16진수)</sub> 6개
```javascript
let str = "color:#121212; background-color:#AA00ef bad-colors:f#fddee #fd2 #12345678";
let regexp = /…/;

// #121212,#AA00ef
alert( str.match(regexp) );
```

##### 참고
- 타 색상 포맷 무시
  - `#123`
  - `rgb(1,2,3)`
  - 기타 등등

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

##### `#nnnnnn` <sub>(16진수 숫자)</sub>
- `/#[0-9a-fA-F]{6}/g` <sub>(`/#[0-9a-f]{6}/gi`)</sub>
  - `0` ~ `9`· `a` ~ `f` · `A` ~ `F`
```javascript
let str = "color:#121212; background-color:#AA00ef bad-colors:f#fddee #fd2"
let regexp = /#[a-f0-9]{6}/gi;

// #121212,#AA00ef
alert( str.match(regexp) );
```

##### 6자리 이상 숫자 검색
- 부적절한 결과
```javascript
// #123456
alert( "#12345678".match( /#[a-f0-9]{6}/gi ) );
```
- 끝에 `\b` 추가
```javascript
// #123456
alert( "#123456".match( /#[a-f0-9]{6}\b/gi ) );

// null (불일치)
alert( "#12345678".match( /#[a-f0-9]{6}\b/gi ) );
```
