형 변환
====

##### 형 변환
- 함수 · 연산자 전달 값 대다수
  - 적절한 자료형 자동 변환 발생
- 전달 값 의도적인 변환 가능
  - 적절한 자료형 명시적 변환
- ex\) `alert` <sub>(함수)</sub>
  - 전달 값 → 문자열 <sub>(자동 변환)</sub>
- ex\) 수학 연산자
  - 전달 값 → 숫자 <sub>(자동 변환)</sub>

### `string` 형 변환

##### 자동 형 변환
- `string` 값 필요 시 발생
- ex\) `alert` <sub>(함수)</sub> `string` 매개변수
  - 상이 타입 전달 시 자동 형 변환 발생

##### 명시적 형 변환
- `String(value)` <sub>(함수)</sub>
```javascript
let bool = true;

alert(typeof bool); // boolean

// true    → "true" (값 변환)
// boolean → string (형 변환)
let str = String(bool);

alert(typeof str); // string
```

### `number` 형 변환


##### 자동 형 변환
- 수학 관련 함수 · 표현식 사용 시 발생
```javascript
// 자동 형 변환 (string → number) 후 연산
alert( "6" / "2" ); // 3
```

##### 명시적 형 변환
- `Number(value)` <sub>(함수)</sub>
```javascript
let str = "123";

alert(typeof str); // string

// "123"  → 123    (값 변환)
// string → number (형 변환)
let num = Number(str);

alert(typeof num); // number
```

##### 문자 기반 폼 숫자 입력
- 명시적 형 변환 필수
  - `Number(str)` <sub>(`string` → `number`)</sub>

##### 숫자 외 글자 포함 문자열 → `number` 형 변환
- `NaN` 반환
```javascript
let age = Number("임의의 문자열 123");

alert(age); // NaN (형 변환 실패)
```

##### 형 변환 규칙

|값|형 변환|
|---|---|
|`undefined`|`NaN`|
|`null`|`0`|
|`true` · `false`|`1` · `0`|
|`string`|① 처음 · 끝 공백 제거|
||② ⓐ `0` <sub>(문자열 無)</sub>|
||② ⓑ<br />- 문자열 내 숫자 읽기 <sub>(가능한 만큼)</sub><br />- `NaN` <sub>(형 변환 실패 시)</sub>|

```javascript
alert( Number(true) );  // 1
alert( Number(false) ); // 0

// 처음 · 끝 공백 제거
alert( Number("   123   ") ); // 123

// NaN
// - "z" → 숫자 (변환) 실패
alert( Number("123z") );
```

### `boolean` 형 변환

##### 자동 형 변환
- 논리 연산 수행 시 발생

##### 명시적 형 변환
- `Boolean(value)` <sub>(함수)</sub>
```javascript
let num = 123;

alert(typeof num); // number

// 123    → true    (값 변환)
// number → boolean (형 변환)
let bool = Boolean(num);

alert(typeof bool); // boolean;
```

##### 형 변환 규칙

|값|형 변환|
|---|---|
|직관적으로 '비어있는' 값<br />- `0` <sub>(숫자)</sub><br /> - `""` <sub>(빈 문자열)</sub><br /> - `null` <br /> - `undefined`<br /> - `NaN`|`false`|
|이 외 값|`true`|

```javascript
alert( Boolean(1) ); // 1 (숫자) → true
alert( Boolean(0) ); // 0 (숫자) → false

// 문자열 → true
alert( Boolean("hello") );

// "" (빈 문자열) → false
alert( Boolean("") );
```

<br />

<img src="../../images/commons/icons/triangle-exclamation-solid.svg" /> **`"0"` <sub>(문자열)</sub> == `true` 주의**

##### 특정 언어 <sub>(PHP 등)</sub>
- `"0"` <sub>(문자열)</sub> == `false`

##### JS
- `"0"` <sub>(문자열)</sub> == `true`
- 비어있지 않은 문자열 == `true`

```javascript
alert( Boolean("0") ); // true

// 공백 문자열 == 비어있지 않은 문자열
alert( Boolean(" ") ); // true
```

요약
====

##### 자주 발생하는 형 변환
- `string`
- `number`
- `boolean`

#### `string` 형 변환

##### 보통 출력 관련
- 자동 형 변환
  - `string` 값 필요 시 발생
- 명시적 형 변환
  - `String(value)` <sub>(함수)</sub>

##### 원시 자료형 → `string` 형 변환
- 매우 명시적인 방식 발생
  - 대부분 결과 예상 가능

#### `number` 형 변환

##### 보통 수학 관련 연산
- 자동 형 변환
  - 수학 관련 함수 · 표현식 사용 시 발생
- 명시적 형 변환
  - `Number(value)` <sub>(함수)</sub>

##### 형 변환 규칙
- 본문 참조

#### `boolean` 형 변환

##### 보통 논리 연산
- 자동 형 변환
  - 논리 연산 수행 시 형 변환 발생
- 명시적 형 변환
  - `Boolean(value)` <sub>(함수)</sub>

##### 형 변환 규칙
- 본문 참조

##### 형 변환 예외
- `undefined` → `number`
  - `NaN`
- `"0"` <sub>(문자열)</sub> · `""` <sub>(빈 문자열)</sub> → `boolean`
  - `true`
