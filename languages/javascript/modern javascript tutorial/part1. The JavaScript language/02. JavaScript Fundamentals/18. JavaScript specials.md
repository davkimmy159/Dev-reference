### 코드 구조

#### 구문 구분

##### `;` <sub>(세미콜론)</sub> 기준
```javascript
alert('Hello'); alert('World');
```

##### 줄 바꿈 기준
```javascript
alert('Hello')
alert('World')
```
- `;` <sub>(세미콜론)</sub> 자동 삽입
  - 일부 경우 작동 X
```javascript
alert("이 메시지가 출력된 후에 에러가 발생합니다.")

[1, 2].forEach(alert)
```

##### `;` <sub>(세미콜론)</sub> 위치
- 문장 끝
  - 항상 붙이기 권장
- `{ … }` <sub>(코드 블록)</sub> · 블록 구성 문법 <sub>(반복문 등)</sub>
  - X
```javascript
function f() {
  // 함수 선언문 끝
  // - ; (세미콜론) 필요 X
}

for(;;) {
  // 반복문 끝
  // - ; (세미콜론) 필요 X
}
```

##### 여분의 `;` <sub>(세미콜론)</sub>
- 무시 <sub>(에러 발생 X)</sub>

### 상호작용

##### 브라우저 환경
- UI 함수 사용해 상호작용

##### `prompt(question, [default])` <sub>(함수)</sub>
- 프롬프트 창
  - `question` <sub>(매개변수)</sub> 내용
- '확인' 버튼
  - 사용자 입력값 반환
- '취소' 버튼
  - `null` 반환

##### `confirm(question)` <sub>(함수)</sub>
- 컨펌 대화상자
  - `question` <sub>(매개변수)</sub> 내용
- '확인' 버튼
  - `true` 반환
- 그 외
  - `false` 반환

##### `alert(message)` <sub>(함수)</sub>
- 얼럿 창 표시
  - `message` <sub>(매개변수)</sub> 내용

##### 공통점
- 모달 창 표시
  - 닫히기 전까지 코드 실행 중지
- 모달 창 외
  - 페이지 상 그 무엇과도 상호작용 X
```javascript
let userName = prompt("이름을 알려주세요.", "영희");
let isTeaWanted = confirm("차 한 잔 드릴까요?");

alert( "방문객: " + userName );         // 영희
alert( "차 주문 여부: " + isTeaWanted ); // true
```

### 연산자

##### 산술 연산자
- `*` · `+` · `-` · `/` <sub>(사칙 연산자)</sub>
- `%` <sub>(나머지 연산자)</sub>
- `**` <sub>(거듭제곱 연산자)</sub>

##### 이항 덧셈 연산자 <sub>(`+`)</sub>
- 피연산자 중 하나 : 문자열
  1. 나머지 하나 → 문자형 형 변환
  2. 두 피연산자 연결
```javascript
alert( '1' +  2  ); // '12', 문자열
alert(  1  + '2' ); // '12', 문자열
```

##### 할당 연산자
- `a = b` <sub>(일반)</sub>
- `a *= 2` <sub>(복합)</sub>

##### [비트 연산자](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators)
- 인수 → 정수 변환 후 이진 연산 수행

##### 조건부 연산자 <sub>(`?`)</sub>
- 매개변수 3개 유일
```javascript
/*
 condition
 - truthy
   - resultA 반환
 - falsy
   - resultB 반환
 */
condition ? result1 : result2
```

##### 논리 연산자
- `&&` <sub>(AND 연산자)</sub> · `||`<sub>(OR 연산자)</sub>
  - 단락 평가 수행
  - 평가 멈춘 시점 값 반환
- `!` <sub>(NOT 연산자)</sub>
  - 불린형 형 변환 후 역 반환

##### nullish 병합 연산자 <sub>(`??`)</sub>
- 실제 값 정의된 피연산자 찾기
```javascript
/*
 a == `null` · `undefined`
 - no
   - 평가 결과 : a
 - yes
   - 평가 결과 : b
 */
let value = a ?? b;
```

##### 비교 연산자
- `==` <sub>(동등 연산자)</sub>
  - 형 다른 값
    - 숫자형 형 변환
  - `null` · `undefined`
    - 자기끼리 : `true`
    - 타 자료형 : `falsy`
```javascript
alert( 0 == false ); // true
alert( 0 == '' );    // true
```
- `<` · `>` · `<=` · `>=` <sub>(기타 비교 연산자)</sub>
 - 숫자형 형 변환
- `===` <sub>(일치 연산자)</sub>
  - 형 변환 X
  - 형 불일치
    - 무조건 `false` 반환
- `null` · `undefined`
  - `==` <sub>(동등 연산자)</sub>
    - `true` 반환
  - 다른 값과 비교
    - 무조건 `false` 반환
- 문자열 비교
  - 글자 단위 비교
  - 타 자료
    - 숫자형 형 변환

##### 기타 연산자
- `,` <sub>(쉼표 연산자)</sub>
- 기타 등등

### 반복문
- `while`
```javascript
while (condition) {
  …
}
```
- `do…while`
```javascript
do {
  …
} while (condition);
```
- `for`
```javascript
for(let i = 0; i < 10; i++) {
  ...
}
```

- `for(let...)` 안쪽에 선언한 변수는 오직 반복문 내에서만 사용할 수 있습니다. `let`을 생략하고 기존에 선언되어있는 변수를 사용하는 것도 가능합니다.
- 지시자 `break`나 `continue`는 반복문 전체나 현재 실행 중인 반복을 빠져나가는 데 사용됩니다. 레이블은 중첩 반복문을 빠져나갈 때 사용합니다.

자세한 내용은 while과 for 반복문에서 살펴보시기 바랍니다.

자바스크립트는 이 외에도 객체를 다룰 수 있게 해주는 반복문도 제공합니다. 이에 대해선 추후에 학습하겠습니다.

### `'switch’`문
`'switch’`문은 `if`문을 사용해 재작성할 수 있습니다. `'switch’`문은 조건을 확인할 때 내부적으로 일치 연산자 `===`를 사용해 비교를 진행합니다.

예시:
```javascript
let age = prompt('나이를 알려주세요.', 18);

switch (age) {
  case 18:
    alert("Won't work"); // prompt 함수는 항상 문자열을 반환하므로, 이 case문엔 절대 도달할 수 없습니다.
    break;

  case "18":
    alert("낭랑 18세이시군요!");
    break;

  default:
    alert("어떤 case문에도 해당하지 않습니다.");
}
```

자세한 내용은 switch문에서 살펴보시기 바랍니다.

### 함수
세 가지 방법으로 함수를 만들 수 있습니다.

1. 함수 선언문: 주요 코드 흐름을 차지하는 방식
```javascript
function sum(a, b) {
  let result = a + b;

  return result;
}
```

2. 함수 표현식: 표현식 형태로 선언된 함수
```javascript
let sum = function(a, b) {
  let result = a + b;

  return result;
};
```

3. 화살표 함수:
```javascript
// 화살표(=>) 우측엔 표현식이 있음
let sum = (a, b) => a + b;

// 중괄호{ ... }를 사용하면 본문에 여러 줄의 코드를 작성할 수 있음. return문이 꼭 있어야 함.
let sum = (a, b) => {
  // ...
  return a + b;
}

// 인수가 없는 경우
let sayHi = () => alert("Hello");

// 인수가 하나인 경우
let double = n => n * 2;
```
- 함수는 지역 변수를 가질 수 있습니다. 지역 변수는 함수의 본문에 선언된 변수로, 함수 내부에서만 접근할 수 있습니다.
- 매개변수에 기본값을 설정할 수 있습니다. 문법은 다음과 같습니다. `function sum(a = 1, b = 2) {...}`
- 함수는 항상 무언가를 반환합니다. `return`문이 없는 경우는 `undefined`를 반환합니다.

자세한 내용은 함수와 화살표 함수 기본에서 살펴보시기 바랍니다.
