기본 문법 요약
====

### 코드 구조

#### 구문 구분

##### a. `;` <sub>(세미콜론)</sub> 기준
```javascript
alert('Hello'); alert('World');
```

##### b. 줄 바꿈 기준
- `;` <sub>(세미콜론)</sub> 자동 삽입
```javascript
alert('Hello')
alert('World')
```
- 자동 삽입 미작동 경우 有
```javascript
alert("이 메시지가 출력된 후에 에러가 발생합니다.")

// 에러
[1, 2].forEach(alert)
```

##### 문장 끝 `;` <sub>(세미콜론)</sub>
- 항상 명시 권장
- `{…}` <sub>(코드 블록)</sub> · 블록 구성 문법 <sub>(반복문 등)</sub>
  - 불필요
```javascript
// `;` (세미콜론) 불필요
function f() { … }
for (…;…;…) { … }
```

##### 여분 <sub>(불필요)</sub> `;` <sub>(세미콜론)</sub>
- 무시 <sub>(에러 X)</sub>

### 엄격 모드

- 위치

  - 함수 본문 최상단 <sub>(부분 적용)</sub>
```javascript
'use strict';

…
```

##### `'use strict'` <sub>(지시자)</sub> 無
- 스크립트 과거 방식 동작
  - '하위 호환성' 유지

##### 일부 모던 기능 <sub>(클래스 · 모듈 등)</sub>
- 엄격 모드 자동 활성화

### 변수

##### 선언 키워드
- `let` <sub>(모던)</sub>
- `const` <sub>(상수)</sub>
- `var` <sub>(구식)</sub>

##### 변수명 규칙
- 문자 · 숫자 <sub>(1번째 글자 X)</sub> 사용
- 일부 특수기호 사용 가능
  - `$` <sub>(달러)</sub>
  - `_` <sub>(밑줄)</sub>
- 非라틴계 언어 · 상형 문자 사용 가능 <sub>(지양)</sub>
  - 실무 드물게 사용

##### 동적 타이핑 <sub>(자료형 고정 X)</sub>
- 상이 자료형 값 할당 가능
```javascript
let x = 5;  // 숫자 할당
x = "John"; // 문자열 할당
```

##### JS 기본 자료형 <sub>(8가지)</sub>

|자료형|저장|
|---|---|
|`number`|정수 · 부동소수점|
|`BigInt`|아주 큰 숫자|
|`string`|문자열|
|`boolean`|논리값 <sub>(`true` · `false`)</sub>|
|`null`|`null` 값 <sub>('존재하지 않음' · '비어있음')</sub>|
|`undefined`|`undefined` 값 <sub>(값 미할당 상태)</sub>|
|객체형|복잡한 자료구조|
|`Symbol`|고유 식별자|

##### `typeof` <sub>(연산자)</sub>
- 값 자료형 반환 <sub>(문자열)</sub>
  - 예외 有 <sub>(`null` · 함수)</sub>
```javascript
/* 언어 자체 오류 */
// "object"
alert( typeof null );

// "function"
alert( typeof function(){} );
```

### 상호작용

##### 브라우저 환경
- UI 함수 사용해 상호작용

##### `prompt(question, [default])` <sub>(함수)</sub>
- 프롬프트 창
  - `question` <sub>(매개변수)</sub> 내용
- '확인' 버튼
  - 사용자 입력값 반환
- '취소' 버튼
  - `null` 반환

##### `confirm(question)` <sub>(함수)</sub>
- 컨펌 대화상자
  - `question` <sub>(매개변수)</sub> 내용
- '확인' 버튼
  - `true` 반환
- 이 외
  - `false` 반환

##### `alert(message)` <sub>(함수)</sub>
- 얼럿 창 표시
  - `message` <sub>(매개변수)</sub> 내용

##### 공통점
- 모달 창 표시
  - 닫히기 전까지 코드 실행 중지
- 모달 창 외
  - 페이지 상 그 무엇과도 상호작용 X
```javascript
let userName = prompt("이름을 알려주세요.", "영희");
let isTeaWanted = confirm("차 한 잔 드릴까요?");

// 영희
alert( "방문객: " + userName );

// true
alert( "차 주문 여부: " + isTeaWanted );
```

### 연산자

##### 산술 연산자
- `*` · `+` · `-` · `/` <sub>(사칙 연산자)</sub>
- `%` <sub>(나머지 연산자)</sub>
- `**` <sub>(거듭제곱 연산자)</sub>

##### 이항 덧셈 연산자 <sub>(`+`)</sub>
- 피연산자 중 하나 : 문자열
  1. 나머지 하나 → `string` 형 변환
  2. 두 피연산자 연결
```javascript
alert( '1' +  2  ); // '12' (문자열)
alert(  1  + '2' ); // '12' (문자열)
```

##### 할당 연산자
- `a = b` <sub>(일반)</sub>
- `a *= 2` <sub>(복합)</sub>

##### [비트 연산자](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators)
- 인수 → 정수 변환 후 이진 연산 수행

##### 조건부 연산자 <sub>(`?`)</sub>
- 매개변수 3개 유일
```javascript
/*
 condition
 - truthy
   - resultA 반환
 - falsy
   - resultB 반환
 */
condition ? result1 : result2
```

##### 논리 연산자
- `&&` <sub>(AND 연산자)</sub> · `||` <sub>(OR 연산자)</sub>
  - 단락 평가 수행
  - 평가 멈춘 시점 값 반환
- `!` <sub>(NOT 연산자)</sub>
  - `boolean` 형 변환 후 역 반환

##### nullish 병합 연산자 <sub>(`??`)</sub>
- 실제 값 정의된 피연산자 찾기
```javascript
/*
 a == `null` · `undefined`
 - no
   - 평가 결과 : a
 - yes
   - 평가 결과 : b
 */
let value = a ?? b;
```

##### 비교 연산자
- `==` <sub>(동등 연산자)</sub>
  - 형 다른 값
    - `number` 형 변환
  - `null` · `undefined`
    - 자기끼리 : `true`
    - 타 자료형 : `falsy`
```javascript
alert( 0 == false ); // true
alert( 0 == '' );    // true
```
- `<` · `>` · `<=` · `>=` <sub>(기타 비교 연산자)</sub>
 - `number` 형 변환
- `===` <sub>(일치 연산자)</sub>
  - 형 변환 X
  - 형 불일치
    - `false` 반환
- `null` · `undefined`
  - `==` <sub>(동등 연산자)</sub>
    - `true` 반환
  - 다른 값과 비교
    - `false` 반환
- 문자열 비교
  - 글자 단위 비교
  - 타 자료
    - `number` 형 변환

##### 기타 연산자
- `,` <sub>(쉼표 연산자)</sub>
- 기타 등등

### `switch` <sub>(문)</sub>
- `if` <sub>(문)</sub> 재작성
- 조건 확인
  - `===` <sub>(일치 연산자)</sub> 내부 사용
```javascript
let age = prompt('나이를 알려주세요.', 18);

switch (age) {

  // prompt (함수)
  // - 항상 문자열 반환
  //   - 해당 case (문) 도달 불가능
  case 18:
    alert("Won't work");
    break;

  case "18":
    alert("낭랑 18세이시군요!");
    break;

  default:
    alert("어떤 case문에도 해당하지 않습니다.");
}
```

### 함수

##### 함수 생성 방법 <sub>(3가지)</sub>
1. 함수 선언문
    - 주요 코드 흐름 차지
```javascript
function sum(a, b) {
  let result = a + b;

  return result;
}
```
2. 함수 표현식
    - 표현식 형태
```javascript
let sum = function(a, b) {
  let result = a + b;

  return result;
};
```

3. 화살표 함수
```javascript
// '=>' (화살표) 우측
// - 표현식 위치
let sum = (a, b) => a + b;

// `{…}` (중괄호)
// - 본문 여러 줄 작성 가능
// - return (문) 필수
let sum = (a, b) => {
  …
  return a + b;
}

// 인수 無
let sayHi = () => alert("Hello");

// 인수 1개
let double = n => n * 2;
```

#### 특징

##### 지역 변수
- 함수 본문 선언 변수
- 함수 내부에서만 접근 가능

#### 매개변수
- 기본값 설정 가능
```javascript
function sum(a = 1, b = 2) { … }
```

##### 항상 무언가 반환
- `return` <sub>(문)</sub> 無
  - `undefined` 반환
