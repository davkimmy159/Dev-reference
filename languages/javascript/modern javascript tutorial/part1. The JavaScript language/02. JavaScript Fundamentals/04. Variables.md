변수 · 상수
====

```javascript
let m1;
m1 = 'Hello!';
let m2 = 'Hello!';

// 미권장
let m3 = 'John', m4 = 25, m5 = 'Hello';

// 권장
let m6 = 'John';
let m7 = 25;
let m8 = 'Hello';

// 취향
let m9 = 'John',
  m10 = 25,
  m11 = 'Hello';
let m12 = 'John'
  , m13 = 25
  , m14 = 'Hello';
```

<img src="../../images/commons/icons/triangle-exclamation-solid.svg" /> **변수 두 번 선언 : 에러**

- 변수 한 번만 선언
- 동일 변수 다수 선언 시 에러
```javascript
let message = "This";

// let 반복 시 에러
// - SyntaxError: 'message' has already been declared
let message = "That";
```

### 대문자 상수

## <img src="../../images/commons/icons/circle-check-solid.svg" /> 과제

### 변수 가지고 놀기
1. 변수 선언
    - `admin`
    - `name`
2. `name` <sub>(변수)</sub> 값 할당
    - `"John"`
3. `name` <sub>(변수)</sub> 값 복사
    - `admin` <sub>(변수)</sub>
4. `admin` <sub>(변수)</sub> 값
    - `alert` 창 표시
    - 결과 : `"John"` 출력

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

```javascript
// 변수 2개 동시 선언
let admin, name;

name = "John";
admin = name;

alert( admin ); // "John"
```

### 올바른 이름 선택하기

##### 변수 생성
- 변숫값
  - 현재 살고있는 행성명

##### 데이터 저장 변수 생성 <sub>(웹사이트 개발 중 가정)</sub>
- 현재 접속 중인 사용자 <sub>(`user`)</sub> 이름 <sub>(`name`)</sub>

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

#### 행성 변수명
- 간단한 이름
  - 의미 모호
```javascript
let planet = "Earth";
```
- 변수명 길이 적당 시
  - 설명 좀 더 곁들어주기
```javascript
let ourPlanetName = "Earth";
```

#### 현재 방문자 표시 변수명
- 짧은 이름
  - 너무 짧음
```javascript
let userName = "John";
```
- '현재 접속 중인' 사용자 정보 표현
```javascript
let currentUserName = "John";
```

### 대문자 상수 올바르게 사용하기
```javascript
const birthday = '18.04.1982';
const age = someCode(birthday);
```

##### `birthday` <sub>(상수)</sub>
- 태어난 일자 정보

##### `age` <sub>(상수)</sub>
- 나이 값
  - `birthday` <sub>(상수)</sub> 조작하여 값 도출

##### `birthday` · `age` <sub>(상수)</sub>
- 대문자 상수 적절성
```javascript
const BIRTHDAY = '18.04.1982';
const AGE = someCode(BIRTHDAY);
```

<img src="../../images/commons/icons/circle-answer.svg" />

##### 대문자 상수
- '하드 코딩한' 값 별칭
  - 실행 전 이미 값 알고 있음
  - 코드에서 직접 그 값 사용

##### `birthday` <sub>(상수)</sub>
- 대문자 상수 조건 만족

##### `age` <sub>(상수)</sub>
- 대문자 상수 조건 불만족
  - 런타임에 평가 · 계산
