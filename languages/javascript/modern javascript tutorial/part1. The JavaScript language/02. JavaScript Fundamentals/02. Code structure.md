코드 구조
========

### 문

##### 문법 구조 · 명령어
- 어떤 작업 수행
  - ex&#41; `alert('Hello, world!')` <sub>(문)</sub>
- 원하는 만큼 문 작성 가능
- 서로 다른 문 구분
  - `;` <sub>(세미콜론)</sub>

아래 코드는 'Hello World’를 두 개의 alert 문으로 나눈 예시입니다.

alert('Hello'); alert('World');
코드의 가독성을 높이기 위해 각 문은 서로 다른 줄에 작성하는 것이 일반적입니다.

alert('Hello');
alert('World');
