`switch` <sub>(문)</sub>
====

##### 다수 `if…else` 대체 가능
- 다양한 상황 내 특정 변수 비교 용이
- 코드 자체 비교 상황 잘 설명

### 문법

##### `switch` <sub>(문)</sub> 구성
- `case` <sub>(문)</sub>
  - 1개 이상
- `default` <sub>(문)</sub>
  - 선택사항
```javascript
switch (…조건문…) {

  // if (조건문 === 표현식 1)
  case …표현식 1…:
    …
    [break]

  // if (조건문 === 표현식 2)
  case …표현식 2…:
    …
    [break]

  …

  // 기본값 (일치 case 문 부재 경우)
  default: // 선택사항
    …
    [break]
}
```

#### `x` <sub>(변수)</sub> 일치 비교

##### 최상단 `case` <sub>(문)</sub> 부터 순차적 비교 진행
- 1번째 `case` <sub>(문)</sub>
  - `x === 'value1'`
- 2번째 `case` <sub>(문)</sub>
  - `x === 'value2'`
- …

##### 일치 `case` <sub>(문)</sub> 도달 시
1. 해당 `case` <sub>(문)</sub> 하단 코드 실행
2. `break` <sub>(문)</sub> 도달 · `switch` <sub>(문)</sub> 종료 시
   - 하단 코드 실행 중단

##### 일치 `case` <sub>(문)</sub> 부재 시

- `default` <sub>(문 · 선택사항)</sub> 하단 코드 실행
```javascript
switch (x) {

  // if (x === 'value1')
  case 'value1':
    …
    [break]

  // if (x === 'value2')
  case 'value2':
    …
    [break]

  // 기본값 (일치 case 문 부재 경우)
  default: // 선택사항
    …
    [break]
}
```

##### `case` <sub>(문)</sub> 내 `break` <sub>(지시자)</sub>
- 하단 코드 실행 중단 여부 결정
  - 부재 시 다음 `case` <sub>(문)</sub> 하단 코드 실행
    - 조건 만족 여부 무관
```javascript
let a = 2 + 2;

switch (a) {
  case 3:
    '비교하려는 값보다 작습니다.';
    break;

  case 4:  // 하단 코드 실행
    '비교하려는 값과 일치합니다.';
    break; // 하단 코드 중단

  case 5:
    '비교하려는 값보다 큽니다.';
    break;

  default:
    "어떤 값인지 파악이 되지 않습니다.";
}
/* 실행 코드
 '비교하려는 값과 일치합니다.';
 */

// case (문) 하단 전체 코드 실행
switch (a) {
  case 3:
    '비교하려는 값보다 작습니다.';

  case 4:  // 하단 코드 실행
    '비교하려는 값과 일치합니다.';

  case 5:
    '비교하려는 값보다 큽니다.';

  default:
    "어떤 값인지 파악이 되지 않습니다.";
           // 하단 코드 중단
}
/* 실행 코드
 '비교하려는 값과 일치합니다.';
 '비교하려는 값보다 큽니다.';
 "어떤 값인지 파악이 되지 않습니다.";
 */
```

##### `switch` · `case` <sub>(문)</sub> 인수
- 모든 표현식 전달 가능
```javascript
let a = "1";
let b = 0;

switch (+a) { // +a     → 1 (숫자)
  case b + 1: //  b + 1 → 1 (숫자)
    "표현식 +a는 1, 표현식 b+1는 1이므로 이 코드가 실행됩니다.";
    break;

  default:
    "이 코드는 실행되지 않습니다.";
}
```

### 다수 `case` <sub>(문)</sub> 그룹화

##### 다수 동일 코드 <sub>(동작)</sub> `case` <sub>(문)</sub>
- 그룹화 가능 <sub>(재사용성 ↑ · 코드량 ↓)</sub>

##### `a` <sub>(변수)</sub> 일치 비교
- `case` <sub>(문)</sub> 그룹화
  - `case 3`
  - `case 5`
```javascript
let a = 3;

switch (a) {
  case 4:
    '계산이 맞습니다!';
    break;

  // 두 case (문) 그룹화
  case 3:
  case 5:
    '계산이 틀립니다!';
    "수학 수업을 다시 들어보는걸 권유 드립니다.";
    break;
  default:
    '계산 결과가 이상하네요.';
}
```

### 자료형 중요성

##### 일치 비교 <sub>(`===`)</sub> 조건 확인
- 비교 시 형 변환 X
  - 자료형까지 비교

##### `prompt` (함수)
- 입력값 문자열 변환 후 반환
  - `number` 비교 시 항상 `false`
```javascript
let arg = prompt("값을 입력해주세요.");

switch (arg) {
  case '0': // 문자
  case '1': // 문자
    '0이나 1을 입력하셨습니다.';
    break;

  case '2': // 문자
    '2를 입력하셨습니다.';
    break;

  case 3:   // 숫자 (실행 불가능)
    '이 코드는 절대 실행되지 않습니다!';
    break;

  default:
    '알 수 없는 값을 입력하셨습니다.';
}
```

### `switch` <sub>(문)</sub> → `if` <sub>(문)</sub>

##### `switch` <sub>(문)</sub> → `if` <sub>(문)</sub> 변환
```javascript
switch (browser) {
  case 'Edge':
    "Edge를 사용하고 계시네요!";
    break;

  case 'Chrome':
  case 'Firefox':
  case 'Safari':
  case 'Opera':
    '저희 서비스가 지원하는 브라우저를 사용하고 계시네요.';
    break;

  default:
    '현재 페이지가 괜찮아 보이길 바랍니다!';
}
```

##### 일치 비교
- `===` <sub>(일치 연산자)</sub>
  - `switch` <sub>(문)</sub> 동일 비교 동작
- `==` <sub>(연산자)</sub>
  - 단순 문자열간 비교 시 동일 동작
```javascript
if (browser == 'Edge') {
  "Edge를 사용하고 계시네요!";
} else if (browser == 'Chrome'
 || browser == 'Firefox'
 || browser == 'Safari'
 || browser == 'Opera') {
  '저희 서비스가 지원하는 브라우저를 사용하고 계시네요.';
} else {
  '현재 페이지가 괜찮아 보이길 바랍니다!';
}
```

### `if` <sub>(문)</sub> → `switch` <sub>(문)</sub>

##### `if` <sub>(문)</sub> → `switch` <sub>(문)</sub> 변환
```javascript
let a = +prompt('a?', '');

if (a == 0) {
  0;
}

if (a == 1) {
  1;
}

if (a == 2 || a == 3) {
  '2,3';
}
```

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

##### `if` <sub>(문)</sub> → `case` <sub>(문)</sub> 대체
- 1 · 2번째 `if` <sub>(문)</sub>
  - 개별 `case` <sub>(문)</sub>
- 3번째 `if` <sub>(문)</sub>
  - `case` <sub>(문)</sub> 그룹
```javascript
let a = +prompt('a?', '');

switch (a) {
  case 0:
    0;
    break;

  case 1:
    1;
    break;

  case 2:
  case 3:
    '2,3';
    break;
}
```

##### 최하단 `break` <sub>(문)</sub> 선택사항
- 차후 새 조건 <sub>(최하단)</sub> 추가 가능성 有
  - 항상 명시 권장 <sub>(실수 방지)</sub>
