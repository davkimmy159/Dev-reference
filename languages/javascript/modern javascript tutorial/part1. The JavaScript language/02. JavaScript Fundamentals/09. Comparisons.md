비교 연산자
====

##### 비교 연산

|비교|의미|
|---|:---:|
|`a > b`|초과|
|`a < b`|미만|
|`a >= b`|이상 <sub>(≥)</sub>|
|`a <= b`|이하 <sub>(≤)</sub>|
|`a == b`|동등 <sub>(=)</sub>|
|`a != b`|부동 <sub>(≠)</sub>|
|`a === b`|일치 <sub>(=)</sub>|
|`a !== b`|불일치 <sub>(≠)</sub>|

### `boolean` 반환

##### 비교 결과
- `true`
  - '긍정'
  - '참'
  - '사실'
- `false`
  - '부정'
  - '거짓'
  - '사실이 아님'
```javascript
alert( 2 >  1 ); // true
alert( 2 == 1 ); // false
alert( 2 != 1 ); // true

// 비교 결과 변수 할당 가능
let result = 5 > 4;

alert( result ); // true
```

### 문자열 비교

##### 사전<sub>(편집)</sub>순 문자열 비교
- 사전 앞쪽 문자열 \< 사전 뒤쪽 문자열 <sub>(크기)</sub>
```javascript
alert( 'A'    < 'Z' );    // true
alert( 'Glee' < 'Glow' ); // true
alert( 'Be'   < 'Bee' );  // true
```

##### 비교 알고리즘 <sub>(두 문자열 글자 간 비교)</sub>
1. 1번째 글자 비교
   - 상이 시 비교 종료
     - 글자 크기 → 문자열 크기
   - 동일 시 다음 글자 진행
2. 2번째 글자 비교
   - …
3. n번째 글자 비교
   - 글자 간 비교 끝날 때까지 반복
4. 비교 종료
   - 문자열 길이 → 문자열 크기
     - 동일 → 동일 문자열
     - 상이 → 크기 비교 확정

##### `A` vs `Z`
- 1번째 글자 비교 종료 <sub>(문자열 크기 결정)</sub>

##### `Glee` vs `Glow`
- 3번째 글자 비교 종료 <sub>(문자열 크기 결정)</sub>
  - `e` \< `o` <sub>(`Glee` \< `Glow`)</sub>

### 상이 자료형 비교

##### 타 자료형 → `number` 형 변환
```javascript
// '2'  (문자열) → 2 (숫자)
alert( '2'  >  1 );  // true

// '01' (문자열) → 1 (숫자)
alert( '01' == 1 );  // true

// true  → 1 (숫자)
alert( true  == 1 ); // true

// false → 0 (숫자)
alert( false == 0 ); // true
```

<br />

<img src="../../images/commons/icons/circle-exclamation-solid.svg" /> **흥미로운 상황**

##### `==` <sub>(동등 비교)</sub> vs `Boolean(…)` <sub>(논리 평가)</sub>
- `==` <sub>(동등 비교 연산자)</sub>
  - 피연산자 자동 형 변환 <sub>(`number`)</sub>
- `Boolean(…)` <sub>(`boolean` 생성자)</sub>
  - 인수 명시적 형 변환 <sub>(`boolean`)</sub>
```javascript
let a = 0;

//  0  (숫자)  → false
alert( Boolean(a) ); // false

let b = "0";

// "0" (문자열) → true
alert( Boolean(b) ); // true

// "0" (문자열) → 0 (숫자)
// - 0 == 0
alert(a == b);       // true
```

<br />

### `==` vs `===` <sub>(동등 vs 일치 연산자)</sub>

##### `==` <sub>(동등 연산자)</sub>
- falsy 값 구별 불가능
  - `-0` <sub>(숫자)</sub>
  - `""` <sub>(빈 문자열)</sub>
  - 기타 등등
- 타 자료형 비교 시
  - `number` 형 자동 변환

|falsy 값|형 변환|
|---|---|
|`false`|`0` <sub>(숫자)</sub>|
|`-0` <sub>(숫자)</sub>|`0` <sub>(숫자)</sub>|
|`""` <sub>(빈 문자열)</sub>|`0` <sub>(숫자)</sub>|

```javascript
alert( 0  == false ); // true
alert( '' == false ); // true
```

##### `===` <sub>(일치 연산자)</sub>
- 형 변환 없이 값 비교
  - 자료형 동등 여부까지 검사
- 비교 결과 명확
  - 에러 확률 ↓
```javascript
// false
// - 피연산자 형 다름
alert( 0 === false );
```

#### `!=` <sub>(부동 연산자)</sub> vs `!==` <sub>(불일치 연산자)</sub>
- `==` · `===` <sub>(동등 · 일치 연산자)</sub> 관계 비슷

### `null` · `undefined` 비교

##### `==` <sub>(동등 연산자)</sub>
- '특별한 규칙' 적용 <sub>('각별한 커플' 취급)</sub>
  - `true` 반환
```javascript
alert( null == undefined ); // true
```

##### `===` <sub>(일치 연산자)</sub>
- 두 값 자료형 다름
  - `false` 반환
```javascript
alert( null === undefined ); // false
```

##### 산술 · 기타 비교 <sub>(`<` · `>` · `<=` · `>=`)</sub> 연산자
- `number` 형 변환

|값|형 변환 후|
|---|---|
|`null`|`0` <sub>(숫자)</sub>|
|`undefined`|`NaN`|

#### `null` vs `0`

##### 기타 비교 <sub>(`<` · `>` · `<=` · `>=`)</sub> 연산자
- `null` → `0` <sub>(`number`)</sub> 변환

##### `==` <sub>(동등 연산자)</sub>
- `null` · `undefined`
  - 형 미변환
  - 서로 비교 시
    - `true` 반환
- 이 외 경우 <sub>(다른 값과 비교)</sub>
  - `false` 반환
```javascript
alert( null >  0 ); // false
alert( null >= 0 ); // true
alert( null == 0 ); // false
```

#### 비교 불가능한 `undefined`
- 다른 값과 비교 불가능

##### 기타 비교 <sub>(`<` · `>` · `<=` · `>=`)</sub> 연산자
- `undefined` → `NaN` <sub>(`number`)</sub> 변환
- 피연산자 `NaN` 일 경우
  - 항상 `false` 반환

##### `==` <sub>(동등 연산자)</sub>
- 항상 `null` · `undefined` 일치
- 이 외 값 불일치
```javascript
alert( undefined >  0 ); // false
alert( undefined <  0 ); // false
alert( undefined == 0 ); // false
```

#### 함정 피하기

##### 비교 연산자 <sub>(`===` 일치 연산자 제외)</sub>
- `null` · `undefined` 대상 연산 방지

##### `null` · `undefined` 가능 변수
- 피연산자 되지 않도록 주의
- 별도 처리 코드 추가
```javascript
 0        ==  false     // true , 형 변환
-0        ==  false     // true , 형 변환
''        ==  false     // true , 형 변환
 0        === false     // false, 형 변환 X
-0        === false     // false, 형 변환 X
''        === false     // false, 형 변환 X

null      ==  undefined // true , null · undefined 형 변환 X
null      === undefined // false, 형 변환 X

null      >   0         // false, null → 0 (number)
null      >=  0         // true , null → 0 (number)
null      <   0         // false, null → 0 (number)
null      <=  0         // true , null → 0 (number)
null      ==  0         // false, null · undefined 형 변환 X
null      ==  undefined // true , null · undefined 형 변환 X

undefined >  0          // false, undefined → NaN (number)
undefined <  0          // false, undefined → NaN (number)
undefined == 0          // false, null · undefined 형 변환 X
undefined == null       // true , null · undefined 형 변환 X

/*
산술 · 기타 비교 연산자
- null → 0
- undefined → NaN

비교 연산자 (=== 일치 연산자 제외)
- `null` · `undefined` 대상 연산 방지
*/
```

요약
====

##### 비교 연산자
- `boolean` 값 반환

##### 문자열 비교
- 문자 단위
  - 사전 순서 <sub>(비교 기준)</sub>

##### 서로 다른 타입 값 비교
- `number` 형 변환 <sub>(`===` 일치 연산자 제외)</sub>

##### `null` · `undefined`
- 동등 비교 <sub>(`==`)</sub>
  - 동일
  - 다른 값
    - 상이

##### `null` · `undefined` 가능성 있는 변수
- 비교 연산자 <sub>(`>` · `<` 등)</sub> 의 피연산자 시
  - 주의 요망
- 별도 확인 코드 추가 <sub>(습관화 필요)</sub>

## 비교
- 표현식들 결과 예측
```javascript
5 > 4
"apple" > "pineapple"
"2" > "12"
undefined == null
undefined === null
null == "\n0\n"
null === +"\n0\n"
```

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

```javascript
// 명백히 true
5 > 4 → true

// 문자열 비교
// - 사전순서 기준
//   - "a" < "p"
"apple" > "pineapple" → false

// 두 문자열 피연산자
// - 사전순 비교 진행
//   - "2" (좌측 1번째 글자) > "1" (우측 1번째 글자)
"2" > "12" → true

// null == undefined
// - == (동등 연산자)
//   - 특별 규칙
undefined == null → true

// null !== undefined
// - === (일치 연산자)
//   - 타입 확인
undefined === null → false

// null
// - 전부 부동 (undefined 제외)
null == "\n0\n" → false

// null
// - === (일치 연산자)
//   - 타입 확인
null === +"\n0\n" → false
```
