비교 연산자
==========

- `a > b`
- `a < b`
- `a >= b`
- `a <= b`
- `a == b` &nbsp;&nbsp;<sub>(동등)</sub>
- `a != b` &nbsp;&nbsp;<sub>(부동)</sub>
- `a === b` <sub>(일치)</sub>
- `a !== b` <sub>(불일치)</sub>

### 불린형 반환

##### `true`
- '긍정'
- '참'
- '사실'

##### `false`
- '부정'
- '거짓'
- '사실이 아님'
```javascript
alert( 2 >  1 ); // true
alert( 2 == 1 ); // false
alert( 2 != 1 ); // true

// 비교 결과 변수에 할당
let result = 5 > 4;
alert( result ); // true
```

### 문자열 비교

##### '사전' 순 문자열 비교
- 사전편집순
- 사전 앞쪽 문자열 \< 사전 뒤쪽 문자열
```javascript
alert( 'A'    < 'Z' );    // true
alert( 'Glee' < 'Glow' ); // true
alert( 'Be'   < 'Bee' );  // true
```

##### 과정
1. 두 문자열 첫 글자 비교
2. 첫 글자 <sub>(전자 문자열)</sub> \> <sub>(\<)</sub> 첫 글자 <sub>(후자 문자열)</sub>
    - **결론 : 전자 문자열 \> <sub>(\<)</sub> 후자 문자열**
    - 비교 종료
3. 두 문자열 첫 글자 동일
    - 2번째 글자 동일 방식 비교
4. 글자 간 비교 끝날 때까지 과정 반복
5. 비교 종료
    - **결론 : 두 문자열 길이**
      - **동일 : 두 문자열 동일**
      - **상이 : 더 긴 문자열 → 더 큼**

##### 대 · 소문자 구별
- `'A'` \< `'a'`

### 다른 형 값 간 비교

##### 자료형 다르면 숫자형 변환
```javascript

// true
// - '2' → 2
alert( '2'  >  1 );

// true
// - '01'→ 1
//   - 문자열 → 숫자 (변환)
alert( '01' == 1 );

/* 불린값 */
// true
// - true  → 1
alert( true  == 1 );

// true
// - false → 0
//   - 불린 값 → 숫자 (변환)
alert( false == 0 );
```

<img src="../../images/commons/icons/circle-exclamation-solid.svg" /> **흥미로운 상황**

##### `==` <sub>(동등 연산자)</sub>
- 자동 변환 <sub>(숫자형)</sub>
- 피연산자 형 변환

##### `Boolean(a)`
- 명시적 변환 <sub>(불린형)</sub>
- 인수 형 변환 X
```javascript
let a = 0;
alert( Boolean(a) ); // false

let b = "0";
alert( Boolean(b) ); // true

alert(a == b);       // true
```

<br />

### `==` <sub>(동등 연산자)</sub> vs `===` <sub>(일치 연산자)</sub>

##### `==` <sub>(동등 연산자)</sub>
- falsy 값 구별 X
  - `-0` <sub>(숫자)</sub>
  - `""` <sub>(빈 문자열)</sub>
  - `false`
- 형 다른 피연산자 비교 시
  - 숫자형 변환

|값|형 변환 후|
|---|---|
|`""` <sub>(빈 문자열)</sub>|`0` <sub>(숫자)</sub>|
|`false`|`0` <sub>(숫자)</sub>|

```javascript
alert( 0  == false ); // true
alert( '' == false ); // true
```

##### `===` <sub>(일치 연산자)</sub>
- 형 변환 없이 값 비교
  - 자료형 동등 여부까지 검사
- 비교 결과 명확
  - 에러 발생 확률 ↓
```javascript
// false
// - 피연산자 형 다름
alert( 0 === false );
```

#### `!=` <sub>(부동 연산자)</sub> vs `!==` <sub>(불일치 연산자)</sub>
- `==` · `===` <sub>(동등 · 일치 연산자)</sub> 관계 비슷

### `null` · `undefined` 비교

##### `==` <sub>(동등 연산자)</sub>
- '특별한 규칙' 적용 <sub>('각별한 커플' 취급)</sub>
  - `true` 반환
```javascript
alert( null == undefined ); // true
```

##### `===` <sub>(일치 연산자)</sub>
- 두 값 자료형 다름
  - `false` 반환
```javascript
alert( null === undefined ); // false
```

##### 산술 · 기타 비교 <sub>(`<` · `>` · `<=` · `>=`)</sub> 연산자
- 숫자형 변환

|값|형 변환 후|
|---|---|
|`null`|`0` <sub>(숫자)</sub>|
|`undefined`|`NaN`|

#### `null` vs `0`

##### 기타 비교 <sub>(`<` · `>` · `<=` · `>=`)</sub> 연산자
- `null` → `0` <sub>(숫자형)</sub> 변환

##### `==` <sub>(동등 연산자)</sub>
- `null` · `undefined`
  - 형 변환 X
  - 서로 비교 시
    - `true` 반환
- 그 외 경우 <sub>(다른 값과 비교)</sub>
  - `false` 반환
```javascript
alert( null >  0 ); // false
alert( null >= 0 ); // true
alert( null == 0 ); // false
```

#### 비교 불가능한 `undefined`
- 다른 값과 비교 X

##### 기타 비교 <sub>(`<` · `>` · `<=` · `>=`)</sub> 연산자
- `undefined` → `NaN` <sub>(숫자형)</sub> 변환
- 피연산자 `NaN` 일 경우
  - 항상 `false` 반환

##### `==` <sub>(동등 연산자)</sub>
- 항상 `null` · `undefined` 일치
- 그 외 값 불일치

```javascript
alert( undefined >  0 ); // false
alert( undefined <  0 ); // false
alert( undefined == 0 ); // false
```

#### 함정 피하기

##### 비교 연산자 <sub>(`===` (일치 연산자) 제외)</sub>
- 피연산자 : `null` · `undefined` X

##### `null` · `undefined` 가능성 있는 변수
- 피연산자 되지 않도록 주의
- 따로 처리 코드 추가

```javascript
 0        ==  false     // true , 형 변환
-0        ==  false     // true , 형 변환
''        ==  false     // true , 형 변환
 0        === false     // false, 형 변환 X
-0        === false     // false, 형 변환 X
''        === false     // false, 형 변환 X

null      ==  undefined // true , null · undefined 형 변환 X
null      === undefined // false, 형 변환 X

null      >   0         // false, null → 0 (숫자형)
null      >=  0         // true , null → 0 (숫자형)
null      <   0         // false, null → 0 (숫자형)
null      <=  0         // true , null → 0 (숫자형)
null      ==  0         // false, null · undefined 형 변환 X
null      ==  undefined // true , null · undefined 형 변환 X

undefined >  0          // false, undefined → NaN (숫자형)
undefined <  0          // false, undefined → NaN (숫자형)
undefined == 0          // false, null · undefined 형 변환 X
undefined == null       // true , null · undefined 형 변환 X

/*
산술 · 기타 비교 연산자
- null → 0
- undefined → NaN

비교 연산자 (=== 일치 연산자 제외)
- 피연산자에 null · undefined 사용 X
*/
```

## 요약

##### 비교 연산자
- 불린값 반환

##### 문자열 비교
- 문자 단위
  - 사전 순서 <sub>(비교 기준)</sub>

##### 서로 다른 타입 값 비교
- 숫자형 형 변환
  - `===` <sub>(일치 연산자)</sub> 제외

##### `null` · `undefined`
- 동등 비교 <sub>(`==`)</sub>
  - 동일
  - 다른 값
    - 상이

##### `null` · `undefined` 가능성 있는 변수
- 비교 연산자 <sub>(`>` · `<` 등)</sub> 의 피연산자 시
  - 주의 요망
- 따로 체크 코드 추가
  - 습관 들이

## 비교
- 표현식들 결과 예측
```javascript
5 > 4
"apple" > "pineapple"
"2" > "12"
undefined == null
undefined === null
null == "\n0\n"
null === +"\n0\n"
```

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

```javascript
// 명백히 true
5 > 4 → true

// 문자열 비교
// - 사전순서 기준
//   - "a" < "p"
"apple" > "pineapple" → false

// 두 문자열 피연산자
// - 사전순 비교 진행
//   - "2" (좌측 1번째 글자) > "1" (우측 1번째 글자)
"2" > "12" → true

// null == undefined
// - == (동등 연산자)
//   - 특별 규칙
undefined == null → true

// null !== undefined
// - === (일치 연산자)
//   - 타입 체크
undefined === null → false

// null
// - 전부 부동 (undefined 제외)
null == "\n0\n" → false

// null
// - === (일치 연산자)
//   - 타입 체크
null === +"\n0\n" → false
```
