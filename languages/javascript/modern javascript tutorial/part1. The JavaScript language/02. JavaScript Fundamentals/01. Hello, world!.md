### `script` <sub>(태그)</sub>
- HTML 문서 대부분 위치에 삽입 가능

```html
<!DOCTYPE HTML>
<html>

<body>

  <p>스크립트 전</p>

  <script>
    alert( 'Hello, world!' );
  </script>

  <p>스크립트 후</p>

</body>

</html>
```
