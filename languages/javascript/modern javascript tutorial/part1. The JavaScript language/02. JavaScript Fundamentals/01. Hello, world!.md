Hello, world!
====

##### 코어 JS
- 실행 환경 영향 無 <sub>(독립적)</sub>
- 학습 실행 환경
  - 브라우저

### `<script>` <sub>(태그)</sub>

##### 대다수 HTML 문서 위치 내 JS 프로그램 삽입 가능
- 태그 도달 시 내부 코드 자동 처리 <sub>(브라우저)</sub>
```html
<!DOCTYPE HTML>
<html>

<body>
  <p>스크립트 전</p>

  <script>
    'Hello, world!';
  </script>

  <p>스크립트 후</p>
</body>

</html>
```

### 모던 마크업

##### `<script>` <sub>(태그)</sub> 속성
- 최근 드물게 사용
- 구식 코드 내 발견 가능

##### `type` <sub>(속성)</sub>
- HTML4
  - 필수사항
  - `type="text/javascript"`
- 모던 HTML
  - 선택사항
  - 속성 의미 변경 <sub>(JS 모듈)</sub>

##### `language` <sub>(속성)</sub>
- 사용중인 스크립트 언어
- 현재 JS 기본 <sub>(의미 퇴색)</sub>
  - 사용 불필요

##### 스크립트 전후에 위치한 주석
```html
<script type="text/javascript"><!--
    ...
//--></script>
```

##### 스크립트 전후 위치 주석 <sub>(트릭)</sub>
- `<script>` <sub>(태그)</sub> 처리 불가능 브라우저
  - 해당 스크립트 읽기 방지
- 아주 오래된 책 · 가이드 · 코드 내 발견 가능
  - 모던 JS 불필요
```html
<script type="text/javascript"><!--
  …
//--></script>
```

### 외부 스크립트 <sub>(src 속성)</sub>

##### 대용량 JS 코드
- 다수 별개 파일 소분 후 저장 가능
```html
<script src="…"></script>
```

##### 경로
- 전체 경로
  - `https://cdn…/…/lodash.js`
- 절대 경로 <sub>(사이트 루트 기준)</sub>
  - `/path/to/script.js`
- 상대 경로 <sub>(현재 페이지 기준)</sub>
  - `script.js`
```html
<!-- 전체 경로 -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/lodash.js/4.17.11/lodash.js"></script>

<!-- 절대 경로 -->
<script src="/path/to/script.js"></script>

<!-- 상대 경로 -->
<script src="script.js"></script>
```

##### 다수 스크립트 삽입
- 다수 태그 사용
```html
<script src="/js/script1.js"></script>
<script src="/js/script2.js"></script>
…
```

##### HTML 내 직접 스크립트 작성
- 보통 아주 간단한 스크립트
- 스크립트 코드량 ↑
  - 별개 파일 분리 저장 필요

#### 성능 이점

##### 스크립트 별도 파일 작성
- 다운로드 후 [캐시](https://en.wikipedia.org/wiki/Web_cache) 저장 <sub>(브라우저)</sub>

##### 다수 페이지 내 동일 스크립트 사용
- 페이지 변경 시
  - 스크립트 다시 다운로드 X
  - 캐시 내 스크립트 가져오기

##### 스크립트 파일 한 번만 다운로드
- 트래픽 절약
- 웹페이지 실제 속도 ↑

<br />

<img src="../../images/commons/icons/triangle-exclamation-solid.svg" /> **`src` <sub>(속성)</sub> 태그 내부 코드 무시**

##### `<script src="…">…</script>`
- `src` <sub>(속성)</sub> · 내부 코드 동시 동작 X
  - 내부 코드 무시
```html
<!-- src (속성) 有 → 내부 코드 무시 -->
<script src="file.js">
  1;
</script>
```

##### 스크립트 2개 분리
- `src` <sub>(속성)</sub> · 내부 코드 둘 다 동작
```html
<!-- 외부 파일 연결 -->
<script src="file.js"></script>

<!-- 내부 코드 실행 -->
<script>
  1;
</script>
```
