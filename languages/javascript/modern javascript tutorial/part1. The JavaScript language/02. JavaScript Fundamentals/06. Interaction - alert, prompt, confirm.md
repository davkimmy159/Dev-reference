상호작용
====

##### 브라우저 사용자 인터페이스 <sub>(함수)</sub>
- `alert`
- `prompt`
- `confirm`

##### 모달 창
- 메시지 있는 작은 창
- '모달' <sub>(단어)</sub> 의미
  - 페이지 나머지 부분 상호작용 불가능
- 사용자
  - 타 행동 불가능 <sub>(확인 버튼 누르기 전까지)</sub>

### `alert` <sub>(함수)</sub>

##### 문법
```javascript
alert(text);
```
- `text`
  - 사용자 표시 문자열

##### 모달 창 구성
- 표시 메시지
  - `text` <sub>(문자열 매개변수)</sub>
- 확인 버튼
  - 창 닫기

### `prompt` <sub>(함수)</sub>

##### 문법
```javascript
result = prompt(title, [default]);
```
- `title`
  - 사용자 표시 문자열
- `[default]`
  - 입력 필드 초깃값

<br />

<img src="../../images/commons/icons/circle-exclamation-solid.svg" /> **`[…]` <sub>(대괄호)</sub> 인수**

##### 해당 매개변수
- 선택 값 <sub>(필수 X)</sub>

<br />

##### 모달 창 구성
- 표시 메시지
  - `title` <sub>(문자열 매개변수)</sub>
- 입력 필드
- 확인 · 취소 버튼
  - 창 닫기

##### 상호작용
1. 입력 필드 내 값 입력 후 확인 버튼 누르기
    - 입력값 제출
    - 모달 창 종료
2. 취소 버튼 · ESC 키 누르기
    - 입력값 미제출 <sub>(`null`)</sub>
    - 모달 창 종료

##### 반환 값
- 문자열 <sub>(사용자 입력값)</sub>
- `null` <sub>(사용자 미입력 · 입력 취소)</sub>
```javascript
let age = prompt('나이를 입력해주세요.', 100);

// "당신의 나이는 100살입니다."
`당신의 나이는 ${age}살 입니다.`;
```

<br />

<img src="../../images/commons/icons/triangle-exclamation-solid.svg" /> **IE '기본값' 필수

##### IE
- 2번째 매개변수 <sub>(`[default]`)</sub> 부재 시
  - 입력 필드 내 `undefined` 명시
```javascript
let test = prompt("Test");
```

##### 2번째 매개변수 <sub>(`[default]`)</sub> 항상 전달 권장
- 모든 사용자 대상 깔끔한 프롬프트 표시
```javascript
// IE 사용자 위한 매개변수 처리
let test = prompt("Test", '');
```

<br />

### `confirm` <sub>(함수)</sub>

##### 문법
```javascript
result = confirm(question);
```
- `question`
  - 사용자 표시 문자열 <sub>(질문)</sub>

##### 모달 창 구성
- 표시 메시지 <sub>(질문)</sub>
  - `question` <sub>(문자열 매개변수)</sub>
- 확인 · 취소 버튼
  - 창 닫기

##### 반환 값 <sub>(`boolean`)</sub>
- `true`
  - 확인 버튼
- `false`
  - 취소 버튼
  - ESC 키
```javascript
let isBoss = confirm("당신이 주인인가요?");

isBoss); // true  (확인 버튼)
         // false (취소 버튼 · ESC 키)
```

요약
====

##### 브라우저 · 사용자 상호작용
- 함수 <sub>(3가지)</sub> 제공
  - `alert`
  - `prompt`
  - `confirm`

#### `alert` <sub>(함수)</sub>
- 메시지 표시

#### `prompt` <sub>(함수)</sub>
- 메시지 표시
  - 사용자 텍스트 입력 유도
- 입력 필드 표시
  - 사용자 입력값

##### 반환 값
- 문자열 <sub>(사용자 입력값)</sub>
  - 확인 버튼
- `null` <sub>(사용자 미입력 · 입력 취소)</sub>
  - 취소 버튼
  - ESC 키

#### `confirm` <sub>(함수)</sub>
- 메시지 표시

##### 반환 값 <sub>(`boolean`)</sub>
- `true`
  - 확인 버튼
- `false`
  - 취소 버튼
  - ESC 키

##### 공통점
- 모달 창 표시
  - 스크립트 실행 일시 중단
- 나머지 페이지 상호작용 불가능
  - 사용자 창 닫기 전까지

##### 공통 제약사항 <sub>(간결성 위한 대가)</sub>
1. 모달 창 위치 <sub>(브라우저 결정)</sub>
    - 보통 브라우저 중앙
2. 모달 창 모양
    - 브라우저별 상이
    - 창 모양 수정 불가능

## <img src="../../images/commons/icons/circle-check-solid.svg" /> 과제

### 간단한 페이지 만들기

##### 페이지 작성
1. 사용자 이름 질문
2. 입력받은 이름 그대로 출력

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

##### JS 코드
```javascript
let name = prompt("이름을 입력해 주세요.", "");
name;
```

##### 전체 페이지
```html
<!DOCTYPE html>
<html>

<body>
  <script>
    'use strict';

    let name = prompt("이름을 입력해 주세요.", "");
    name;
  </script>
</body>

</html>
```
