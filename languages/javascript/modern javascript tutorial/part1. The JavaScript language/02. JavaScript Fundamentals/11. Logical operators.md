논리 연산자
====

##### 피연산자 · 연산 결과
- 모든 자료형 값 가능

### `||` <sub>(OR 연산자)</sub>

##### `|` <sub>(수직선)</sub> 2개
```javascript
let result = a || b;
```

##### `boolean` 값 조작 <sub>(기본 기능)</sub>
- 피연산자 중 `true` 有
  - `true` 반환
- 피연산자 전부 `false`
  - `false` 반환
```javascript
alert( true  || true );  // true
alert( false || true );  // true
alert( true  || false ); // true
alert( false || false ); // false
```

##### 피연산자 자료형 제약 無
- 타 자료형 → `boolean` 형 변환
```javascript
// 1 → true
// 0 → false
if (1 || 0) { // if ( true || false )
  alert( 'truthy!' );
}
```

##### `if` <sub>(문)</sub> 조건문 내 자주 사용
- 다양한 다수 조건 가능
```javascript
let hour = 12;
let isWeekend = true;

if (hour < 10 || 18 < hour || isWeekend) {
  alert( '영업시간이 아닙니다.' );
}
```

### 1번째 truthy 찾는 `||` <sub>(OR 연산자)</sub>

##### 1번째 truthy 검색 <sub>(JS 추가 기능)</sub>
- 피연산자 중 1번째 truthy 검색
  - 검색 성공 시 기존 자료형 값 반환

#### JS 추가 기능 알고리즘
- _다수 피연산자 경우_
```javascript
let result = value1 || value2 || value3;
```

##### 1. 피연산자 평가
- 가장 좌측 → 우측

##### 2. 각 피연산자 `boolean` 형 변환
- 변환 후 값 `true` 경우
  - 평가 중단
  - 해당 피연산자 기존값 <sub>(형 변환 X)</sub> 반환

##### 3. 피연산자 모두 평가 후 <sub>(모든 피연산자 : `false`)</sub>
- 마지막 피연산자 기존값 반환

##### 반환 값
- 기존값 <sub>(형 변환 X)</sub>
```javascript
// 1 (truthy)
alert( 1 || 0 );

// 1 (truthy)
alert( null || 1 );

// 1 (truthy)
alert( null || 0 || 1 );

// 0 (모두 falsy)
// - 마지막 값 반환
alert( undefined || null || 0 );
```

#### ⓐ 변수 · 표현식 구성 목록에서 1번째 truthy 얻기
- 실제 값 들어있는 변수 찾아 반환
- 모든 변숫값 없는 경우
  - 마지막 변수 반환
```javascript
let firstName = "";
let lastName  = "";
let nickName  = "바이올렛";

// 바이올렛
alert( firstName || lastName || nickName || "익명");

nickName = "";

// 익명 (모든 변수 falsy)
alert( firstName || lastName || nickName || "익명");
```

#### ⓑ 단락 평가
- 평가 진행 중 <sub>(좌 → 우)</sub> truthy 만나면
  - 평가 중단 <sub>(나머지 요소 미처리)</sub>
- 부수적인 효과 <sub>(side effect)</sub>
  - ex\) 2번째 피연산자 변수 할당 등
- 주 용도
  - 연산자 좌측 조건 falsy 때만 명령어 실행
```javascript
// true 만나면 평가 중단 · alert (함수) 미실행
true  || alert("not printed");

false || alert("printed");
```

### `&&` <sub>(AND 연산자)</sub>

##### `&` <sub>(ampersand)</sub> 2개
```javascript
let result = a && b;
```

##### `boolean` 값 조작 <sub>(기본 기능)</sub>
- 피연산자 모두 `true`
  - `true` 반환
- 피연산자 중 `false` 有
  - `false` 반환
```javascript
alert( true  && true );  // true
alert( false && true );  // false
alert( true  && false ); // false
alert( false && false ); // false
```

##### 피연산자 자료형 제약 無
- 타 자료형 → `boolean` 형 변환
```javascript
// 1 → true
// 0 → false
if (1 && 0) { // if ( true && false )
  alert( "if 문 안에 falsy가 들어가 있으므로 alert창은 실행되지 않습니다." );
}
```

### 1번째 falsy 찾는 `&&` <sub>(AND 연산자)</sub>

##### 1번째 falsy 검색 <sub>(JS 추가 기능)</sub>
- 피연산자 중 1번째 falsy 검색
  - 검색 성공 시 기존 자료형 값 반환

#### JS 추가 기능 알고리즘
- _다수 피연산자 경우_
```javascript
let result = value1 && value2 && value3;
```

##### 1. 피연산자 평가
- 가장 좌측 → 우측

##### 2. 각 피연산자 `boolean` 형 변환
- 변환 후 값 `false` 경우
  - 평가 중단
  - 해당 피연산자 기존값 <sub>(형 변환 X)</sub> 반환

##### 3. 피연산자 모두 평가 <sub>(모든 피연산자 : `true`)</sub>
- 마지막 피연산자 기존값 반환

##### 반환 값
- 기존값 <sub>(형 변환 X)</sub>
```javascript
// 1번째 피연산자 truthy
// - 2번째 피연산자 반환
alert( 1 && 0 ); // 0
alert( 1 && 5 ); // 5

// 1번째 피연산자 falsy
// - 1번째 피연산자 반환
// - 2번째 피연산자 무시
alert( null && 5 ); // null
alert( 0 && "아무거나 와도 상관없습니다." ); // 0

// 3번째 피연산자 (falsy) 반환
// - null
alert( 1 && 2 && null && 3 );

// 3 (마지막 값)
// - 피연산자 모두 truthy
//   - 마지막 피연산자 반환
alert( 1 && 2 && 3 );
```

#### `||` <sub>(OR 연산자)</sub> vs  `&&` <sub>(AND 연산자)</sub>
- `||` <sub>(OR 연산자)</sub>
  - 1번째 truthy 반환
- `&&` <sub>(AND 연산자)</sub>
  - 1번째 falsy 반환

<br />

<img src="../../images/commons/icons/circle-exclamation-solid.svg" /> **우선순위 : `&&` \> `||`**

```javascript
/* 동일 동작 */
 a && b  ||  c && d
(a && b) || (c && d) // 가독성 ↑
```

<br />

<img src="../../images/commons/icons/triangle-exclamation-solid.svg" /> **`if` <sub>(문)</sub> → `||` · `&&` 대체 X**

##### 종종 `if` <sub>(문)</sub> 대신 사용
- 짧게 줄이는 용도
  - 명확성 ↓
  - 가독성 ↓
```javascript
let x = 1;
(x > 0) && alert( '0보다 큽니다!' );

↓↓↓

// 명확성 · 가독성 ↑
let x = 1;
if (x > 0) {
  alert( '0보다 큽니다!' );
}
```

### `!` <sub>(NOT 연산자)</sub>

##### `!` <sub>(느낌표)</sub> 1개
```javascript
let result = !value;
```
- 인수 하나 요구

##### 연산 순서
1. 피연산자 → `boolean` 형 변환
2. 변환 값의 역 반환
```javascript
alert( !true ); // false
alert( !0 );    // true
```

##### `!!` <sub>(2개 연달아 사용)</sub>
- 피연산자 값 → `boolean` 형 변환
- `Boolean` <sub>(내장 함수)</sub> 동일 기능
```javascript
/* !! */
// true
alert( !!"non-empty string" );

// false
alert( !!null );

/* Boolean */
// true
alert( Boolean("non-empty string") );

// false
alert( Boolean(null) );
```

##### 우선순위
- `!` > `&&` > `||`

### `"if"` 관한 고찰

##### 어떤 `alert` <sub>(함수)</sub> 실행될지 예측
- `if (…)` 내 표현식
```javascript
if (-1 || 0) alert( 'first' );
if (-1 && 0) alert( 'second' );
if (null || -1 && 1) alert( 'third' );
```

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

##### 1 · 3번째 표현식 내 `alert` <sub>(함수)</sub> 실행
```javascript
// -1 || 0 → -1 (truthy)
if (-1 || 0) alert( 'first' );

// -1 && 0 → 0 (falsy)
if (-1 && 0) alert( 'second' );

/*
 우선순위
 - && > ||
 -1 && 1 먼저 실행
 1. null || -1 && 1
 2. null || 1
 3. 1 (truthy) → alert 실행
 */
if (null || -1 && 1) alert( 'third' );
```

### 로그인 구현하기

##### 간이 로그인 창 구현 <sub>(`prompt` 대화상자 이용)</sub>
- 사용자 `"Admin"` 입력
  - 비밀번호 질문 대화상자
- 미입력 · Esc 키 누르기
  - "취소되었습니다." 메시지 표시
- 틀린 비밀번호 입력
  - "인증에 실패하였습니다." 메시지 표사

##### 비밀번호 확인 절차
- `"TheMaster"` 입력
  - `"환영합니다!"` 메시지 표시
- 틀린 비밀번호 입력
  - `"인증에 실패하였습니다."` 메시지 표시
- 빈 문자열 입력 · 입력 취소
  - "취소되었습니다." 메시지 표시

##### 순서도

![assignment-implementing-login](../../images/01/02/11/assignment-implementing-login.svg)

##### 조건
- 중첩 `if` 블록 사용
- 코드 전체 가독성 고려

##### 힌트
- 프롬프트 창에 미입력
  - `''` <sub>(빈 문자열)</sub>
- ESC 키 누르기
  - `null` 반환

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

```javascript
let userName = prompt("사용자 이름을 입력해주세요.", '');

if (userName == 'Admin') {

  let pass = prompt('비밀번호:', '');

  if (pass == 'TheMaster') {
    alert( '환영합니다!' );
  } else if (pass == '' || pass == null) {
    alert( '취소되었습니다.' );
  } else {
    alert( '인증에 실패하였습니다.' );
  }

} else if (userName == '' || userName == null) {
  alert( '취소되었습니다.' );
} else {
  alert( "인증되지 않은 사용자입니다." );
}
```
