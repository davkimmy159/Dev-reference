논리 연산자
====

##### 피연산자 · 연산 결과
- 모든 자료형 값 가능

### `||` <sub>(OR 연산자)</sub>

##### `|` <sub>(수직선)</sub> 2개
```javascript
let result = a || b;
```

#### 기본 기능

##### `boolean` 값 조작
- 피연산자 중 `true` 有
  - `true` 반환
- 피연산자 전부 `false`
  - `false` 반환
```javascript
alert( true  || true );  // true
alert( false || true );  // true
alert( true  || false ); // true
alert( false || false ); // false
```

##### 피연산자 자료형 제약 無
- 타 자료형 → `boolean` 형 변환
```javascript
// 1 → true
// 0 → false
if (1 || 0) { // if ( true || false )
  alert( 'truthy!' );
}
```

##### `if` <sub>(문)</sub> 조건문 내 자주 사용
- 다양한 다수 조건 가능
```javascript
let hour = 12;
let isWeekend = true;

if (hour < 10 || 18 < hour || isWeekend) {
  alert( '영업시간이 아닙니다.' );
}
```

#### JS 추가 기능

##### 1번째 truthy 검색
- 피연산자 중 1번째 truthy 검색
  - 검색 성공 시 기존 자료형 값 반환

##### 알고리즘
```javascript
let result = value1 || value2 || value3;
```
1. 피연산자 평가
   - 좌측 → 우측 진행
2. 각 피연산자 `boolean` 형 변환
   - `true` 변환 시 평가 중단
     - 기존값 반환
3. 모든 피연산자 평가 종료 <sub>(`true` 無)</sub>
   - 마지막 피연산자 기존값 반환
```javascript
// 1 (truthy)
//     ↓
alert( 1 || 0 );

// 1 (truthy)
//      X      ↓
alert( null || 1 );

// 1 (truthy)
//      X      X    ↓
alert( null || 0 || 1 );

// 0 (falsy)
// - 전부 falsy (마지막 값)
//         X         X      ↓
alert( undefined || null || 0 );
```

#### 활용

##### a. 변수 · 표현식 구성 목록 내 1번째 truthy 검색
- 다수 falsy 가능 변수
  - 실제 값 <sub>(truthy)</sub> 저장 변수 검색
  - 전부 falsy 값 → 기본값 반환
```javascript
let firstName = "";
let lastName  = "";
let nickName  = "바이올렛";

// 바이올렛 (truthy)
//         X           X           ↓
alert( firstName || lastName || nickName || "익명");

nickName = "";

// 익명 (falsy)
// - 전부 falsy (마지막 값)
//         X           X           X          ↓
alert( firstName || lastName || nickName || "익명");
```

##### b. 단락 평가
- 평가 진행 <sub>(좌 → 우)</sub> 중 truthy 검색 성공 시
  - 즉시 평가 중단 <sub>(나머지 피연산자 평가 X)</sub>
- 부수적인 효과 <sub>(부작용)</sub> 적용 용이
  - ex\) 다음 피연산자 변수 할당 등
- 주 용도
  - 연산자 좌측 조건 falsy → 명령어 실행
```javascript
// true 검색 성공 시 평가 중단
// - alert (함수) 미실행
true  || alert("not printed");
false || alert("printed"); // printed
```

### `&&` <sub>(AND 연산자)</sub>

##### `&` <sub>(ampersand)</sub> 2개
```javascript
let result = a && b;
```

#### 기본 기능

##### `boolean` 값 조작
- 피연산자 모두 `true`
  - `true` 반환
- 피연산자 중 `false` 有
  - `false` 반환
```javascript
alert( true  && true );  // true
alert( false && true );  // false
alert( true  && false ); // false
alert( false && false ); // false
```

##### 피연산자 자료형 제약 無
- 타 자료형 → `boolean` 형 변환
```javascript
// 1 → true
// 0 → false
if (1 && 0) { // if ( true && false )
  alert( "if 문 안에 falsy가 들어가 있으므로 alert창은 실행되지 않습니다." );
}
```

#### JS 추가 기능

##### 1번째 falsy 검색
- 피연산자 중 1번째 falsy 검색
  - 검색 성공 시 기존 자료형 값 반환

##### 알고리즘
```javascript
let result = value1 && value2 && value3;
```
1. 피연산자 평가
   - 좌측 → 우측 진행
2. 각 피연산자 `boolean` 형 변환
   - `false` 변환 시 평가 중단
     - 기존값 반환
3. 모든 피연산자 평가 종료 <sub>(`false` 無)</sub>
   - 마지막 피연산자 기존값 반환
```javascript
// 0 (falsy)
//     X    ↓
alert( 1 && 0 ); // 0

// 5 (truthy)
// - 전부 truthy (마지막 값)
//     X   X(↓)
alert( 1 && 5 ); // 5

// null (falsy)
//      ↓
alert( null && 5 ); // null

// 0 (falsy)
//     ↓
alert( 0 && "아무 값" ); // 0

// null (falsy)
//     X    X     ↓
alert( 1 && 2 && null && 3 ); // null

// 3 (truthy)
// - 전부 truthy (마지막 값)
//     X    X   X(↓)
alert( 1 && 2 && 3 );
```

|연산자|검색 값|
|:---:|---|
|`\|\|` <sub>(OR)</sub>|1번째 truthy|
|`&&` <sub>(AND)</sub>|1번째 falsy|

##### `||` <sub>(OR 연산자)</sub> \< `&&` <sub>(AND 연산자)</sub>
```javascript
/* 동일 동작 */
 a && b  ||  c && d
(a && b) || (c && d) // 가독성 ↑
```

<br />

<img src="../../images/commons/icons/triangle-exclamation-solid.svg" /> **`if` <sub>(문)</sub> 대체 X**

##### 종종 `if` <sub>(문)</sub> 대신 사용 <sub>(지양)</sub>
- 간결성 ↑
- 명확성 · 가독성 ↓
```javascript
let x = 1;

// 명확성 · 가독성 ↓
(x > 0) && alert( '0보다 큽니다!' );

↓↓↓

// 명확성 · 가독성 ↑
if (x > 0) {
  alert( '0보다 큽니다!' );
}
```

### `!` <sub>(NOT 연산자)</sub>

##### `!` <sub>(느낌표)</sub> 1개
```javascript
let result = !value;
```

##### 기능
- `boolean` 형 반대 값 반환

##### 연산 순서
1. 피연산자 → `boolean` 형 변환
2. 변환 값 역 반환
```javascript
alert( !true ); // false
alert( !0 );    // true
```

##### `!!` <sub>(연산자 2개 연달아 사용)</sub>
- 피연산자 → `boolean` 형 변환
- `Boolean` <sub>(내장 함수)</sub> 동일 기능
```javascript
/* !! */
alert( !!"non-empty" ); // true
alert( !!null );        // false

/* Boolean */
alert( Boolean("non-empty") ); // true
alert( Boolean(null) );        // false
```

##### 연산자 우선순위
- `||` <sub>(OR)</sub> \< `&&` <sub>(AND)</sub> \< `!` <sub>(NOT)</sub>

### `"if"` 관한 고찰

##### 어떤 `alert` <sub>(함수)</sub> 실행될지 예측
- `if (…)` 내 표현식
```javascript
if (-1 || 0) alert( 'first' );
if (-1 && 0) alert( 'second' );
if (null || -1 && 1) alert( 'third' );
```

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

##### 1 · 3번째 표현식 내 `alert` <sub>(함수)</sub> 실행
```javascript
// -1 || 0 → -1 (truthy)
if (-1 || 0) alert( 'first' );

// -1 && 0 → 0 (falsy)
if (-1 && 0) alert( 'second' );

/*
 우선순위
 - && > ||
 -1 && 1 먼저 실행
 1. null || -1 && 1
 2. null || 1
 3. 1 (truthy) → alert 실행
 */
if (null || -1 && 1) alert( 'third' );
```

### 로그인 구현하기

##### 간이 로그인 창 구현 <sub>(`prompt` 대화상자 이용)</sub>
- 사용자 `"Admin"` 입력
  - 비밀번호 질문 대화상자
- 미입력 · Esc 키 누르기
  - "취소되었습니다." 메시지 표시
- 틀린 비밀번호 입력
  - "인증에 실패하였습니다." 메시지 표사

##### 비밀번호 확인 절차
- `"TheMaster"` 입력
  - `"환영합니다!"` 메시지 표시
- 틀린 비밀번호 입력
  - `"인증에 실패하였습니다."` 메시지 표시
- 빈 문자열 입력 · 입력 취소
  - "취소되었습니다." 메시지 표시

##### 순서도

![assignment-implementing-login](../../images/01/02/11/assignment-implementing-login.svg)

##### 조건
- 중첩 `if` 블록 사용
- 코드 전체 가독성 고려

##### 힌트
- 프롬프트 창에 미입력
  - `''` <sub>(빈 문자열)</sub>
- ESC 키 누르기
  - `null` 반환

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

```javascript
let userName = prompt("사용자 이름을 입력해주세요.", '');

if (userName == 'Admin') {

  let pass = prompt('비밀번호:', '');

  if (pass == 'TheMaster') {
    alert( '환영합니다!' );
  } else if (pass == '' || pass == null) {
    alert( '취소되었습니다.' );
  } else {
    alert( '인증에 실패하였습니다.' );
  }

} else if (userName == '' || userName == null) {
  alert( '취소되었습니다.' );
} else {
  alert( "인증되지 않은 사용자입니다." );
}
```
