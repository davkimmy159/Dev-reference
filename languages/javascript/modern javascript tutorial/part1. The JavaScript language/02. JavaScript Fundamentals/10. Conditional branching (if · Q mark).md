`if` · `?` <sub>(조건 처리)</sub>
====

### `if` <sub>(문)</sub>

##### `if (…)` <sub>(괄호)</sub> 내 조건문 평가 <sub>(`boolean`)</sub>
- `true` 평가 시 `{}` <sub>(코드 블록)</sub> 실행
```javascript
if (…조건문…) {…코드 블록…}
```
```javascript
let year = prompt('ECMAScript-2015 명세는 몇 년도에 출판되었을까요?', '');

if (year == 2015) alert( '정답입니다!' );
```

##### 코드 블록 감싼 `{…}` <sub>(중괄호)</sub>
- 다수 실행문 작성 가능
- 항상 사용 권장 <sub>(가독성 ↑)</sub>
```javascript
if (…조건문…) {
  {…코드 블록…}
}
```
```javascript
// 다수 실행문
if (year == 2015) {
  alert( "정답입니다!" );
  alert( "아주 똑똑하시네요!" );
}

// {…} (중괄호) 미사용 (가독성 ↓)
if (year == 201) alert( "정답입니다!" );

// {…} (중괄호) 사용 (가독성 ↑)
if (year == 2015) {
  alert( "아주 똑똑하시네요!" );
}
```

### `boolean` 형 변환

##### `if (…)` <sub>(괄호)</sub> 내 표현식 <sub>(조건문)</sub> 평가 후 결과 형 변환
- 결과 값 → `boolean` 값

|표현식|형 변환|
|---|---|
|'falsy' <sub>(거짓 같은)</sub> 값<br />- `0` <sub>(숫자)</sub><br /> - `""` <sub>(빈 문자열)</sub><br /> - `null` <br /> - `undefined`<br /> - `NaN`|`false`|
|'truthy' <sub>(참 같은)</sub> 값<br />- falsy 값 외 값|`true`|

```javascript
// 항상 미실행
if (0) { // 0 (숫자) → false (falsy)
  …
}

// 항상 실행
if (1) { // 1 (숫자) → true (truthy)
  …
}

// 평가 후 확정된 boolean 값 전달 가능
let cond = (year == 2015);
if (cond) {
  …
}
```

### `else` <sub>(절)</sub>

##### `if (…)` <sub>(괄호)</sub> 내 조건문 <sub>(`boolean`)</sub>
- `false` 평가 시 `{}` <sub>(코드 블록)</sub> 실행
```javascript
if (…조건문…) {
  {…코드 블록…} // true  평가 시 실행
} else {
  {…코드 블록…} // false 평가 시 실행
}
```
```javascript
let year = prompt('ECMAScript-2015 명세는 몇 년도에 출판되었을까요?', '');

if (year == 2015) {
  alert( '정답입니다!' );

// 2015 외 값 입력 시 실행
} else {
  alert( '오답입니다!' );
}
```

### `else if` <sub>(문)</sub>

##### 다수 조건 처리
- 직전 조건문 `false` 평가 시 평가 진행
- 마지막 `else` <sub>(절)</sub>
  - 선택사항
```javascript
if (…조건문 1…) {
  {…코드 블록…}
} else if (…조건문 2…) { // 조건문 1 false 평가 시 진행
  {…코드 블록…}
} else if (…조건문 n…) { // 조건문 2 false 평가 시 진행
  {…코드 블록…}
} else { // 선택사항
  {…코드 블록…}
}
```
```javascript
let year = prompt('ECMAScript-2015 명세는 몇 년도에 출판되었을까요?', '');

if (year < 2015) {
  alert( '숫자를 좀 더 올려보세요.' );
} else if (year > 2015) {
  alert( '숫자를 좀 더 내려보세요.' );
} else {
  alert( '정답입니다!' );
}
```
### `?` <sub>(조건부 연산자)</sub>

##### 조건별 상이 값 변수 할당
- 더 짧고 간결
- 유일한 삼항 연산자 <sub>(피연산자 3개)</sub>
```javascript
// 조건문 평가
// - true  평가 시 (truthy) trueValue  반환
// - false 평가 시 (falsy)  falseValue 반환
…조건문… ? trueValue : falseValue
```
```javascript
let accessAllowed;
let age = prompt('나이를 입력해 주세요.', '');

// if (문)
if (age > 18) {
  accessAllowed = true;
} else {
  accessAllowed = false;
}

// ? (조건부 연산자)
let accessAllowed1 = (age > 18) ? true : false;

// `(…)` (괄호) 생략 가능 (우선순위 ↓)
// - 가독성 ↓
let accessAllowed2 =  age > 18  ? true : false;

alert(accessAllowed);
```

<br />

<img src="../../images/commons/icons/circle-exclamation-solid.svg" /> **주의**

##### 비교 연산자 자체 `true` · `false` 반환
- `?` <sub>(조건부 연산자)</sub> 불필요
```javascript
accessAllowed = (age > 18) ? true : false;
accessAllowed =  age > 18;
```

### 다중 `?` <sub>(조건부 연산자)</sub>

##### 다수 조건 처리
- 직전 조건문 `false` 평가 시
  - 다음 조건문 평가 진행
```javascript
let age = prompt('나이를 입력해주세요.', 18);
let message;

// 다중 ? (조건부 연산자)
message =
  (age < 3)   ? '아기야 안녕?' : // ①
  (age < 18)  ? '안녕!'      : // ②
  (age < 100) ? '환영합니다!'  : // ③
  '이 세상 사람이 아니시군요!';     // ④

// else if (문)
if (age < 3) {
  message = '아기야 안녕?';
} else if (age < 18) {
  message = '안녕!';
} else if (age < 100) {
  message = '환영합니다!';
} else {
  message = '이 세상 사람이 아니시군요!';
}

alert( message );
```

### 부적절한 `?` <sub>(조건부 연산자)</sub>

##### 종종 `if` <sub>(문)</sub> 대신 사용 <sub>(지양)</sub>
- 간결성 ↑
- 가독성 ↓ <sub>(긴 수평 코드)</sub>
```javascript
let company = prompt('JS는 어떤 회사가 만들었을까요?', '');

// 코드량 수평 (↔) 증가
// - 지양 (가독성 ↓)
(company == 'Netscape') ? alert('정답입니다!') : alert('오답입니다!');

// 코드량 수직 (⭥) 증가
// - 권장 (가독성 ↑)
if (company == 'Netscape') {
  alert('정답입니다!');
} else {
  alert('오답입니다!');
}
```

##### 용도별 사용법 상이
- `if` <sub>(문)</sub>
  - 다수 분기 처리
- `?` <sub>(조건부 연산자)</sub>
  - 조건별 상이 값 반환

### `if` <sub>(문)</sub> · `'0'` <sub>(문자열)</sub>

##### `alert` <sub>(함수)</sub> 실행 여부
```javascript
if ("0") {
  alert( 'Hello' );
}
```

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

##### 코드 실행
- `""` <sub>(빈 문자열)</sub> 외 모든 문자열
  - 논리 평가 시 `true` 반환
- `"0"` <sub>(문자열)</sub>
  - 비어있지 않은 문자열
```javascript
if ("0") {          // true 평가
  alert( 'Hello' ); // 실행
}
```

### 입력 숫자 부호 표시하기

##### 입출력 코드 작성
1. 사용자 숫자 1개 입력받기 <sub>(프롬프트 창)</sub>
2. 조건별 결과 얼럿 창 출력 <sub>(`if…else`)</sub>
   - 사용자 항상 숫자 입력 가정

|조건|결과|
|---|:---:|
|`입력값 > 0`|`1`|
|`입력값 < 0`|`-1`|
|`입력값 == 0`|`0`|

[데모](https://ko.js.cx/task/sign/if_sign/)

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

```javascript
let value = prompt('숫자를 입력하세요.', 0);

if (value > 0) {
  alert( 1 );
} else if (value < 0) {
  alert( -1 );
} else {
  alert( 0 );
}
```

### `if` <sub>(문)</sub> → `?` <sub>(조건부 연산자)</sub> 교체

##### 코드 수정
- `if` <sub>(문)</sub> → `?` <sub>(조건부 연산자)</sub>
  - 동작 결과 동일 필수
```javascript
let result;

if (a + b < 4) {
  result = '미만';
} else {
  result = '이상';
}
```

```javascript
let result = (a + b < 4) ? '미만' : '이상';
```

### `if…else` → `?` <sub>(조건부 연산자)</sub> 교체

##### 코드 수정
- `if…else` → `?` <sub>(조건부 연산자)</sub>
  - 동작 결과 동일 필수
  - 표현식 다수 줄 분할 <sub>(가독성 ↑)</sub>
```javascript
let message;

if (login == '직원') {
  message = '안녕하세요.';
} else if (login == '임원') {
  message = '환영합니다.';
} else if (login == '') {
  message = '로그인이 필요합니다.';
} else {
  message = '';
}
```

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

```javascript
let message =
  (login == '직원') ? '안녕하세요.'        :
  (login == '임원') ? '환영합니다.'        :
  (login == '')    ? '로그인이 필요합니다.' :
  '';
```
