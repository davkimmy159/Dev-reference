`while` · `for` <sub>(반복문)</sub>
====

### `while` <sub>(반복문)</sub>

##### `while(…)` <sub>(괄호)</sub> 내 조건문 평가
- `true` 평가 시 <sub>(truthy)</sub> 반복문 본문 실행
```javascript
while (condition) {
  /* body (반복문 본문) */
}
```

##### `i < 3` <sub>(조건)</sub> 만족 시 계속 `i` 출력
```javascript
let i = 0;

while (i < 3) {
  alert( i ); // 0, 1, 2
  i++;
}
```

##### 반복문 조건
- 모든 표현식 · 변수 사용 가능
- 평가 결과 형 변환
  - 결과 값 → `boolean` 값
```javascript
let i = 3;

// i (변수) → 0 (숫자) 도달 시
// - falsy (false 평가)
while (i) { // while (i != 0)
  alert( i );
  i--;
}
```

<br />

<img src="../../images/commons/icons/circle-exclamation-solid.svg" /> **한 줄 본문 `{…}` <sub>(중괄호)</sub> 생략 가능</sub>**

```javascript
let i = 3;

// 지양 (가독성 ↓)
while (i) alert(i--);

// 항상 사용 권장 (가독성 ↑)
while (i) {
  alert(i--);
}
- 
```

### `do…while` <sub>(반복문)</sub>

##### 조건 위치
- 반복문 본문 아래

##### 본문 먼저 실행 후 조건 확인
- 조건 == truthy 경우
  - 본문 계속 실행
- 본문 최소한 1번 실행
```javascript
do {
  // body (반복문 본문)
} while (condition);
```
```javascript
let i = 0;
do {
  alert( i );
  i++;
} while (i < 3);
```

### `for` <sub>(반복문)</sub>
```javascript
for (begin; condition; step) {
  // body (반복문 본문)
}
```
```javascript
for (let i = 0; i < 3; i++) {
  alert(i); // 0, 1, 2
}
```

<br />

<img src="../../images/commons/icons/circle-exclamation-solid.svg" /> **인라인 변수 선언**

##### `i` <sub>(카운터 변수)</sub>
- 반복문 내 선언
- 반복문 안에서만 접근 가능
```javascript
for (let i = 0; i < 3; i++) {
  alert(i); // 0, 1, 2
}

// Error: i is not defined
alert(i);
```

##### 기존 정의된 변수 사용 가능
```javascript
let i = 0;

// 기존 정의된 변수 사용
for (i = 0; i < 3; i++) {
  alert(i); // 0, 1, 2
}
alert(i); // 3
```

#### 구성 요소 생략
```javascript
let i = 0;
for (; i < 3; i++) {
  alert( i ); // 0, 1, 2
}
```
```javascript
let i = 0;
for (; i < 3;) { // while (i < 3)
  alert( i++ );  // 0, 1, 2
}
```

##### 모든 구성 요소 생략
- 무한 반복문
```javascript
for (;;) {
  // 끊임 없이 본문 실행
}
```

### 반복문 빠져나오기

##### `break` <sub>(지시자)</sub>
```javascript
let sum = 0;
while (true) {
  let value = +prompt("숫자를 입력하세요.", '');
  if (!value) break;
  sum += value;
}
alert( '합계: ' + sum );
```

### 다음 반복으로 넘어가기

##### `continue` <sub>(지시자)</sub>
- 전체 반복문 중단 X
- 현재 실행 중인 이터레이션 중단 후
  - 다음 이터레이션 강제 실행
- 현재 반복 중단 후
  - 다음 반복 실행
```javascript
for (let i = 0; i < 10; i++) {

  // if (문) 조건 truthy
  // - 남아있는 본문 미실행
  if (i % 2 == 0) continue;
  alert(i); // 1, 3, 5, 7, 9
}
```

<br />

<img src="../../images/commons/icons/circle-exclamation-solid.svg" /> **`continue` <sub>(지시자)</sub> : 중첩 감소 도움**

##### 위 예시
- `continue` <sub>(지시자)</sub> 미사용 시
```javascript
for (let i = 0; i < 10; i++) {

  /*
   중첩 레벨 1개 증가
   if (문) 코드량 ↑
   - 전체 가독성 ↓
   */
  if (i % 2) {
    alert( i );
  }
}
```

<br />

<img src="../../images/commons/icons/triangle-exclamation-solid.svg" /> **`?` 우측 `break` · `continue` X**

##### `?` <sub>(3항 연산자)</sub>
- 문법 구조
- 표현식 X
  - `break` <sub>(지시자)</sub>
  - `continue` <sub>(지시자)</sub>
  - 기타 등등
```javascript
if (i > 5) {
  alert(i);
} else {
  continue;
}

↓↓↓

(i > 5) ? alert(i) : continue; // 문법 에러
```

### `break` · `continue` · `레이블`

##### 레이블 <sub>(`label`)</sub>
- 중첩 반복문 한 번에 빠져나오기
```javascript
labelName: for (…) {
  …
}
```
```javascript
outer: for (let i = 0; i < 3; i++) {
  for (let j = 0; j < 3; j++) {
    let input = prompt(`(${i},${j})의 값`, '');

    // outer (레이블) 붙은 반복문 찾은 후
    // - 해당 반복문 빠져나옴
    if (!input) break outer;
    …
  }
}
alert('완료!');
```
- 별도 줄 가능
```javascript
outer:
for (let i = 0; i < 3; i++) {
  …
}
```

##### `continue` 함께 사용 시
- 레이블 붙은 반복문의 다음 반복 실행

<img src="../../images/commons/icons/triangle-exclamation-solid.svg" /> **레이블 마음대로 이동 불가능**

##### `break` · `continue` <sub>(지시자)</sub>
- 반복문 안에서만 사용

##### 레이블
  - `break` · `continue` <sub>(지시자)</sub> 위에 위치

요약
====

##### 반복문 3가지
- `while (…)`
  - 각 반복 전 조건 확인
- `do…while (…)`
  - 각 반복 후 조건 확인
- `for (…;…;…)`
  - 각 반복 전 조건 확인
  - 추가 세팅 가능

##### '무한' 반복문
- `while (true)` · `for (;;)`

##### 반복 중단
- `break` <sub>(지시자)</sub>

##### 다음 반복 넘어가기
- `continue` <sub>(지시자)</sub>

##### 레이블 <sub>(반복문 앞)</sub> · `break` · `continue` <sub>(지시자)</sub>
- 중첩 반복문 빠져나오기
- 외부 반복문 이동

### `while` <sub>(반복문)</sub> 출력값 예상하기

##### 반복문 순차 실행될 때마다 출력값 예상
1. `++i` <sub>(전위형 증가 연산자)</sub>
```javascript
let i = 0;
while (++i < 5) alert( i );
```
2. `i++` <sub>(후위형 증가 연산자)</sub>
```javascript
let i = 0;
while (i++ < 5) alert( i );
```

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

##### 1. `++i` <sub>(전위형 증가 연산자)</sub>
- `1` ~ `4` 출력
- `i` 증가 · 새 값 반환
```javascript
let i = 0;
while (++i < 5) // while (1 < 5)
  alert(i)      // alert (1)
while (++i < 5) // while (2 < 5)
  alert(i)      // alert (2)
while (++i < 5) // while (3 < 5)
  alert(i)      // alert (3)
while (++i < 5) // while (4 < 5)
  alert(i)      // alert (4)
while (++i < 5) // while (5 < 5), 실행 중단
```

##### 2. `i++` <sub>(후위형 증가 연산자)</sub>
- `1` ~ `5` 출력
- `i` 증가 · 기존값 반환
```javascript
while (i++ < 5) // while (0 < 5)
  alert(i)      // alert (1)
while (i++ < 5) // while (1 < 5)
  alert(i)      // alert (2)
while (i++ < 5) // while (2 < 5)
  alert(i)      // alert (3)
while (i++ < 5) // while (3 < 5)
  alert(i)      // alert (4)
while (i++ < 5) // while (4 < 5)
  alert(i)      // alert (5)
while (i++ < 5) // while (5 < 5) // 실행 중단
```

### `for` <sub>(반복문)</sub> 출력값 예상하기

##### 반복문 순차 실행될 때마다 출력값 예상
1. `++i` <sub>(전위형 증가 연산자)</sub
```javascript
for (let i = 0; i < 5; ++i) alert( i );
```
2. `i++` <sub>(후위형 증가 연산자)</sub>
```javascript
for (let i = 0; i < 5; i++) alert( i );
```

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

#### 두 경우 모두 `0` ~ `4` 출력

##### `for` <sub>(반복문)</sub> 알고리즘
1. 모든 작업 시작 전
    - `i = 0`
2. 조건 만족 확인
    - `i < 5`
3. 조건 == `true`
    - 본문 <sub>(`alert(i)`)</sub> 실행
    - 이후 `i` 증가
- `i` 증가
  - 조건 확인 단계와 별개
- 증가 연산자 반환 값
  - 조건 확인 단계 사용 불가능

### 유횻값 입력 때까지 프롬프트 창 표시

#### `100` 보다 큰 숫자 입력 사용자 안내

##### `100` 이하 값 입력 시
- 동일 프롬프트 창 표시 <sub>(반복문 사용)</sub>

##### `100` 초과 값 입력 · 취소 버튼 · 입력 없이 확인 버튼
- 프롬프트 창 종료

##### 조건
- 숫자만 입력 가정
- 예외 상황 처리 X
  - 숫자 외 값 입력 등

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

##### `do…while` <sub>(반복문)</sub>
```javascript
let num;

do {
  num = prompt("100을 초과하는 숫자를 입력해주세요.", 0);
} while (num <= 100 && num);
```

##### 두 조건 모두 truthy 인 경우 창 표시
- `num <= 100`
- `num` ≠ `null` · `""` <sub>(빈 문자열)</sub>

##### 참고 <sub>(조건 2개 이유)</sub>
- `num` == `null` <sub>(`null` → `0`)</sub>
  - `num <= 100` : `true`

### 소수 출력하기

##### [소수](https://ko.wikipedia.org/wiki/%EC%86%8C%EC%88%98_(%EC%88%98%EB%A1%A0))
- 자신보다 작은 2개의 자연수 곱하여 만들 수 없는 1보다 큰 자연수
- 1 · 자신 외 자연수로 나눌 수 없는 자연수
  - ex\) 5

##### `2` ~ `n` 중 소수만 출력
- ex\) `n` == `10`
  - `2,3,5,7` 출력

##### 주의
- 임의 숫자 <sub>(`n`)</sub> 대해 동작

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

#### 소수 판단 알고리즘 다양

##### 중첩 반복문
```
i (범위 내 모든 숫자) {
  1 ~ i 사이 제수 존재 확인
  - 有 → 소수 X
  - 無 → 소수 (출력)
}

```

##### 레이블
```javascript
let n = 10;

nextPrime:
// 각 i : 반복문 실행
for (let i = 2; i <= n; i++) {

  // 제수 (나눗수) 찾기
  for (let j = 2; j < i; j++) {

    // 소수 X → 다음 i
    if (i % j == 0) continue nextPrime;
  }

  alert( i ); // 소수
}
```

##### 최적화 필요
- 제수 찾기
  - `2` ~ `i` 제곱근
- 아주 큰 `n` <sub>(복잡한 알고리즘 적용)</sub>
  - [이차 체](https://ko.wikipedia.org/wiki/%EC%9D%B4%EC%B0%A8_%EC%B2%B4) <sub>(Quadratic sieve)</sub>
  - [수 체](https://en.wikipedia.org/wiki/General_number_field_sieve) <sub>(General number field sieve)</sub>
