`var` <sub>(구식)</sub>
====

### `{…}` <sub>(블록)</sub> 스코프 X

##### 함수 · 전역 스코프 <sub>(`{…}` 블록 무시)</sub>
- `if` <sub>(문)</sub>
```javascript
if (true) {
  var test = true; // 전역 변수
}

test; // true
```
- 반복문
```javascript
for (var i = 0; i < 9; i++) { // 전역 변수
  …
}

i;    // 10
```

##### 함수 내 `{…}` <sub>(코드 블록)</sub>
- 함수 수준 스코프
```javascript
function sayHi() {
  if (true) {

    // 함수 내 지역 변수
    var phrase = "Hello";
  }

  phrase; // Hello
}

sayHi();

phrase; // Error: phrase is not defined
```

##### 과거 JS 잔재
- 과거 블록 수준 렉시컬 환경 無

### 변수 중복 선언 허용

##### 1번째 이후 선언 무시
- 할당 연산 정상 동작
```javascript
var user = "Pete";

// var 선언 무시 (할당 연산 동작)
var user = "John";

user; // John
```

### '호이스팅' <sub>(hoisting)</sub>

##### `var` 사용 시 발생 현상
- `var` 선언 <sub>(정의)</sub> 변수
  - 함수 최상단 자동 이동

##### 함수 <sub>(중첩 X)</sub> 본문 내 `var` 선언 변수
- 변수 정의 위치 자동 이동 <sub>(선언 위치 무관)</sub>
  - 함수 본문 시작 지점 <sub>(함수 시작 직후 처리)</sub>
```javascript
function Same() {
  phrase = "Hello";

  phrase;

  var phrase; // var 선언 변수
}

Same();

↓↓↓

function Same() {
  var phrase; // var 선언 변수 (최상단 이동)

  phrase = "Hello";

  phrase;
}

same();
```

##### `{…}` <sub>(코드 블록)</sub> 무시 <sub>(유무 여부 무관)</sub>
```javascript
function same() {
  phrase = "Hello";

  if (false) {
    var phrase; // `{…}` (블록) 무시
  }

  phrase;
}

same();

↓↓↓

function same() {
  var phrase // `{…}` (블록) 무시 (최상단 이동)

  phrase = "Hello";

  if (false) {}

  phrase;
}

same();
```

##### 할당
- 호이스팅 미발생
```javascript
function same() {
  phrase;

  // var 변수 선언 · 값 할당
  var phrase = "Hello";
}

same();

↓↓↓

function same() {
  var phrase; // var 변수 선언 (최상단 이동)

  phrase;     // undefined

  phrase = "Hello"; // 값 할당 (기존 위치 유지)
}

same();
```

### 즉시 실행 함수 표현식

##### immediately-invoked function expressions <sub>(IIFE)</sub>
- `var` 선언 변수
  - `{…}` <sub>(블록)</sub> 스코프 부여

#### 형태

##### `(…)` <sub>(괄호)</sub> 내 함수 위치 · 즉시 실행
- `(function(…){…})()`
```javascript
(function() {
  var message = "Hello"; // 함수 전용 변수

  message; // Hello
})();
```

##### `(…)` <sub>(괄호)</sub> 미사용 시 에러
- `function` <sub>(키워드)</sub>
  - 함수 선언문 시작 간주
    - 함수명 필수
```javascript
// 함수명 無 (에러)
function() { // Error: Function statements require a function name
  var message = "Hello";

  message; // Hello
}();
```

##### `(…)` <sub>(괄호)</sub> 미사용 · 함수명 추가 시 에러
- 함수 선언문 정의 함수
  - 정의 · 호출 동시 수행 불허 <sub>(에러)</sub>
```javascript
function go() {
  …
}(); // 함수 선언 직후 호출 불가능 (에러)
```

##### `(…)` <sub>(괄호)</sub> 내 함수 <sub>`(function(…){…})`</sub>
- 표현식 <sub>(함수 선언문 X)</sub> 인식
  - 무기명 가능
  - 즉시 호출 가능
- 모던 JS 불필요

##### 다양한 IIFE
```javascript
// ( function(…){…} )();
(function() {
  "`()` (괄호) 내 함수";
})();

// ( function(…){…}() );
(function() {
  "`()` (괄호) 내 전체";
}());

// !function(…){…}()
!function() {
  "표현식 앞 `!` (비트 NOT 연산자) 추가";
}();

// +function(…){…}()
+function() {
  "표현식 앞 `+` (단항 덧셈 연산자) 추가";
}();
```

요약
====

##### `var` 선언 변수 주요 특성 <sub>(3가지)</sub>
- a. 스코프
  - `{…}` <sub>(블록)</sub> 스코프 X
  - 함수 스코프
- b. 선언 처리 시점
  - 함수 시작 지점
  - 스크립트 시작 지점 <sub>(전역)</sub>
- c. 전역 선언 <sub>(브라우저)</sub>
  - 전역 객체 내 프로퍼티 자동 지정
- 대다수 경우 나쁜 부작용 발생
  - `let` 표준 도입 이유 <sub>(부작용 제거)</sub>
