함수 바인딩
====

### `this` 유실

##### 객체 내 메서드
- 타 위치 전달 시 `this` 정보 유실
  - ex\) 콜백 전달 등

##### `setTimeout` <sub>(메서드)</sub>
- 객체 내 메서드 전달 시 `this` 유실
```javascript
let user = {
  firstName: "John",
  sayHi() {
    alert(`Hello, ${this.firstName}!`);
  },
};

// Hello, undefined!
setTimeout(user.sayHi, 1000);
```
- 객체 내 메서드 → 별도 함수 분리 발생
```javascript
setTimeout(user.sayHi, 1000);

↓↓↓

let f = user.sayHi;
setTimeout(f, 1000); // user (객체) 컨텍스트 유실
```

##### 브라우저 환경 `setTimeout` <sub>(메서드)</sub>
- 전달 함수 호출 시
  - `this` → `window` 할당
- ex\) `this.firstName`
  - `window.firstName` <sub>(`undefined`)</sub>

##### 해결 방법 <sub>(2가지)</sub>
- a. 래퍼 함수
- b. `function.bind` <sub>(내장 메서드)</sub>

### a. 래퍼 함수

##### 래퍼 함수 내 객체 내 메서드 호출
- 외부 렉시컬 환경 내 객체 검색
```javascript
let user = {
  firstName: "John",
  sayHi() {
    alert(`Hello, ${this.firstName}!`);
  }
};

setTimeout(function() {
  user.sayHi();
}, 1000);

// 화살표 함수 (간결)
setTimeout(() => user.sayHi(), 1000);
```

##### 취약성 有
- `setTimeout` 실행 전 해당 객체 변경 발생 시
  - 변경된 객체 내 메서드 호출
```javascript
let user = {
  firstName: "John",
  sayHi() {
    alert(`Hello, ${this.firstName}!`);
  },
};

// Hello, 또 다른 사용자!
setTimeout(() => user.sayHi(), 1000);

// 1초 경과 전 user (객체) 값 변경
user = {
  sayHi() {
     alert("Hello, 또 다른 사용자!");
  },
};
```

### b. [`function.bind`](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Function/bind) <sub>(내장 메서드)</sub>

##### 함수 내 `this` 변경 후 고정
```javascript
let boundFunc = func.bind(context);
```

##### `boundFunc` <sub>(특수 객체)</sub>
- 호출 가능 <sub>(≒ 함수)</sub>
- `func` 반환 <sub>(`this` → `context` 고정)</sub>
```javascript
let user = {
  firstName: "John"
};

function func() {
  alert(this.firstName);
}

// func 의 this → user (객체) 바인딩
let funcUser = func.bind(user);
funcUser(); // John
```

##### 원본 함수에 인수 그대로 전달
```javascript
let user = {
  firstName: "John"
};

function func(phrase) {
  alert(phrase + ', ' + this.firstName);
}

// this → user 바인딩
let funcUser = func.bind(user);

// 인수 : "Hello"
// this → user (객체) 고정
funcUser("Hello"); // Hello, John
```

##### 객체 메서드 `bind`
- 인수 그대로 전달
- `this` 고정
```javascript
let user = {
  firstName: "John",
  sayHi() {
    alert(`Hello, ${this.firstName}!`);
  }
};

/* user.sayHi (메서드) → user (객체) 바인딩
 sayHi (함수)
 - '묶인 (bound)' 함수
 - 단독 호출 가능
 */
let sayHi = user.sayHi.bind(user);

// 객체 없이 객체 메서드 호출
sayHi(); // Hello, John!

// 지연 호출
// - Hello, John!
setTimeout(sayHi, 1000);

// 1초 이내 user (객체) 값 변해도 지연 호출 내 기존값 사용
user = {
  sayHi() { alert("또 다른 사용자!"); }
};
```
```javascript
let user = {
  firstName: "John",
  say(phrase) {
    alert(`${phrase}, ${this.firstName}!`);
  }
};

let say = user.say.bind(user);

say("Hello"); // Hello, John
              // - 인수 : "Hello"
say("Bye");   // Bye, John
              // - 인수 : "Bye"
```

### 부분 적용

##### 인수 바인딩
- 드물게 사용
- 가끔 유용
```javascript
let bound = func.bind(context, [arg1], [arg2], …);
```

##### `arg1, arg2, …`
- 바인딩 함수 전달 인수
```javascript
function mul(a, b) {
  return a * b;
}

let double = mul.bind(null, 2);

alert( double(3) ); // 6 , mul(2, 3)
alert( double(4) ); // 8 , mul(2, 4)
alert( double(5) ); // 10, mul(2, 5)
```

##### 가독성 좋은 이름 가진 독립 함수 생성 가능
- `double`
- `triple`
- 기타 등등

##### 1번째 인수 고정
- 매번 인수 전달 불필요

##### 매우 포괄적인 함수 기반
- 덜 포괄적인 <sub>(더 구체적인)</sub> 변형 함수
- ex\) `send(from, to, text)`
  - → `sendTo(to, text)`
    - 전송 주체 : `user` <sub>(현재 사용자)</sub>

### 컨텍스트 없는 부분 적용

##### 네이티브 `bind`
- 컨텍스트 <sub>(1번째)</sub> 인수 생략 불가능

##### 인수만 바인딩 시
- `partial` <sub>(헬퍼 함수)</sub> 구현
```javascript
function partial(func, ...argsBound) {

  // 래퍼 함수
  return function(...args) {
    return func.call(this, ...argsBound, ...args);
  }
}

let user = {
  firstName: "John",
  say(time, phrase) {
    alert(`[${time}] ${this.firstName}: ${phrase}!`);
  }
};

// 시간 고정 부분 메서드 추가
user.sayNow = partial(user.say, new Date().getHours() + ':' + new Date().getMinutes());

// [10:00] John: Hello!
user.sayNow("Hello");
```

##### `partial(func[, arg1, arg2, … )` 호출 시
- 래퍼 함수 반환

##### 래퍼 호출 시
- 동일한 `this`
- `user.sayNow`
  - `user` <sub>(객체)</sub> 대상 호출
- `...argsBound`
  - `partial` <sub>(헬퍼 함수)</sub> 호출 시 받은 인수
    - `new Date().getHours()`
- `...args`
  - 래퍼 <sub>(함수)</sub> 전달 인수 <sub>(`"Hello"`)</sub>

##### lodash <sub>(JS 라이브러리)</sub>
- `_.partial`

### `this` 값 == `undefined` 함수 고치기

##### `askPassword` <sub>(함수)</sub>
1. 비밀번호 확인
2. 결과 따라 `user.login[Ok·Fail]` 호출
- 현재 함수 호출 시 에러

##### 정상 동작 하도록 코드 수정
```javascript
function askPassword(ok, fail) {
  let password = prompt("비밀번호를 입력해주세요.", '');
  if (password == "rockstar") {
    ok();
  } else {
    fail();
  }
}

let user = {
  name: 'John',

  loginOk() {
    alert(`${this.name}님이 로그인하였습니다.`);
  },

  loginFail() {
    alert(`${this.name}님이 로그인에 실패하였습니다.`);
  },

};

askPassword(user.loginOk, user.loginFail);
```

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

##### `ask` <sub>(함수)</sub>
- 객체 없이 `login[Ok·Fail]` <sub>(메서드)</sub> 가져옴
- 함수 내 메서드 호출 시
  - `this = undefined`

##### `bind` <sub>(메서드)</sub> 사용해 컨텍스트 고정
```javascript
function askPassword(ok, fail) {
  let password = prompt("비밀번호를 입력해주세요.", '');
  if (password == "rockstar") ok();
  else fail();
}

let user = {
  name: 'John',

  loginOk() {
    alert(`${this.name}님이 로그인하였습니다.`);
  },

  loginFail() {
    alert(`${this.name}님이 로그인에 실패하였습니다.`);
  },

};

askPassword(user.loginOk.bind(user), user.loginFail.bind(user));
```

##### 화살표 함수
- 보통 잘 동작 · 가독성 ↑
- 복잡한 상황 <sub>(중간에 `user` 변수 변경 등)</sub> 발생 시
  - 오작동 가능성 有
```javascript
…
askPassword(() => user.loginOk(), () => user.loginFail());
```

### 로그인에 부분 함수 적용하기

##### `askPassword` <sub>(함수)</sub> 메서드 전달 코드 수정
- `user.login(true)`
  - `ok` 호출
- `user.login(false)`
  - `fail` 호출
```javascript
function askPassword(ok, fail) {
  let password = prompt("비밀번호를 입력해주세요.", '');
  if (password == "rockstar") {
    ok();
  } else {
    fail();
  }
}

let user = {
  name: 'John',

  login(result) {
    alert( this.name + (result ? ' 로그인 성공' : ' 로그인 실패') );
  }
};

askPassword(?, ?); // ?
```

<img src="../../images/commons/icons/circle-answer.svg" />

##### 1. 래퍼 함수 · 화살표 함수
- `askPassword` <sub>(함수)</sub>
  - 외부 변수에서 `user` 가져옴
```javascript
askPassword(() => user.login(true), () => user.login(false));
```

##### 2. 부분 적용 함수
- 컨텍스트 : `user`
- 올바른 1번째 인수 설정
```javascript
askPassword(user.login.bind(user, true), user.login.bind(user, false));
```
