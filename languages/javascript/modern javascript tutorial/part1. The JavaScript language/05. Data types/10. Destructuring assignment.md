구조 분해 할당
====

##### 기능
- 객체 · 배열 → 변수 <sub>(분해)</sub>

##### 용도
- 다수 매개변수 처리
- 매개변수 기본값
- 기타 등등

### 배열 분해
```javascript
let array = ["Bora", "Lee"]

// 구조 분해 할당
// - let firstName = array[0]
// - let surname   = array[1]
let [firstName, surname] = array;

alert(firstName); // Bora
alert(surname);   // Lee
```

##### `String.split` <sub>(메서드)</sub> 조합 <sub>(응용)</sub>
- 문자열 → 배열 분해 후 구조 분해 할당
```javascript
// - let firstName = split(…)[0]
// - let surname   = split(…)[1]
let [firstName, surname] = "Bora Lee".split(' ');
```

##### 분해 대상 복사 후 변수 분해 <sub>(수정 · 파괴 X)</sub>
- 직접 변수 할당 대비 코드량 ↓
```javascript
let array = […];

// 구조 분해 할당
let [firstName1, surname1] = array;

// 직접 할당
let firstName2 = array[0];
let surname2   = array[1];
```

<img src="../../images/commons/icons/circle-exclamation-solid.svg" /> **요소 무시**

##### `,` <sub>(쉼표)</sub> · 공백 조합
- 특정 요소 제외
```javascript
// 2번째 요소 제외
//     [0]    [1]  [2]
let [firstName, , title] = [
  "Julius", // [0] ([firstName)
  "Caesar", // [1] (제외)
  "Consul", // [2] (title)
  "of the Roman Republic"
];

alert( title ); // Consul (3번째 요소 · [2])
```

<br />

<img src="../../images/commons/icons/circle-exclamation-solid.svg" /> **`= …` <sub>(할당 연산자 우측)</sub> ← 이터러블 <sub>(객체)</sub>**

```javascript
// let a = "a";
// let b = "b";
// let c = "c";
let [a, b, c] = "abc"; // ["a", "b", "c"]

// let one   = 1;
// let two   = 2;
// let three = 3;
let [one, two, three] = new Set([1, 2, 3]);
```

<br />

<img src="../../images/commons/icons/circle-exclamation-solid.svg" /> **`… =` <sub>(할당 연산자 좌측)</sub> ← "할당 가능한" 전부**

```javascript
let user = {};

// 객체 내 프로퍼티 ← 문자열 배열 요소
// - user.name    = "Bora"; // split(…)[0]
// - user.surname = "Lee";  // split(…)[1]
[user.name, user.surname] = "Bora Lee".split(' ');

alert(user.name); // Bora
```

##### `Object.entries` <sub>(메서드)</sub> 조합 <sub>(객체 내 키 · 값 쌍)</sub>
```javascript
let obj = {
  name: "John",
  age: 30
};

// 객체 내 키 · 값 순회
for (let [key, value] of Object.entries(obj)) {

  // name: John, age: 30
  alert(`${key}:${value}`);
}
```

##### `Map` <sub>(객체)</sub> 조합 <sub>(키 · 값 쌍 순회)</sub>
```javascript
let map = new Map();

map.set("name", "John");
map.set("age", "30");

// Map (객체) 키 · 값 순회
for (let [key, value] of map) {

  // name: John, age: 30
  alert(`${key}:${value}`);
}
```

<img src="../../images/commons/icons/circle-exclamation-solid.svg" /> **변수 교환 트릭**

##### 두 변숫값 교환
```javascript
let guest = "Jane";
let admin = "Pete";

[guest, admin] = [admin, guest];

// Pete Jane
alert(`${guest} ${admin}`);
```

#### `...` : 나머지 요소 가져오기
- 나머지 요소 → 배열 변수 할당
  - 가장 마지막 위치
```javascript
let [name1, name2, ...rest] = ["Julius", "Caesar", "Consul", "of the Roman Republic"];

alert(name1); // Julius
alert(name2); // Caesar

// rest (배열)
alert(rest[0]); // Consul
alert(rest[1]); // of the Roman Republic
alert(rest.length); // 2
```

#### 기본값

##### 할당 변수 개수 \> 분해 대상 배열 길이
- 에러 X
- 할당 값 부재 시
  - `undefined` 취급
```javascript
let [firstName, surname] = [];

alert(firstName); // undefined
alert(surname);   // undefined
```

##### 기본값 <sub>(할당 값 X)</sub> 설정
```javascript
// 기본값
let [name = "Guest", surname = "Anonymous"] = ["Julius"];

alert(name);    // Julius (배열 값)
alert(surname); // Anonymous (기본값)
```

##### 표현식 · 함수 호출 할당
```javascript
// 값 미제공 때만 함수 호출
// name 의 prompt 한 번만 호출
let [surname = prompt('성을 입력하세요.'), name = prompt('이름을 입력하세요.')] = ["김"];

alert(surname); // 김 (배열 값)
alert(name);    // prompt 값
```

### 객체 분해하기

##### 우측
- 분해 대상 객체

##### 좌측
- 상응 객체 내 프로퍼티 패턴
```javascript
let {var1, var2} = {var1:…, var2:…}
```
```javascript
/* 객체 내 프로퍼티 키 목록 패턴 */
let options = {
  title: "Menu",
  width: 100,
  height: 200
};

let {title, width, height} = options;

alert(title);  // Menu
alert(width);  // 100
alert(height); // 200
```

##### 순서 중요성 ↓
```javascript
// `{…}` (중괄호) 내 순서
// - 바뀌어도 동일 동작
let {height, width, title} = {title: "Menu", height: 200, width: 100}
```

#### 더 복잡한 패턴

##### 객체 내 프로퍼티 · 변수 연결 조정
- `:` <sub>(콜론)</sub>
  - `prop: var`
```javascript
// 객체 내 프로퍼티 · 다른 이름 변수 연결
// - `:` (콜론) 사용
let options = {
  title: "Menu",
  width: 100,
  height: 200
};

// {객체 내 프로퍼티: 목표 변수}
let {width: w, height: h, title} = options;

/*
 width  → w
 height → h
 title  → title
 */
alert(title); // Menu
alert(w);     // 100
alert(h);     // 200
```

##### 기본값 <sub>(프로퍼티 X)</sub> 설정
```javascript
let options = {
  title: "Menu"
};

let {width = 100, height = 200, title} = options;

alert(title);  // Menu
alert(width);  // 100
alert(height); // 200
```

##### 표현식 · 함수 호출 할당
```javascript
let options = {
  title: "Menu"
};

let {width = prompt("width?"), title = prompt("title?")} = options;

alert(title);  // Menu
alert(width);  // prompt 창 입력값
```

##### `:` <sub>(콜론)</sub> · `=` <sub>(할당 연산자)</sub> 동시 사용
- 변수 연결 · 기본값
```javascript
let options = {
  title: "Menu"
};

let {width: w = 100, height: h = 200, title} = options;

alert(title);  // Menu
alert(w);      // 100
alert(h);      // 200
```

##### 특정 정보 뽑아오기
- 복잡한 객체
```javascript
let options = {
  title: "Menu",
  width: 100,
  height: 200
};

// title 만 변수로 뽑아내기
let {title} = options;

alert(title); // Menu
```

#### `...` <sub>(나머지 패턴)</sub>
- 나머지 프로퍼티 할당
- 모던 브라우저 지원
- 일부 구식 브라우저 <sub>(IE 등)</sub> 미지원
  - 바벨 <sub>(Babel)</sub> 이용
```javascript
let options = {
  title: "Menu",
  height: 200,
  width: 100
};

// 나머지 프로퍼티들
// - rest
let {title, ...rest} = options;

// rest → {height: 200, width: 100} 할당
alert(rest.height); // 200
alert(rest.width);  // 100
```

##### 기존 변수 사용
```javascript
/* 잘못된 코드 */
let title, width, height;

// SyntaxError: Unexpected token '='
// 코드 블록으로 인식
{title, width, height} = {title: "Menu", width: 200, height: 100};
```

##### `{…}` <sub>(중괄호)</sub>
- 주요 코드 흐름 상 존재 시 <sub>(표현식 내부 X)</sub>
  - 코드 블록 인식
- 코드 블록 본래 용도
  - 문 <sub>(statement)</sub> 묶기
```javascript
{
  // 코드 블록
  let message = "Hello";
  …
  alert( message );
}
```

##### `(…)` <sub>(소괄호)</sub>
- 할당문 감싸기
- `(…)` <sub>(소괄호)</sub> 내 `{…}` <sub>(중괄호)</sub>
  - 코드 블록 X
  - 표현식 해석
```javascript
let title, width, height;

({title, width, height} = {title: "Menu", width: 200, height: 100});

alert( title ); // Menu
```

### 중첩 구조 분해

##### 중첩 객체 · 배열
- 더 복잡한 패턴 사용
```javascript
let options = {
  size: { // 중첩 객체
    width: 100,
    height: 200
  },

  // 중첩 배열
  items: ["Cake", "Donut"],
  extra: true
};

/*
 대입 연산자 좌측 패턴
 - options (객체) 동일 구조
 코드 다수 줄 걸쳐 작성
 - 의도하는 바 명확히 드러냄
 */
let {

  // size (중첩 객체)
  size: {
    width,
    height
  },

  // items (중첩 배열)
  items: [item1, item2],

  // title (프로퍼티) 부재 시
  // - 기본값 사용
  title = "Menu"
} = options;

alert(title);   // Menu
alert(width);   // 100
alert(height);  // 200
alert(item1);   // Cake
alert(item2);   // Donut
// alert(extra) // extra 미할당

// 중첩 객체 · 배열 전용 변수 X (할당 X)
// - 중첩 객체 · 배열 내 정보만 할당
```

### 똑똑한 함수 매개변수

##### 함수 매개변수
- 종종 선택적으로 사용
```javascript
function showMenu(title = "Untitled", width = 200, height = 100, items = []) {
  …
}
```

##### 문제 발생
- 인수 전달 순서 틀림
- 다수 매개변수 기본값 설정 시
  - 빈 인수 불필요한 전달
```javascript
// 기본값 설정 시
// - 다수 undefined 전달
// - 가독성 ↓
showMenu("My Menu", undefined, undefined, ["Item1", "Item2"])
```

##### 객체로 모아 전달
- 함수 객체 분해하여 변수 할당
```javascript
// 함수에 전달할 객체
let options = {
  title: "My menu",
  items: ["Item1", "Item2"]
};

// 전달받은 객체 분해 후
// - 변수 즉시 할당
function showMenu({title = "Untitled", width = 200, height = 100, items = []}) {

  // title, items  → options (객체) 값
  // width, height → 기본값

  // My Menu 200 100
  alert( `${title} ${width} ${height}` );

  // Item1, Item2
  alert( items );
}

showMenu(options);
```

##### 중첩 객체 · `:` <sub>(콜론)</sub> 조합
- 더 복잡한 구조 분해
```javascript
let options = {
  title: "My menu",
  items: ["Item1", "Item2"]
};

function showMenu({
  title = "Untitled",
  width: w = 100,  // width  → w
  height: h = 200, // height → h

  // items 1번째 요소 → item1
  // items 2번째 요소 → item2
  items: [item1, item2]
}) {

  // My Menu 100 200
  alert( `${title} ${w} ${h}` );

  alert( item1 ); // Item1
  alert( item2 ); // Item2
}

showMenu(options);
```

##### 똑똑한 함수 매개변수 문법
- 구조 분해 할당 문법 동일
```javascript
// incomingProperty → varName
// - 값 부재 시 defaultValue (기본값) 사용
function({
  incomingProperty: varName = defaultValue
  …
})
```

##### 매개변수 구조 분해
- 모든 인수 기본값 할당 시
  - `{}` <sub>(빈 객체)</sub> 명시적 전달
```javascript
showMenu({}); // 모든 인수 기본값 할당
showMenu();   // 에러 가능성 有
```

##### 어떤 경우든 분해할 것 존재 시
- 인수 전부 미전달 시 에러 X

##### 인수 전체 기본값
- `{}` <sub>(빈 객체)</sub> 설정
```javascript
function showMenu({title = "Menu", width = 100, height = 200} = {}) {
  alert( `${title} ${width} ${height}` );
}

showMenu(); // Menu 100 200
```

#### 구조 분해 할당
- 객체 · 배열
  - 변수 연결

##### 객체 분해
```javascript
let {prop : varName = default, ...rest} = object
```
- `prop` <sub>(`object` 프로퍼티)</sub> · `default` <sub>(부재 시)</sub>
  - `varName` <sub>(변수)</sub> 할당
- 나머지 프로퍼티들 <sub>(연결 변수 X)</sub>
  - `rest` <sub>(배열)</sub> 저장

##### 배열 분해
```javascript
let [item1 = default, item2, ...rest] = array
```
- 1번째 요소
  - `item1` <sub>(변수)</sub> 할당
- 2번째 요소
  - `item2` <sub>(변수)</sub> 할당
- 나머지 요소들
  - `rest` <sub>(배열)</sub> 저장
- 좌측 패턴 == 우측 구조 <sub>(`=` 할당 연산자 기준)</sub>
  - 복잡한 구조 내 데이터 연결 가능
    - 중첩 배열 · 객체

### 구조 분해 할당

##### 구조 분해 할당 사용해 작업 수행
- `name` <sub>(프로퍼티)</sub> 값

- `years` <sub>(프로퍼티)</sub> 값
  - `age` <sub>(변수)</sub> 할당
- `isAdmin` <sub>(프로퍼티)</sub> 값 · `false` <sub>(부재 시)</sub>
  - `isAdmin` <sub>(변수)</sub> 할당
```javascript
let user = {
  name: "John",
  years: 30
};

// `=` (할당 연산자) 좌측 코드 작성
// … = user

alert( name );    // John
alert( age );     // 30
alert( isAdmin ); // false
```

### 최대 급여 계산하기

##### `salaries` <sub>(객체)</sub>
- 급여 관련 정보 저장

##### `topSalary(salaries)` <sub>(함수)</sub> 작성
- 가장 많은 급여 받는 사람 이름 반환
```javascript
let salaries = {
  "John": 100,
  "Pete": 300,
  "Mary": 250
};
```

##### 조건
- 비어있는 `salaries` <sub>(객체)</sub>
  - `null` 반환
- 최대 급여 받는 사람 다수
  - 그 중 아무나 한 명 반환

##### 힌트
- `Object.entries` · 구조 분해
  - 키 · 값 쌍 순회

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

```javascript
function topSalary(salaries) {

  let max = 0;
  let maxName = null;

  for (const [name, salary] of Object.entries(salaries)) {
    if (max < salary) {
      max = salary;
      maxName = name;
    }
  }

  return maxName;
}
```
