### 요소 추가 · 제거

##### `delete` <sub>(연산자)</sub> 배열 사용 가능 <sub>(배열 == 객체)</sub>
- 특정 요소 1개 제거
  - 요소 개수 변화 X
- 사용 지양
  - 배열 메서드 사용
```javascript
let arr = ["I", "go", "home"];

// "go" (2번째 요소) 제거
delete arr[1];
//  arr = ["I",     , "home"];

arr[1];     // undefined
arr.length; // 3
```

#### [`splice`](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Array/splice) <sub>(배열 자체 수정)</sub>

##### 요소 추가 · 제거 · 교체
```javascript
array.splice(index[, deleteCount, elem1, …, elemN])
```
- `index`
  - 조작 시작점 인덱스
- `[deleteCount]`
  - 제거 요소 개수
- `[elem1, …, elemN]`
  - 추가 요소

##### 요소 제거
```javascript
let arr = ["I", "study", "JavaScript"];

// ① 인덱스 1 ("study") 시작점 설정
// ② 요소 1개 제거 ("study")
arr.splice(1, 1);
// ["I", "JavaScript"]
```

##### 요소 교체
```javascript
let arr = ["I", "study", "JavaScript", "right", "now"];

// ① 인덱스 0 ("I") 시작점 설정
// ② 요소 3개 제거 ("I" · "study" · "JavaScript")
// ③ 새 요소 추가
arr.splice(0, 3, "Let's", "dance");
// ["Let's", "dance", "right", "now"]
```

##### 반환 값
- 제거 요소 구성 배열
```javascript
let arr = ["I", "study", "JavaScript", "right", "now"];

// ① 인덱스 0 ("I") 시작점 설정
// ② 요소 2개 제거 ("I" · "study")
// ③ 제거 요소 구성 배열 반환 (변수 저장)
let removed = arr3.splice(0, 2);
// arr     = ["JavaScript", "right", "now"];
// removed = ["I", "study"]
```

##### 요소 추가
- `[deleteCount]` <sub>(제거 요소 개수 · 2번째 인수)</sub>
  - `0` 설정 <sub>(요소 제거 X)</sub>
```javascript
let arr = ["I", "study", "JavaScript"];

// ① 인덱스 2 ("JavaScript") 시작점 설정
// ② 새 요소 추가
arr.splice(2, 0, "complex", "language");
// ["I", "study", "complex", "language", "JavaScript"]
```

##### 배열 관련 메서드
- 음수 인덱스 사용 가능

```javascript
// -1 (배열 끝 1번째 요소) 부터 0개 요소 제거 후 3, 4 추가
let arr = [1, 2, 5];

// ② 새 요소 추가 (3 · 4)
arr.splice(-1, 0, 3, 4);
// [1, 2, 3, 4, 5]
```

#### [`slice`](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Array/slice)

##### 특정 범위 복사
- `String.slice` <sub>(문자열 메서드)</sub> 동작 유사
```javascript
array.slice([start, end])
```
- `[start]`
  - 범위 시작
- `[end]`
  - 범위 종료 <sub>(`[end]` 제외)</sub>
- 음수 인수 가능
  - 배열 끝 기준
- 인수 無
  - 배열 복사본 생성
```javascript
let arr = ["t", "e", "s", "t"];

//        s         e
// ["t", "e", "s", "t"]
arr.slice(1, 3); // e,s

//             s
// ["t", "e", "s", "t"]
arr.slice(-2);  // s,t
```

#### [`concat`](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Array/concat)

##### 배열 병합 <sub>(새 결과 배열 생성)</sub>
```javascript
array.concat(arg1, arg2 …)
```
- `arg1, arg2 …`
  - 배열 · 값
  - 개수 제한 X

##### 새 배열 내 병합 결과 저장 · 반환
- 호출 배열 <sub>(`array`)</sub> 내 모든 요소 복사
- 인수 <sub>(`arg1, arg2 …`)</sub> 복사
  - 배열 전달 시 모든 요소
```javascript
let arr = [1, 2];

// 1,2,3,4
// - [1, 2] + [3, 4]
arr.concat([3, 4]);

// 1,2,3,4,5,6
// - [1, 2] + [3, 4] + [5, 6]
arr.concat([3, 4], [5, 6]);

// 1,2,3,4,5,6
// - [1, 2] + [3, 4] + 5 + 6
arr.concat([3, 4], 5, 6);
```

##### 객체형 인수 <sub>(유사 배열 객체 등)</sub>
- 객체 자체 <sub>(참조 값)</sub> 복사 <sub>(분해 X)</sub>
```javascript
let arr = [1, 2];
let arrayLike = { // 유사 배열 객체 (배열 X)
  0: "something",
  length: 1
};

// 1,2,[object Object]
// - [1, 2] + arrayLike
let result = arr.concat(arrayLike);

result[2] === arrayLike; // true
```

##### `[Symbol.isConcatSpreadable]` <sub>(프로퍼티)</sub>
- 유사 배열 객체 배열 취급 여부 결정
  - 객체 내 프로퍼티 값 복사 가능
```javascript
let arr = [1, 2];
let arrayLike = { // 유사 배열 객체 (배열 X)
  0: "something",
  1: "else",
  [Symbol.isConcatSpreadable]: true,
  length: 2
};

// [1, 2] + {0: "something", 1: "else"}
// 1,2,something,else
arr.concat(arrayLike);
```

#### [`forEach`](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Array/forEach)

##### 전체 요소 대상 반복 작업 수행
```javascript
array.forEach(function(item, index, array) {…});
```
- 함수형 인수 <sub>(`function(item, index, array)`)</sub>
  - 전체 요소 대상 순차 실행
  - 반환 값 무시

```javascript
// 각 요소 alert 실행

["Bilbo", "Gandalf", "Nazgul"].forEach((item, index, array) => {
  `${item} is at index ${index} in ${array}`;
});
```

### 배열 탐색

#### `[i·lastI]ndexOf`

##### 배열 내 특정 요소 검색
```javascript
array.[i·lastI]ndexOf(item, [from])
```
- `item`
  - 검색 대상 <sub>(요소)</sub>
- `[from]` <sub>(숫자)</sub>
  - 검색 시작 위치 <sub>(인덱스)</sub>
- 반환 값
  - 인덱스 <sub>(검색 성공)</sub>
  - `-1` <sub>(검색 실패)</sub>
- 각각 반대 방향 진행
  - `indexOf` <sub>(정방향)</sub>
  - `lastIndexOf` <sub>(역방향)</sub>
```javascript
let arr = [1, 0, false];

arr.indexOf(0);     //  1
arr.indexOf(false); //  2
arr.indexOf(null);  // -1
```

#### [`includes`](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Array/includes)

##### 배열 내 특정 요소 유무 여부 검사
```javascript
array.includes(item, [from])
```
- `item`
  - 검색 대상 <sub>(요소)</sub>
- `[from]` <sub>(숫자)</sub>
  - 검색 시작 위치 <sub>(인덱스)</sub>
- 요소 위치 X
  - 배열 내 유무 여부만 확인
```javascript
let arr = [1, 0, false];

arr.includes(1); // true
```

##### `[i·lastI]ndexOf` vs `includes`
- 요소 검색 시 `===` <sub>(일치 연산자)</sub> 사용 <sub>(공통점)</sub>
  - ex\) `false`
    - 정확히 `false` 만 검색
    - `0` <sub>(falsy 값)</sub> 검색 X
- `NaN` 검색 여부 <sub>(차이점)</sub>
  - `[i·lastI]ndexOf` 불가능
  - `includes` 가능
```javascript
const arr = [NaN];

arr.indexOf(NaN);  // -1
arr.includes(NaN); // true
```

#### `find[Index]`

##### 특정 조건 부합 단일 요소 · 인덱스 검색
```javascript
array.find(function(item, index, array) {…})
```
- 함수형 인수 <sub>(`function(item, index, array)`)</sub>
  - 전체 요소 대상 순차 실행
  - 조건 부합 여부 반환
    - `true` <sub>(검색 성공 · 탐색 중단)</sub>
    - `false` <sub>(탐색 진행)</sub>
- 반환 값
  - 요소 · 인덱스 <sub>(검색 성공)</sub>
  - `undefined` · `-1` <sub>(검색 실패)</sub>
```javascript
let users = [
  {id: 1, name: "John"},
  {id: 2, name: "Pete"},
  {id: 3, name: "Mary"}
];
let user = users.find(item => item.id == 1);
// {id: 1, name: "John"}

user.name; // John
```

#### [`filter`](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Array/filter)

##### 특정 조건 부합 다수 요소 검색
```javascript
array.filter(function(item, index, array) {…})
```
- 함수형 인수 <sub>(`function(item, index, array)`)</sub>
  - 전체 요소 대상 순차 실행
  - 조건 부합 여부 반환
    - `true` <sub>(결과 배열 내 순차 삽입)</sub>
    - `false` <sub>(결과 배열 내 삽입 X)</sub>
- 반환 값
  - 조건 부합 요소 구성 배열 <sub>(검색 성공)</sub>
  - `[]` <sub>(빈 배열 · 검색 실패)</sub>
```javascript
let users = [
  {id: 1, name: "John"},
  {id: 2, name: "Pete"},
  {id: 3, name: "Mary"}
];
let someUsers = users.filter(item => item.id < 3);
/*
[
  {id: 1, name: "John"},
  {id: 2, name: "Pete"},
]
 */

someUsers.length; // 2
```

#### [`sort`](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Array/sort) <sub>(배열 자체 수정)</sub>

##### 요소 정렬
```javascript
array.sort([function(x, y) {…}])
```
- 함수형 인수 <sub>(`[function(x, y)]`)</sub>
  - 전체 요소 대상 순차 실행
  - 인접 요소 간 비교 수행
    - 비교 결과 반환 필수
- 반환 값
  - 재정렬된 <sub>(기존)</sub> 배열
  - 사용 빈도 ↓ <sub>(배열 자체 수정)</sub>

##### 기본 재정렬 <sub>(함수형 인수 미전달)</sub>
- 오름차순
- 요소 → 문자열 취급 <sub>(`string` 형 변환 후 재정렬)</sub>
  - 사전<sub>(편집)</sub> 순 비교 수행
    - ex\) `"15" < "2"`
```javascript
let arr = [15, 2, 1];

// 문자열 (사전편집 순) 비교 수행
arr.sort();
// 1, 15, 2
```

##### 커스텀 재정렬 <sub>(함수형 인수 전달)</sub>
- 정렬 함수 <sub>(정렬 기준 정의)</sub>
  - 요소 2개 비교 수행
  - 반환 값 필수
```javascript
let arr = [15, 2, 1];

// 오름차순
function compareNumeric(a, b) {
  if (a >  b) return  1; // 1번째 값 > 2번째 값
  if (a == b) return  0; // 두 값 동일
  if (a <  b) return -1; // 1번째 값 < 2번째 값
}

// 숫자 (오름차순) 비교 수행
arr.sort(compareNumeric);
// 1, 2, 15
```

##### 포괄적인 정렬 알고리즘 이용 <sub>(내부 구현)</sub>
- 보통 최적화된 [quicksort](https://en.wikipedia.org/wiki/Quicksort) 사용
- 함수형 인수 사용 <sub>(정렬 기준 설정)</sub>

##### 정렬 과정 내 비교 요소 확인 트릭
- 정렬 중 특정 요소 간 비교 다수 발생 가능
  - 비교 횟수 최소화 필요
```javascript
[1, -2, 15, 2, 0, 8].sort(function(a, b) {
  a + " <> " + b;

  return a - b;
});
```

<img src="../../images/commons/icons/circle-exclamation-solid.svg" /> **정렬 함수 내 모든 숫자 반환 가능**

##### 정렬 함수 내 반환 값 제약 無
- 양수
  - 1번째 인수 \> 2번째 인수
- 음수
  - 1번째 인수 \< 2번째 인수

##### 정렬 함수 코드 축약
```javascript
function …(a, b) {
  if (a >  b) return  1; // 1번째 값 > 2번째 값
  if (a == b) return  0; // 두 값 동일
  if (a <  b) return -1; // 1번째 값 < 2번째 값

  ↓↓↓

  return a - b;
}
```
```javascript
let arr = [15, 2, 1];

// 축약형 (간결)
arr.sort(function(a, b) { return a - b; });
// [1, 2, 15]

// 화살표 함수 (더 간결)
arr.sort( (a, b) => a - b );
// [1, 2, 15]
```

<img src="../../images/commons/icons/circle-exclamation-solid.svg" /> **문자열 정렬 시 `localeCompare` <sub>(메서드)</sub> 사용**

##### `String.localeCompare` <sub>(메서드)</sub>
- 유니코드 기준 글자 비교 수행
  - 문자열 배열 내 합성 글자 처리 가능
    - ex\) `Ö` 등

##### 독일어 국가명 배열 정렬
```javascript
let countries = ['Österreich', 'Andorra', 'Vietnam'];

countries.sort((a, b) => a > b ? 1 : -1);
// Andorra,Vietnam,Österreich
// - 비정상 정렬

countries.sort((a, b) => a.localeCompare(b));
// Andorra,Österreich,Vietnam
// - 정상 정렬
```

#### [`String.split`](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/String/split)

##### 문자열 → 배열 분리
```javascript
String.split([delim, limit])
```
- `delim` <sub>(문자열)</sub>
  - 구분자 <sub>(문자열 분리 기준)</sub>
  - `,` <sub>(쉼표 · 기본값)</sub>
- 반환 값
  - 배열 <sub>(분리된 문자열 저장)</sub>
```javascript
let names = 'Bilbo, Gandalf, Nazgul';
let arr = names.split(', ');

for (let name of arr) {

  // Bilbo에게 보내는 메시지
  // Gandalf에게 보내는 메시지
  // Nazgul에게 보내는 메시지
  `${name}에게 보내는 메시지`;
}
```

##### `[limit]` <sub>(숫자)</sub>
- 반환 배열 길이 제한 <sub>(길이 초과 요소 무시)</sub>
- 사용 빈도 ↓
```javascript
let arr = 'Bilbo, Gandalf, Nazgul, Saruman'.split(', ', 2);
// Bilbo, Gandalf
```

#### `reduce[Right]`

##### 전체 요소 대상 반복 작업 수행 후 결괏값 도출
```javascript
array.reduce(function(acc, item, index, array) {…}, [initial]);
```
- 함수형 인수 <sub>(`function(acc, …)`)</sub>
  - 전체 요소 대상 순차 실행
  - 각 요소 실행 결과
    - 다음 요소 실행 시 사용
  - `acc`
    - 이전 요소 실행 결과
- `[initial]`
  - 초깃값 <sub>(최초 호출 시 사용)</sub>
- 반환 값
  - 끝 요소 실행 결과
- 각각 반대 방향 진행
  - `reduce` <sub>(정방향)</sub>
  - `reduceRight` <sub>(역방향)</sub>

```javascript
function filterRangeInPlace(arr, a, b) {
  for (let i = 0; i < arr.length; i++) {
    let val = arr[i];

    // 범위 외부 요소 제거
    if (val < a || b < val) {
      arr.splice(i, 1);
      i--;
    }
  }
}
```

##### `this.methods` <sub>(프로퍼티)</sub>
- 메서드 추가

##### `calculate` <sub>(메서드)</sub>
- 예외 · `number` 형 변환 처리
  - 추후 좀 더 복잡한 표현식 추가 가능
```javascript
function Calculator() {

  this.methods = {
    "-": (a, b) => a - b,
    "+": (a, b) => a + b
  };

  this.calculate = function(str) {
    let split = str.split(' '),
      a  = +split[0],
      op =  split[1],
      b  = +split[2];

    if (!this.methods[op] || isNaN(a) || isNaN(b)) {
      return NaN;
    }

    return this.methods[op](a, b);
  };

  this.addMethod = function(name, func) {
    this.methods[name] = func;
  };
}
```

- `array` <sub>(배열)</sub> 내 요소 무작위 섞기
- 다수 실행 시 요소 정렬 순서 상이 필수
```javascript
let arr = [1, 2, 3];

shuffle(arr); // [3, 2, 1]
shuffle(arr); // [2, 1, 3]
shuffle(arr); // [3, 1, 2]
…
```

##### 주의
- 모든 순열 동일 확률 <sub>(생성 빈도)</sub> 보장

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

##### 간단한 방법
```javascript
function shuffle(array) {
  array.sort(() => Math.random() - 0.5);
}
```

##### `Math.random() - 0.5` 계산 결과
- 양수 · 음수 중 하나
  - 요소 무작위 재정렬
- `sort` <sub>(메서드)</sub> 용도 부적합
  - 순열 동일 확률 X
```javascript
function shuffle(array) {
  array.sort(() => Math.random() - 0.5);
}

// 전체 순열 빈도 카운트
let count = {
  '123': 0,
  '132': 0,
  '213': 0,
  '231': 0,
  '321': 0,
  '312': 0
};

for (let i = 0; i < 1000000; i++) {
  let array = [1, 2, 3];
  shuffle(array);
  count[array.join('')]++;
}

// 전체 순열 생성 빈도 출력
for (let key in count) {
  `${key}: ${count[key]}`;
}
```

##### 실행 결과
- 한쪽 쏠림 발생 <sub>(엔진별 상이)</sub>
```javascript

132: 124425
213: 249618
231: 124880

321: 125223
```

##### 쏠림 발생 이유
- `sort` <sub>(메서드)</sub> 실행 시 내부 동작
  - 블랙박스 내 수행 有
- 요소 간 비교 시 완전 무작위 확률
  - 블랙발스 예측 불가능
- JS 엔진별 내부 구현방식 상이
  - 혼란 가중

#### 다양한 방법 有

##### [피셔-예이츠 셔플](https://en.wikipedia.org/wiki/Fisher%E2%80%93Yates_shuffle) <sub>(Fisher-Yates shuffle)</sub> 알고리즘
- 역방향 진행 <sub>(처음 ← 끝)</sub>
- 요소 교체
  - 앞 임의 요소 ↔ 해당 요소
- 정렬 연산 無 <sub>(성능 ↑)</sub>
```javascript
function shuffle(array) {
  for (let i = array.length - 1; i > 0; i--) {

    // 무작위 인덱스 (0 ≤ j < i)
    let j = Math.floor(Math.random() * (i + 1));

    // array[i] · array[j] 교체
    let t = array[i];
    array[i] = array[j];
    array[j] = t;
  }
}

// 전체 순열 빈도 카운트
let count = {
  '123': 0,
  '132': 0,
  '213': 0,
  '231': 0,
  '321': 0,
  '312': 0
};

for (let i = 0; i < 1000000; i++) {
  let array = [1, 2, 3];
  shuffle(array);
  count[array.join('')]++;
}

// 전체 순열 생성 빈도 출력
for (let key in count) {
  `${key}: ${count[key]}`;
}
```

##### 실행 결과
- 모든 순열 유사 빈도 발생
```javascript

132: 166647
213: 166628
231: 167517
312: 166199
321: 166316
```

### 평균 나이 계산

##### 배열 내 `user` <sub>(객체)</sub> 프로퍼티

##### `getAverageAge(users)` <sub>(함수)</sub> 작성
- 평균 나이 <sub>(`user.age`)</sub> 반환

##### 평균 계산 공식
- `(age1 + age2 + … + ageN) / N`
```javascript
let arr = [
  {name: "John", age: 25},
  {name: "Pete", age: 30},
  {name: "Mary", age: 29},
];

// (25 + 30 + 29) / 3 = 28
getAverageAge(arr); // 28
```

```javascript
function getAverageAge(users) {
  return users.reduce((prev, user) => prev + user.age, 0) / users.length;
}
```

### 미중복 요소 검색

##### `unique(arr)` <sub>(함수)</sub> 작성
- `arr` <sub>(배열)</sub> 내 유일 요소 검색
```javascript
let strings = [
  "Hare",
  "Krishna",
  "Hare",
  "Krishna",
  "Krishna",
  "Krishna",
  "Hare",
  "Hare",
  ":-O"
];

unique(strings); // Hare, Krishna, :-O
```

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

##### 확인 작업 진행
1. 새 <sub>(결과)</sub> 배열 생성
2. 결과 배열 내 요소 유무 여부 확인
   - 有 → 무시
   - 無 → 삽입
```javascript
function unique(arr) {
  let result = [];

  for (let str of arr) {
    if (!result.includes(str)) {
      result.push(str);
    }
  }

  return result;
}
```

#### 성능상 문제 有

##### `result.includes(str)` <sub>(일치 여부 검색)</sub>
- `result` 순회
  - 각 요소 · `str` 비교
- `str` 일치 요소 無
  - `result` 내 요소 개수만큼 비교 진행
- 너무 큰 비교 횟수 <sub>(1억번 등)</sub>
  - 성능 ↓

- `Set` <sub>(내장 객체)</sub> 사용

### 배열 → 객체

##### 배열 내 `user` <sub>(객체)</sub> 프로퍼티
- `id`
- `name`
- `age`

##### `groupById(arr)` <sub>(함수)</sub> 작성
- `arr` <sub>(배열)</sub> 내 요소 → 객체 변환
  - 키
    - `id` <sub>(프로퍼티)</sub>
  - 값
    - 요소 <sub>(객체)</sub>
```javascript
let users = [
  {id: 'john', name: "John Smith",    age: 20},
  {id: 'ann',  name: "Ann Smith",     age: 24},
  {id: 'pete', name: "Pete Peterson", age: 31},
];

let usersById = groupById(users);
/*
usersById = {
  john: {id: 'john', name: "John Smith",    age: 20},
  ann : {id: 'ann',  name: "Ann Smith",     age: 24},
  pete: {id: 'pete', name: "Pete Peterson", age: 31},
}
*/
```

##### 참고
- `id` <sub>(프로퍼티)</sub> 유일 가정
- `reduce` <sub>(메서드)</sub> 사용

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

```javascript
function groupById(array) {
  return array.reduce((obj, value) => {
    obj[value.id] = value;
    return obj;
  }, {})
}
```
