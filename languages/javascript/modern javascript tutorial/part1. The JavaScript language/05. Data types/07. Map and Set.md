### [`Map`](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Map) <sub>(내장 객체)</sub>

##### 키 · 값 쌍 저장 <sub>(≒ 일반 객체)</sub>
- 다양한 자료형 키 허용 <sub>(≠ 일반 객체)</sub>

##### 주요 메서드 · 프로퍼티

|메서드 · 프로퍼티|기능|
|---|---|
|`new Map([iter])`|`Map` <sub>(객체)</sub> 생성|
||`[iter]` <sub>(초기화 객체)</sub><br />- 이터러블 <sub>(키 · 값 쌍 구성 객체)</sub>|
|`set(key, value)`|`key` · `value` 저장 후 자신 반환|
|`get(key)`|`key` 有 → `key` 해당 값 반환|
||`key` 無 → `undefined` 반환|
|`has(key)`|`key` 유무 여부 확인|
|`delete(key)`|`key` 해당 값 제거|
|`clear()`|모든 요소 제거|
|`size`|요소 개수 반환|

##### `Map` <sub>(객체)</sub> 키
- `string` 형 변환 X
  - 자료형 제약 無
```javascript
let map = new Map();

map.set('1', 'str1');   // string  키
map.set(1, 'num1');     // number  키
map.set(true, 'bool1'); // boolean 키

/*
 일반 객체 키
 - string 형 변환
 Map (객체) 키
 - 형 변환 X (그대로 유지)
 */
alert( map.get(1) );   // 'num1'
alert( map.get('1') ); // 'str1'
alert( map.size );     // 3
```

### `Map` <sub>(객체)</sub> 내 요소 반복 작업

```javascript
  alert(amount); // 500, 350, 50
}

// 키 · 값 쌍 순회
for (let entry of recipeMap) {
//  (let entry of recipeMap.entries()) {
  alert(entry);  // cucumber,500 …
}
```

### [`Object.entries`](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Object/entries) <sub>(내장 메서드)</sub>

##### `new Map` <sub>(생성자)</sub> 전달 가능 인수 <sub>(초기화 용도)</sub>
- 배열 <sub>(키 · 값 쌍 요소 구성)</sub>
- 이터러블 <sub>(객체)</sub>
```javascript
// 배열 (키 · 값 쌍 요소 구성) 전달
let map = new Map([
  ['1' , 'str1' ],
  [ 1  , 'num1' ],
  [true, 'bool1'],
]);

alert( map.get('1') ); // str1
```
- 일반 객체 불가능
  - `Object.entries` <sub>(메서드)</sub> 필요
```javascript
let obj = {
  name: "John",
  age: 30
};

let map1 = new Map(obj); // 에러

// 일반 객체 → 배열 (키 · 값 쌍 요소 구성) → Map (객체)

/*
 [
   ["name","John"],
   ["age" , 30]
 ]
 */

```

### [`Object.fromEntries`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object/fromEntries) <sub>(내장 메서드)</sub>

- `{key: value, …}`

```javascript
  ['orange', 2],
  ['meat',   4],

/*
 {
   banana: 1,
   orange: 2,
   meat:   4,
```

```javascript
for (let user of set) {
  alert(user.name); // John, Pete, Mary
}
```

```javascript
let set = new Set(["oranges", "apples", "bananas"]);
```

- 인수 3개

    - Map ↔ Set 교체 용이
 */
```javascript
let set = new Set(["oranges", "apples", "bananas"]);

set.forEach((value, valueAgain, set) => {
```

##### 글자 단위 쪼갠 후 알파벳 순 정렬
- 동일 그룹 내 애너그램 → 동일 단어

|알파벳 순|조합 1|조합 2|조합 3|
|---|---|---|---|
|anp|nap|pan||
|aer|ear|are|era|
|aceehrst|cheaters|hectares|teachers|

##### 키 하나 · 값 하나 저장
- `Map` 키
  - 알파벳 순 정렬 글자
```javascript
function aclean(arr) {
  let map = new Map();

  for (let word of arr) {

    // ① 글자 단위 쪼개기
    // ② 알파벳 순 정렬
    // ③ 문자열로 합치기
    let sorted = word // PAN
      .toLowerCase()  // pan
      .split('')      // ['p','a','n']
      .sort()         // ['a','n','p']
      .join('');      // anp

    /*
    정렬 이후 동일 글자 구성 단어 등장 시
    - 값 덮어씌워짐 (키 동일)
      - 동일 글자 구성 단어 단 한 번만 저장

    map.values()
    - Map 내 값 담은 반복 가능한 객체 반환
    Array.from
    - 반복 가능한 객체 → 배열
     */
    // 'PAN' · 'nap' → 'anp'
    map.set(sorted, word);
  }

  return Array.from(map.values());
}

let arr = [
  "nap",
  "teachers",
  "cheaters",
  "PAN",
  "ear",
  "era",
  "hectares"
];

alert( aclean(arr) );
```

##### 일반 객체 사용 가능
- 키 타입
  - `string`
```javascript
function aclean(arr) {
  let obj = {};

  for (let i = 0; i < arr.length; i++) {
    let sorted = arr[i] // PAN
      .toLowerCase()    // pan
      .split("")        // ['p','a','n']
      .sort()           // ['a','n','p']
      .join("");        // anp

    obj[sorted] = arr[i];
  }

  return Object.values(obj);
}

let arr = [
  "nap",
  "teachers",
  "cheaters",
  "PAN",
  "ear",
  "era",
  "hectares"
];

alert( aclean(arr) );
```

##### `map.keys()` 반환 값
- 배열 X
- 이터러블 <sub>(객체)</sub>

##### `Array.from`
- `Map` → 배열
```javascript
let map = new Map();

map.set("name", "John");

let keys = Array.from(map.keys());

keys.push("more");

alert(keys); // name, more
```
