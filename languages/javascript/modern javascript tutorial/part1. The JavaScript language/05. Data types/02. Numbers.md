`number` <sub>(숫자형)</sub>
====

##### 일반적인 숫자
- 배정밀도 부동소수점 숫자
  - <sub>double precision floating point number</sub>
- 64비트 형식 IEEE-754 저장
- 범위 제약
  - -2<sup>53</sup> \< n \< 2<sup>53</sup>

##### `BigInt`
- 임의 길이 정수
- 아주 특별한 경우에만 사용

### 다양한 숫자 입력 방법

##### 숫자 마지막 `e` · `-e`
- `0` 개수
```javascript
// 10 거듭제곱
1e3     === 1    * 1000    // 1000
1.23e6  === 1.23 * 1000000 // 1230000

// 음수 10 거듭제곱
1e-3    === 1    / 1000    // 0.001
1.23e-6 === 1.23 / 1000000 // 0.00000123
```

#### 16 · 2 · 8진수

##### `0x` <sub>(16진수)</sub>
- 다양한 곳 사용
  - 색 표현
  - 문자 인코딩
  - 기타 등등

##### `0b` · `0o` <sub>(2 · 8진수)</sub>
- 아주 드물게 사용
```javascript
// 16진수 (대 · 소문자 구별 X)
alert( 0xff ); // 255
alert( 0xFF ); // 255

// 2 · 8진수
let a = 0b11111111; // 255 의 2진수
let b = 0o377;      // 255 의 8진수
alert( a == b );    // true (진법만 상이)
```

### `toString(base)`

##### `base` 진법 표현 후
- `string` 형 변환 · 반환
```javascript
let num = 255;

alert( num.toString(16) ); // ff
alert( num.toString(2)  ); // 11111111
```

##### `base`
- 2 ~ 36
- 10 <sub>(기본값)</sub>

|`base`|0 ~ 9|10 ~&nbsp;|용도|
|:---:|:---:|:---:|---|
|16|0 ~ 9|A ~ F|- 16진수 색<br />- 문자 인코딩 등|
|2|0 · 1||- 연산 디버깅 등|
|36|0 ~ 9|A ~ Z|- 긴 숫자 식별자 축약 등<br />　- ex\) url 축약|

```javascript
/* base = 36 */

// `.` (점) 2개
alert(  123456..toString(36) ); // 2n9c

// `(…)` (괄호)
alert( (123456).toString(36) ); // 2n9c
```

- `.` <sub>(점)</sub> 2개 사용

##### `.` <sub>(점)</sub> 1개 사용 시

### 어림수 구하기

##### 어림수 관련 내장 함수

|대상|`floor`|`ceil`|`round`|`trunc`|
|:---:|:---:|:---:|:---:|:---:|
|소수점|첫째 자리<br />내림 <sub>(버림)</sub>|첫째 자리<br />올림|첫째 자리<br />반올림|무시<br /><sub>(IE 미지원)</sub>|
|`3.1`|`3`|`4`|`3`|`3`|
|`3.6`|`3`|`4`|`4`|`3`|
|`-1.1`|`-2`|`-1`|`-1`|`-1`|
|`-1.6`|`-2`|`-1`|`-2`|`-1`|

#### 소수점 n번째 기준 어림수 구하기 <sub>(2가지)</sub>

##### 1. 곱하기 · 나누기
1. 대상 <sub>(숫자)</sub> * 10<sup>n</sup>
2. 어림수 내장 함수 호출
3. 대상 <sub>(숫자)</sub> / 10<sup>n</sup>
```javascript
// 소수점 2번째 수 기준
let num = 1.23456;

/*
 ① 1.23456,
 ② 123.456, 1.23456 * 100
 ③ 123    , Math.floor(123.456)
 ④ 1.23   , 123     / 100
 */
alert( Math.floor(num * 100) / 100 );
```

##### 2. `toFixed(n)`
- 최근접 값 올림 · 버림
  - `Math.round` 유사
- 반환 값
  - 문자열
  - `number` 형 변환 필요
    - `+` <sub>(단항 덧셈 연산자)</sub>
    - `Number()` <sub>(생성자)</sub>
- 소수부 길이 \< 인수
  - 끝에 `'0'` 추가
```javascript
let num1 = 12.34;
alert( num1.toFixed(1) ); // "12.3"

let num2 = 12.36;
alert( num2.toFixed(1) ); // "12.4"

/* number 형 변환 */
// + (단항 덧셈 연산자)
// - 12.4
alert( +num2.toFixed(1) );

// Number() (생성자)
// - 12.4
alert( Number(num2.toFixed(1)) );

/* 소수부 길이 < 인수 */
let num3 = 12.34;

// 5 (소수부 길이) 맞춤
// - "12.34000"
alert( num3.toFixed(5) );
```

#### 문제 해결

##### 1. `toFixed(n)`
- 어림수 생성 메서드

  - 인수 개수 \> 반환 값 소수자리 수
    - 나머지 `'0'` 채움

```javascript
// + (단항 덧셈 연산자)
// - 문자열 → number (형 변환)
```

### `isNaN` · `isFinite`

##### `NaN`
- 에러 표현 값

- 특수 숫자 값
- 그 어떤 숫자보다 큼 · 작음

##### 공통점
- `number`
- 비정상적인 숫자 표현

##### 구분 위한 특별 함수
- `isNaN(value)`
- `isFinite(value)`

##### `isNaN(value)`
- 인수 → 숫자 형 변환 후
  - `NaN` 테스트
```javascript
alert( isNaN(NaN) );   // true
alert( isNaN("str") ); // false

// 자기 자신 포함 그 어떤 값과도 동일 X
// - 함수 필요 이유
```

##### `isFinite(value)`
- 인수 → 숫자 형 변환 후
  - 일반 숫자 : `true` 반환
```javascript
alert( isFinite("15") );     // true

// 종종 문자열 일반 숫자 검증에 사용
// 숫자 아닌 값 · Infinity · -Infinity
// - false
```

### `parse[Int·Float]`

##### `+` <sub>(단항 덧셈 연산자)</sub> · `Number()` <sub>(생성자)</sub>
- `number` 형 변환 시
  - 규칙 엄격 적용
- 숫자 아닌 피연산자
  - 형 변환 실패
- 규칙 엄격 적용 예외
  - 문자열 앞 · 뒤 공백 무시
```javascript
alert( +"100px" ); // NaN
```

##### `parse[Int·Float]`
- 불가능할 때까지 문자열 숫자 읽음
- 숫자 읽는 중 오류 발생 시
  - 수집된 숫자 반환
- `parseInt`
  - 정수
- `parseFloat`
  - 부동소수점 숫자
```javascript
alert( parseInt('100px') );    // 100
alert( parseFloat('12.5em') ); // 12.5

// 정수 부분만 반환
alert( parseInt('12.3') );     // 12

// 2번째 점에서 숫자 읽기 중단
alert( parseFloat('12.3.4') ); // 12.3
```

##### 읽을 수 있는 숫자 부재 시
- `NaN` 반환
```javascript
// a 에서 숫자 읽기 중단
// - 수집된 숫자 없음
alert( parseInt('a123') ); // NaN
```

##### 입력값 無 · 입력 취소 <sub>(입력값 : `null`)</sub> 경우
- `""` <sub>(빈 문자열)</sub> · `null` → `0` <sub>(`number` 형 변환)</sub>
  - `number` 형 변환 X
  - `null` 반환 처리
```javascript
function readNumber() {
  let num;

  do {
    num = prompt("Enter a number please?", 0);
  } while ( !isFinite(num) );

  if (num === null || num === '') return null;

  return +num;
}

alert(`Read: ${readNumber()}`);
```

### `min` ~ `max` 사이 임의 숫자 생성하기

##### `random(min, max)` <sub>(함수)</sub> 작성
- 특정 범위 내 임의 부동소수점 숫자 생성
  - `min` ≤ x \< `max`
- `Math.random()` <sub>(내장 함수)</sub>
  - 특정 범위 무작위 값 생성
    - `0` ≤ x \< `1`
```javascript
alert( random(1, 5) ); // 1.2345623452
alert( random(1, 5) ); // 3.7894332423
alert( random(1, 5) ); // 4.3435234525
```

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

##### 값 변환
- &nbsp;&nbsp;`0` &nbsp;≤ x \< &nbsp;&nbsp;`1`
- `min` ≤ x \< `max`

#### 변환 과정

##### 1. (`0` ≤ x \< `1`) * `(max - min)`
- `0` ≤ x \< `1`
- `0` ≤ x \< `(max - min)`

##### 2. `min` 값 더하기
- &nbsp;&nbsp;`0` &nbsp;≤ x \< `(max - min)`
- `min` ≤ x \< &nbsp;`max`
```javascript
function random(min, max) {
  return min + Math.random() * (max - min);
}

alert( random(1, 5) );
alert( random(1, 5) );
alert( random(1, 5) );
```

### `min` ~ `max` 사이 임의 정수 생성하기

##### `randomInteger(min, max)` <sub>(함수)</sub> 작성
- 특정 범위 내 임의 정수 생성
    - `min` ≤ x ≤ `max`
- 임의 정수 생성 확률
  - 모두 동일 보장
- 이전 과제 참고
```javascript
alert( randomInteger(1, 5) ); // 1
alert( randomInteger(1, 5) ); // 3
alert( randomInteger(1, 5) ); // 5
```

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

##### 단순한 · 잘못된 해결
- 임의 숫자 <sub>(`min` ~ `max`)</sub> 생성 후 소수부 제거
```javascript
function randomInteger(min, max) {
  let rand = min + Math.random() * (max - min);
  return Math.round(rand);
}

alert( randomInteger(1, 3) );
```

##### 함수 정상 작동 · 결과 부정확
- `min` · `max` <sub>(경곗값)</sub> 생성 확률
  - 다른 정수의 절반

##### 함수 다수 호출 시
- `2` 가장 많이 생성

##### `Math.round()`
- 소수부 제거 과정
```javascript
1   ~ 1.4999999999 → 1
1.5 ~ 2.4999999999 → 2
2.5 ~ 2.9999999999 → 3
```

#### 올바른 해결
- 다양한 방법 존재

##### 1. 경곗값 조정
- 경곗값 ± 0.5
  - 생성 확률 동일화
```javascript
function randomInteger(min, max) {

  // (min - 0.5) ≤ x < (max + 0.5)
  let rand = min - 0.5 + Math.random() * (max - min + 1);

  return Math.round(rand);
}

alert( randomInteger(1, 3) );
```

##### 2. `Math.floor`
- `min` ≤ x \< (`max + 1`)
```javascript
function randomInteger(min, max) {

  // min ≤ x < (max + 1)
  let rand = min + Math.random() * (max - min + 1);

  return Math.floor(rand);
}

alert( randomInteger(1, 3) );
```

##### 결과
```javascript
1 ~ 1.9999999999 → 1
2 ~ 2.9999999999 → 2
3 ~ 3.9999999999 → 3
```
