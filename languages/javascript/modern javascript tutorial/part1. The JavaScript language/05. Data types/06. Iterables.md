이터러블 <sub>(객체)</sub>
====

##### 배열 일반화 객체
- `for…of` <sub>(반복문)</sub> 적용 가능
  - 배열
  - 문자열
  - 다수 내장 객체

##### 컬렉션 <sub>(목록 · 집합 등)</sub> 객체 <sub>(배열 X)</sub>
- `for…of` 문법 적용 시
  - 컬렉션 순회 유용

### `[Symbol.iterator]` <sub>(메서드)</sub>

##### 이터러블 객체화 담당 메서드
- `for…of` <sub>(반복문)</sub> 사용 가능

#### `for…of` <sub>(반복문)</sub> 진행 과정

##### 1. `[Symbol.iterator]` <sub>(메서드)</sub> 호출
- 메서드 부재 시 에러
- 이터레이터 <sub>(`next` 메서드 구현 객체)</sub> 반환
  - 실제 반복문 동작 대상

##### 2. `next` <sub>(메서드)</sub> 호출 <sub>(다음 값 필요 시)</sub>
- 값 객체 반환

##### 3. `{done: boolean, value: any}` <sub>(값 객체)</sub>
- `done` <sub>(`boolean`)</sub>
  - 반복 종료 여부

|`done` 값|의미|
|:---:|---|
|`true`|반복 종료|
|`false`|다음 값 → `value` 저장|

##### `range` <sub>(객체)</sub>
- 숫자 간격 표현
```javascript
let range = {
  from: 1,
  to: 5
};

// ①
// 최초 [Symbol.iterator] (메서드) 호출
// - 이터레이터 (객체) 반환
//   - 실제 반복문 동작 대상
range[Symbol.iterator] = function() {
  return {
    current: this.from,
    last: this.to,

    // ②
    // 각 반복 시 next (메서드) 호출
    // - 값 객체 반환
    next() {

      // ③
      // {done: …, value: …} (객체)
      // - 값 저장 · 반복 여부 결정
      if (this.current <= this.last) {
        return {done: false, value: this.current++};
      } else {
        return {done: true};
      }
    }
  };
};

// 결과
for (let num of range) {
  num; // 1, 2, 3, 4, 5
}
```

##### _Separation of concern_ <sub>(SoC)</sub>
- '관심사 분리'
  - 이터러블 <sub>(객체)</sub> 핵심

##### `range` <sub>(객체)</sub>
- `next` <sub>(메서드)</sub> 無
- `[Symbol.iterator]` <sub>(메서드)</sub> 호출 시
  - iterator <sub>(객체)</sub> 반환

##### iterator <sub>(객체)</sub>
- `next` <sub>(메서드)</sub> 구현
  - 반복 내 사용 값 생성

##### 객체 분리 실현
- 반복 대상 객체 <sub>(`range`)</sub>
- iterator <sub>(객체)</sub>

##### 반복 대상 객체 <sub>(`range`)</sub> + iterator <sub>(객체)</sub>
- 코드 단순화
```javascript
let range = {
  from: 1,
  to: 5,
  current: 0,

  [Symbol.iterator]() {
    this.current = this.from;
    return this;
  },

  next() {
    if (this.current <= this.to) {
      return {done: false, value: this.current++};
    } else {
      return {done: true};
    }
  }
};

for (let num of range) {
  num; // 1, 2, 3, 4, 5
}
```

##### `range[Symbol.iterator]` <sub>(메서드)</sub>
- `range` <sub>(객체)</sub> 자체 반환
  - `next` <sub>(객체)</sub> 구현
  - `this.current` <sub>(프로퍼티)</sub>
    - 반복 진행 값 저장
  - 코드량 ↓

##### `for…of` <sub>(반복문)</sub> 동시 2개 사용 불가능 <sub>(단점)</sub>
- iterator <sub>(객체)</sub> 1개 有 <sub>(반복 대상 객체 · `range`)</sub>
  - 두 반복문 반복 상태 공유
- 반복문 동시 2개 사용 경우
  - 발생 빈도 ↓↓

- 값 무한 반환 가능 <sub>(정상 동작)</sub>

### 문자열 == 이터러블 <sub>(객체)</sub>

##### 배열 · 문자열
- 내장 이터러블 <sub>(객체)</sub>

##### `for…of` <sub>(반복문)</sub>
- 문자열 내 각 글자 순회
```javascript
for (let char of "test") {

  // 글자당 1번 실행 (총 4회 호출)
  char; // t,e,s,t
}
```
- 서로게이트 쌍 정상 동작
```javascript
for (let char of '𝒳😂') {
  char; // 𝒳, 😂
}
```

### iterator <sub>(객체)</sub> 명시적 호출 <sub>(발생 빈도 ↓)</sub>

##### 수동 반복
- 문자열 내 각 글자 순회
  - `for…of` <sub>(반복문)</sub> 동일 동작
- 문자열 iterator (객체) 직접 생성 후
  - 값 수동 접근
```javascript
let str = "Hello";

// for…of (반복문) 사용 동일 작업
// - for (let char of str) { char };
let iterator = str[Symbol.iterator]();

while (true) {
  let result = iterator.next();

  if (result.done) {
    break;
  }

  // 글자 1개씩 출력
  result.value; // h,e,l,l,o
}
```

##### 세밀한 반복 과정 통제 가능
- ex\) 반복 과정 쪼개기
  1. 반복 시작
  2. 잠시 중단 · 타 작업 수행
  3. 다시 반복 시작
  4. 기타 작업 등

### 이터러블 <sub>(객체)</sub> vs 유사 배열 객체

##### 이터러블 <sub>(객체)</sub>
- `[Symbol.iterator]` <sub>(메서드)</sub> 구현

##### 유사 배열 객체
- 배열 형태 유사
  - 숫자 인덱스
  - `length` <sub>(프로퍼티)</sub>

##### 문자열 == 혼합 객체 <sub>(이터러블 + 유사 배열)</sub>
- `for…of` <sub>(반복문)</sub> 사용 가능
- 숫자 인덱스 · `length` <sub>(프로퍼티)</sub> 有

<br />

##### 이터러블 <sub>(객체)</sub> ≠ 유사 배열 객체
```javascript
// 유사 배열 객체
// - 숫자 인덱스 · length (프로퍼티) 有
let arrayLike = {
  0: "Hello",
  1: "World",
  length: 2,
};

// 에러 ([Symbol.iterator] 메서드 無)
for (let item of arrayLike) {…}
```

##### 이터러블 · 유사 배열 객체 ≠ 배열
- 배열 메서드 미지원 <sub>(편의성 ↓)</sub>

### [`Array.from`](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Array/from) <sub>(메서드)</sub>

##### 문법
```javascript
Array.from(obj[, mapFn, thisArg])
```
- `obj`
  - 이터러블 <sub>(객체)</sub>
  - 유사 배열 객체
- `[mapFn]` <sub>(매핑 함수)</sub>
  - 새 배열 내 요소 추가 전 적용
- `[thisArg]`
  - 각 요소 `this` 지정

##### 이터러블 · 유사 배열 객체 → 배열 변환
- 배열 메서드 사용 가능
- 이터러블 · 유사 배열 객체 여부 검사
  - 새 배열 생성 후 객체 내 모든 요소 복사

##### 이터러블 <sub>(객체)</sub> → 배열
```javascript
// 이터러블 (객체)
let range = {
  …
  [Symbol.iterator]() {…}
  next() {…}
};

let array = Array.from(range);

array.pop(); // 5 (배열 메서드 정상 동작)
```

##### 유사 배열 객체 → 배열
- `for…of` <sub>(반복문)</sub> 사용 가능
  - 배열 == 이터러블 <sub>(객체)</sub>
```javascript
// 유사 배열 객체
let arrayLike = {
  0: "Hello",
  1: "World",
  length: 2,
};

let array = Array.from(arrayLike);

array.pop(); // World (배열 메서드 정상 동작)

for (let item of array) {
  item; // Hello
}
```

##### `[mapFn]` <sub>(매핑 함수)</sub> 사용
- 거듭제곱 연산 수행
```javascript
let arrayLike = {
  0: 1,
  1: 2,
  2: 3,
  3: 4,
  4: 5,
  length: 2,
};

// 각 숫자 제곱 연산 후 저장
let array = Array.from(arrayLike, num => num * num);

array; // 1,4,9,16,25
```

##### 문자열 → 배열
- `Array.from` <sub>(메서드)</sub>
  - 문자열 자체 이터러블 <sub>(객체)</sub> 속성 이용
    - 서로게이트 쌍 정상 적용
```javascript
let str = '𝒳😂';

// str (문자열) → 배열
let chars = Array.from(str);
/* Array.from (메서드) 내부 실행 반복문
let chars = [];
for (let char of str) {
  chars.push(char);
}
 */

chars[0];     // 𝒳
chars[1];     // 😂
chars.length; // 2
```

##### 커스텀 `slice` <sub>(함수)</sub> 구현 가능
- 서로게이트 쌍 정상 처리
```javascript
function slice(str, start, end) {
  return Array.from(str).slice(start, end).join('');
}

let str = '𝒳😂𩷶';

slice(str, 1, 3); // 😂𩷶

// 순수 내장 메서드
// - 서로게이트 쌍 미지원
str.slice(1, 3); // 쓰레깃값 출력 (타 영역 특수 값)
```

요약
====

##### 이터러블 <sub>(객체)</sub>
- `for…of` <sub>(반복문)</sub> 사용 가능 객체
- `[Symbol.iterator]` <sub>(메서드)</sub> 구현 필수
  - iterator <sub>(객체)</sub> 반환
    - 반복 과정 처리

##### iterator <sub>(객체)</sub>
- `next` <sub>(메서드)</sub> 구현 필수
  - 값 객체 반환 필수
```javascript
{
  // 반복 종료 여부
  done: Boolean,

  // 다음 값
  value: any
}
```

##### `[Symbol.iterator]` <sub>(메서드)</sub>
- `for…of` <sub>(반복문)</sub> 의해 자동 호출
  - 명시적 호출 가능
- 내장 이터러블 <sub>(객체)</sub> 내 구현
  - 문자열
  - 배열
  - 기타 등등

##### 문자열 iterator <sub>(객체)</sub>
- 서로게이트 쌍 지원

##### 유사 배열 객체
- 배열 형태 유사
  - 숫자 인덱스
  - `length` <sub>(프로퍼티)</sub>
- 다양한 프로퍼티 · 메서드 구현 가능
- 배열 내장 메서드 無

##### 대다수 배열 내장 메서드 동작 대상 <sub>(명세서)</sub>
- 이터러블 · 유사 배열 객체 <sub>(배열 X)</sub>
  - 더 추상적 <sub>(범용성 ↑)</sub>

##### `Array.from(obj[, mapFn, thisArg])`
- 이터러블 · 유사 배열 객체 → 배열 변환
  - 배열 메서드 사용 가능
- `[mapFn]` · `[thisArg]`
  - 각 요소 매핑 함수 적용
