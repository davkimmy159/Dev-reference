`async` · `await`
====

### `async` 함수

##### `async` <sub>(키워드)</sub>
- `function` <sub>(키워드)</sub> 앞 위치
```javascript
async function f() {…}
```

##### 항상 `Promise` <sub>(객체)</sub> 반환
- 타 값 반환 시 변환
```javascript
async function f() {
  return 1;
//return Promise.resolve(1);
}

f().then(alert); // 1
```
- 명시적 반환 가능
```javascript
async function f() {
//return 1;
  return Promise.resolve(1);
}

f().then(alert); // 1
```

### `await`

##### `await` <sub>(키워드)</sub>
- `async` 함수 내에서만 동작
```javascript
let value = await promise;
```

##### `Promise` <sub>(객체)</sub> 처리 수행
1. 처리 대기
2. 결과 반환
3. 실행 재개

##### `Promise` <sub>(객체)</sub> 1초 후 이행
```javascript
async function f() {
  let promise = new Promise((resolve, reject) => {

    // 이행 (1초 지연)
    setTimeout(() => resolve("완료!"), 1000);
  });

  /* Promise (객체) 처리
   ① 실행 중단 (처리 대기)
   ② result (변수) → Promise.result 할당 (이행 완료)
   ③ 실행 재개
  let result = await promise;

  result;
}

f(); // 완료! (1초 후)
```

##### `await` 처리 대기 중
- 엔진 타 작업 수행 <sub>(CPU 자원 낭비 X)</sub>
  - 타 스크립트 실행
  - 이벤트 처리
  - 기타 등등

##### `promise.then` 대비 장점
- 가독성 ↑
- 사용 난이도 ↓

##### `showAvatar()` <sub>(함수 · [`Promise` 체이닝](./03.%20Promises%20chaining.md) 챕터)</sub> 재작성
- `async` · `await` 사용
  - `then`<sub>(메서드)</sub> → `await` <sub>(키워드)</sub>
  - `function` 앞 `async` <sub>(키워드)</sub> 추가
```javascript
async function showAvatar() {

  // 원격 서버 정보 JSON 파싱
  let response = await fetch('/article/promise-chaining/user.json');
  let user = await response.json();

  // github 사용자 정보 JSON 파싱
  let githubResponse = await fetch(`https://api.github.com/users/${user.name}`);
  let githubUser = await githubResponse.json();

  // 아바타 이미지 표시
  let img = document.createElement('img');
  img.src = githubUser.avatar_url;
  img.className = "promise-avatar-example";
  document.body.append(img);

  // 3초 대기
  await new Promise(function(resolve, reject) {
    setTimeout(resolve, 3000);
  }

  img.remove(); // 아바타 이미지 제거

  return githubUser;
}

showAvatar();
```

<img src="../../images/commons/icons/circle-exclamation-solid.svg" /> **`await` : `thenable` <sub>(객체)</sub> 사용 가능**

##### 서드파티 객체
- `Promise` <sub>(객체)</sub> 미제공
  - `Promise` <sub>(객체)</sub> 호환 객체 제공
- `.then` <sub>(메서드)</sub> 지원
  - `await` 함께 사용 가능
```javascript
class Thenable {
  constructor(num) {
    this.num = num;
  }
  then(resolve, reject) {
    alert(resolve);

    // 1초 후 이행
    // - result 값 : this.num * 2
    setTimeout(() => resolve(this.num * 2), 1000);
  }
};

async function f() {
  // 1초 후
  // - result (변수) 값 : 2
  let result = await new Thenable(1);
  alert(result);
}

f();
```

##### `await` : 객체 <sub>(`.then` 구현 · `Promise` 객체 X)</sub> 전달 시
1. `.then` <sub>(메서드)</sub> 호출
    - 인수 제공 <sub>(내장 함수)</sub>
      - `resolve`
      - `reject`
    - `Promise executor` 작업 동일
2. 인수 <sub>(`resolve` · `reject`)</sub> 중 하나 호출 대기
    - 호출 결과 함께 다음 작업 진행

### 에러 처리

#### `Promise` <sub>(`await promise`)</sub>

##### 정상 이행 시
- `result` 반환
  - `Promise` <sub>(객체)</sub> 내 저장 값

##### 거부 시
- 에러 던져짐
  - `throw` <sub>(문)</sub> 유사
```javascript
async function f() {
  await Promise.reject(new Error("에러"));
}

// 동일 코드
async function f() {
  throw new Error("에러");
}
```

##### `Promise` <sub>(객체)</sub> 거부 전
- 약간의 시간 지체 발생 경우 有

##### `await`
- 에러 던지기 전 지연 발생

##### `await` 가 던진 에러
- 에러 잡기 가능 <sub>(`try…catch`)</sub>
```javascript
async function f() {

  try {
    let response = await fetch('http://유효하지-않은-주소');
  } catch(err) {
    alert(err); // TypeError: failed to fetch
  }
}

f();
```

##### 에러 발생 시
- 제어 흐름 → `catch(…){…}` <sub>(블록)</sub>

##### 코드 다수 줄 감싸기 <sub>(`try`)</sub>
```javascript
async function f() {

  try {
    let response = await fetch('http://유효하지-않은-주소');
    let user = await response.json();
  } catch(err) {

    // fetch · response.json 내 발생 에러 처리
    alert(err);
  }
}

f();
```

##### `try…catch` 부재 시
- `.catch` <sub>(핸들러)</sub> 추가
  - 거부된 `Promise` <sub>(객체)</sub> 처리
```javascript
async function f() {
  let response = await fetch('http://유효하지-않은-주소');
}

// f()
// - 거부된 Promise (객체)
f().catch(alert); // TypeError: failed to fetch
```

##### `.catch` <sub>(핸들러)</sub> 미추가 시
- 미처리 `Promise` <sub>(객체)</sub> 에러 발생
- `unhandledrejection` 사용
  - 전역 이벤트 핸들러

### `async` · `await` 사용해 코드 변경하기

  - → `async` · `await`
```javascript
function loadJson(url) {
  return fetch(url)
    .then(response => {
      if (response.status == 200) {
        return response.json();
      } else {
        throw new Error(response.status);
      }
    })
}

loadJson('no-such-user.json')
.catch(alert); // Error: 404
```

async function loadJson(url) {

### `async` · `await` 사용해 '다시 던지기' 예시 재작성하기

##### [`Promise` 체이닝](./03.%20Promises%20chaining.md) <sub>(챕터)</sub> 예시 <sub>(다시 던지기)</sub> 재작성
- `.then` · `.catch`
  - → `async` · `await`
- `demoGithubUser` 내 반복
  - 반복문 사용
  - `async` · `await` 사용
```javascript
class HttpError extends Error {
  constructor(response) {
    super(`${response.status} for ${response.url}`);
    this.name = 'HttpError';
    this.response = response;
  }
}

function loadJson(url) {
  return fetch(url)
    .then(response => {
      if (response.status == 200) {
        return response.json();
      } else {
        throw new HttpError(response);
      }
    })
}

// 유효한 사용자 찾을 때까지 반복해서 username 물어봄
function demoGithubUser() {
  let name = prompt("GitHub username을 입력하세요.", "iliakan");

  return loadJson(`https://api.github.com/users/${name}`)
    .then(user => {
      alert(`이름: ${user.name}.`);
      return user;
    })
    .catch(err => {
      if (err instanceof HttpError && err.response.status == 404) {
        alert("일치하는 사용자가 없습니다. 다시 입력해 주세요.");
        return demoGithubUser();
      } else {
        throw err;
      }
    });
}

demoGithubUser();
```

```javascript
class HttpError extends Error {
  constructor(response) {
    super(`${response.status} for ${response.url}`);
    this.name = 'HttpError';
    this.response = response;
  }
}

async function loadJson(url) {
  let response = await fetch(url);
  if (response.status == 200) {
    return response.json();
  } else {
    throw new HttpError(response);
  }
}

// 유효한 사용자 찾을 때까지 반복해서 username 물어봄
async function demoGithubUser() {

  let user;
  while (true) {
    let name = prompt("GitHub username을 입력하세요.", "iliakan");

    try {
      user = await loadJson(`https://api.github.com/users/${name}`);

      // 에러 X
      // - 반복문 빠져나감
      break;
    } catch(err) {
      if (err instanceof HttpError && err.response.status == 404) {

        // 경고 창 표시 이후
        // - 반복문 계속 실행
        alert("일치하는 사용자가 없습니다. 다시 입력해 주세요.");
      } else {

        // 알 수 없는 에러
        // - 다시 던져짐
        throw err;
      }
    }
  }

  alert(`이름: ${user.name}.`);
  return user;
}

demoGithubUser();
```
