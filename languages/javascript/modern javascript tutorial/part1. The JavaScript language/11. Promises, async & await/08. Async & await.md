`async` · `await`
====

### `async` 함수

##### `async` <sub>(키워드)</sub>
- `function` <sub>(키워드)</sub> 앞 위치
```javascript
async function f() {…}
```

##### 항상 `Promise` <sub>(객체)</sub> 반환
- 타 값 반환 시 변환
```javascript
async function f() {
  return 1;
//return Promise.resolve(1);
}

f().then(alert); // 1
```
- 명시적 반환 가능
```javascript
async function f() {
//return 1;
  return Promise.resolve(1);
}

f().then(alert); // 1
```

### `await`

##### `await` <sub>(키워드)</sub>
- `async` 함수 내에서만 동작
```javascript
let value = await promise;
```

##### `Promise` <sub>(객체)</sub> 처리 수행
1. 처리 대기
2. 결과 반환
   - 이행 시 `result` <sub>(프로퍼티)</sub> 값 반환
   - 거부 시 에러 던지기 <sub>(≒ `throw` 문)</sub>
3. 실행 재개

##### `Promise` <sub>(객체)</sub> 1초 후 이행
```javascript
async function f() {
  let promise = new Promise((resolve, reject) => {

    // 이행 (1초 지연)
    setTimeout(() => resolve("완료!"), 1000);
  });

  /* Promise (객체) 처리
   ① 실행 중단 (처리 대기)
   ② result (변수) → Promise.result 할당 (이행 완료)
   ③ 실행 재개
   */
  let result = await promise;

  result;
}

f(); // 완료! (1초 후)
```

##### `await` 처리 대기 중 엔진 타 작업 수행 <sub>(CPU 자원 낭비 X)</sub>
- 타 스크립트 실행
- 이벤트 처리
- 기타 등등

##### `promise.then` 대비 장점
- 가독성 ↑
- 사용 난이도 ↓

##### `showAvatar()` <sub>(함수 · [`Promise` 체이닝](./03.%20Promises%20chaining.md) 챕터)</sub> 재작성
- `async` · `await` 사용
  - `then`<sub>(메서드)</sub> → `await` <sub>(키워드)</sub>
  - `function` 앞 `async` <sub>(키워드)</sub> 추가
```javascript
async function showAvatar() {

  // 원격 서버 정보 JSON 파싱
  let response = await fetch('/article/promise-chaining/user.json');
  let user = await response.json();

  // github 사용자 정보 JSON 파싱
  let githubResponse = await fetch(`https://api.github.com/users/${user.name}`);
  let githubUser = await githubResponse.json();

  // 아바타 이미지 표시
  let img = document.createElement('img');
  img.src = githubUser.avatar_url;
  img.className = "promise-avatar-example";
  document.body.append(img);

  // 3초 대기
  await new Promise(function(resolve, reject) {
    setTimeout(resolve, 3000);
  }

  img.remove(); // 아바타 이미지 제거

  return githubUser;
}

showAvatar();
```

##### 최상위 레벨 코드
- `await` <sub>(키워드)</sub> 사용 시 문법 에러
```javascript
let response = await fetch(…);
let user = await response.json(); // Syntax error
```
- 익명 `async` 함수 내 사용 가능
```javascript
(async () => {
  let response = await fetch(…);
  let user = await response.json();
})();
```

<img src="../../images/commons/icons/circle-exclamation-solid.svg" /> **`thenable` <sub>(객체)</sub> 대상 `await` 사용 가능**

##### 서드파티 객체
- `then` <sub>(메서드)</sub> 제공 시
  - `await` <sub>(키워드)</sub> 사용 가능
```javascript
class Thenable {
  constructor(num) {
    this.num = num;
  }

  then(resolve, reject) {
    resolve; // function { …내장 코드… }

    // 이행 (1초 지연)
    // - this.num * 2 (처리 결과) 반환
    //   - 다음 핸들러 전달 (result 프로퍼티 할당)
    setTimeout(() => resolve(this.num * 2), 1000);
  }
};

async function f() {

  // 이행 (1초 지연)
  // - 2 (처리 결과) 반환 (할당)
  let result = await new Thenable(1);

  result; // 2
}

f(); // 2 (1초 후)
```

### 에러 처리

##### 거부 시 에러 던지기 <sub>(≒ `throw` 문)</sub>
- 거부 지연 발생 시 에러 지연
```javascript
async function f() {
  await Promise.reject(new Error("에러"));
//throw new Error("에러");
}
```

#### 에러 처리

##### a. `try…catch`
- 에러 시 제어 흐름 이동
  - 에러 지점 → `catch(…){…}` <sub>(블록)</sub>
```javascript
async function f() {
  try {
    let response = await fetch('http://유효하지-않은-주소');
  } catch(e) {
    e // TypeError: failed to fetch
  }
}

f();
```
- 다수 `await` 적용 가능
```javascript
async function f() {

  try {
    let response = await fetch('http://유효하지-않은-주소');
    let user = await response.json();
  } catch(e) {
    …fetch · response.json 내 발생 에러 처리…
    e // Error…
  }
}

f();
```

##### b. `catch` 핸들러
- 거부된 `Promise` <sub>(객체)</sub> 처리
```javascript
async function f() {
  let response = await fetch('http://유효하지-않은-주소');
}

f().catch(alert); // TypeError: failed to fetch
```

##### c. `unhandledrejection` <sub>(이벤트)</sub> 핸들러
- 미처리 <sub>(전역)</sub> 에러 처리

### 코드 변경

##### [`Promise` 체이닝](./03.%20Promises%20chaining.md) <sub>(챕터)</sub> 예시 <sub>(다시 던지기)</sub> 재작성
- `then` · `catch` → `async` · `await`
- `demoGithubUser` <sub>(함수)</sub> 내 반복문 사용
```javascript
class HttpError extends Error {
  constructor(response) {
    super(`${response.status} for ${response.url}`);

    this.name = 'HttpError';
    this.response = response;
  }
}

function loadJson(url) {
  return fetch(url)
    .then(response => {
      if (response.status == 200) {
        return response.json();
      } else {
        throw new HttpError(response);
      }
    });
}

// 유효 사용자 검색 성공 전까지 반복해서 username 질문
function demoGithubUser() {
  let name = prompt("GitHub username을 입력하세요.", "iliakan");

  return loadJson(`https://api.github.com/users/${name}`)
    .then(user => {
      `이름: ${user.name}.`;

      return user;
    })
    .catch(e => {
      if (e instanceof HttpError && e.response.status == 404) {
        "일치하는 사용자가 없습니다. 다시 입력해 주세요.";

        return demoGithubUser();
      } else {
        throw e;
      }
    });
}

demoGithubUser();
```

##### 일반 함수 → 비동기 함수
- `async` <sub>(키워드)</sub> 사용
- `then` <sub>(메서드)</sub> → `await` <sub>(키워드)</sub>
- `catch` → `try…catch`
```javascript
class HttpError extends Error {…}

async function loadJson(url) {
  let response = await fetch(url);

  if (response.status == 200) {
    return response.json();
  } else {
    throw new HttpError(response);
  }
}

// 유효 사용자 검색 성공 전까지 반복해서 username 질문
async function demoGithubUser() {
  let user;

  while (true) {
    let name = prompt("GitHub username을 입력하세요.", "iliakan");

    try {
      user = await loadJson(`https://api.github.com/users/${name}`);

      break; // 검색 성공 시 반복문 종료
    } catch(e) {
      if (e instanceof HttpError && e.response.status == 404) {

        // 경고 창 표시 후 반복문 계속 실행
        "일치하는 사용자가 없습니다. 다시 입력해 주세요.";
      } else {
        throw e; // 처리 불가능 에러 다시 던지기
      }
    }
  }

  `이름: ${user.name}.`;

  return user;
}

demoGithubUser();
```
