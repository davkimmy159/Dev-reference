객체
====

##### 자료형 구분
- 원시형 <sub>(데이터 1개 저장)</sub>
- 객체형 <sub>(다양한 데이터 저장)</sub>
  - 데이터 집합 <sub>(키 구분)</sub>
  - 복잡한 개체
  - 기타 등등

##### 객체형
- `{…}` <sub>(중괄호)</sub>
  - 객체 생성
- 프로퍼티 <sub>(`{…}` 중괄호 내)</sub>
  - 키 <sub>(`string` · `symbol`)</sub>
  - 값 <sub>(모든 자료형)</sub>
```javascript
// 객체 생성자
let user = new Object();

// `{…}` (객체 리터럴 · 주로 사용)
let user = {};
```

### 리터럴 · 프로퍼티

##### `{…}` <sub>(중괄호)</sub> 내 프로퍼티
- 키 · 값 쌍 구성
  - `키: 값` <sub>(`:` 콜론 기준 좌 · 우)</sub>
```javascript
// 객체
let user = {

// 키  :   값
  name: "John",

// 키 : 값
  age: 30
};
```

##### 프로퍼티 키
- 프로퍼티명
- 식별자

##### 프로퍼티 값 읽기
- `.` <sub>(점)</sub> 표기법 이용
```javascript
alert( user.name ); // John
alert( user.age );  // 30
```

##### 프로퍼티 추가
- 모든 자료형 값 가능
```javascript
// boolean 값 추가
user.isAdmin = true;
```

##### 프로퍼티 삭제
- `delete` <sub>(연산자)</sub>
```javascript
delete user.age;
```

##### 프로퍼티명 <sub>(키)</sub> 다수 단어 조합
- 따옴표 사용
```javascript
let user = {
  name: "John",
  age: 30,

  // 다수 단어 따옴표 사용
 "likes birds": true
};
```

##### 마지막 프로퍼티 끝 `,` <sub>(쉼표)</sub> 사용 가능
- 늘어지는 <sub>(trailing)</sub> · 매달리는 <sub>(hanging)</sub> 쉼표
- 모든 프로퍼티 동일 구조 유지
  - 프로퍼티 추가 · 삭제 · 이동 용이
```javascript
let user = {
  name: "John",
  age: 30,
}
```

<br />

<img src="../../images/commons/icons/circle-exclamation-solid.svg" /> **상수 객체 수정 가능**

##### `const` 선언 객체
- 할당된 객체 고정
- 할당된 객체 내용 고정 X
  - 수정 가능
```javascript
const user = {      // const 선언
  name: "John"
};

user.name = "Pete"; // 객체 내용 수정

alert(user.name);   // Pete (수정 내용 반영)

user = {            // 에러 (할당 값 수정 불가능)
  title: "good day"
};
```

### `[…]` <sub>(대괄호)</sub> 표기법

##### `.` <sub>(점)</sub> 표기법
- 다수 단어 조합 키 읽기 불가능
- '유효한 변수 식별자' 키만 읽기 가능
  - 공백 無
  - 숫자 시작 X
  - 특수문자 無 <sub>(`$` · `_` 제외)</sub>
```javascript
/* 문법 에러 */
// 공백 有
user.likes birds = true;

// 숫자 시작
user.likes.1irds = true;

// 특수문자 (`#`) 有 (`$` · `_` 제외)
user.likes.bi#ds = true;
```

#### `[…]` <sub>(대괄호)</sub> 표기법

##### 모든 문자열 키 읽기 가능
- 따옴표 사용 필수
```javascript
let user = {};

user["likes birds"] = true; // 할당
alert(user["likes birds"]); // 접근
delete user["likes birds"]; // 삭제
```

##### 표현식 평가 결과 키 읽기 가능
- 런타임 내 표현식 평가 <sub>(도중 값 변화 가능)</sub>
  - 평가 종료 후 결과 사용
```javascript
let key = "likes birds";

// user["likes birds"] = true;
user[key] = true;
```
- 사용자 입력값 키 읽기
```javascript
let user = {
  name: "John",
  age: 30
};
let key = prompt("사용자의 어떤 정보를 얻고 싶으신가요?", "name");

// 변수 사용해 키 읽기
alert( user[key] ); // John

// `.` (점) 표기법 읽기 불가능
alert( user.key )   // undefined
```

#### 계산된 프로퍼티

##### `{…}` <sub>(객체 리터럴)</sub> 내 `[…]` <sub>(대괄호)</sub> 프로퍼티 키
```javascript
let fruit = prompt("어떤 과일을 구매하시겠습니까?", "apple");
let bag = {

  // 프로퍼티명 동적 설정 (fruit 변수 사용)
  [fruit]: 5,
// apple : 5,
};

// 계산된 프로퍼티 미사용 시
// bag[fruit] = 5;

// fruit (변수) → 'apple' (문자열) 할당 시
alert( bag.apple ); // 5
```
- 복잡한 표현식 사용 가능
```javascript
let fruit = 'apple';
let bag = {
  [fruit + 'Computers']: 5,
// appleComputers      : 5,
};
```
##### `[…]` <sub>(대괄호)</sub> 표기법 장단점
- 프로퍼티명 제약 제거
  - `.` <sub>(점)</sub> 표기법 보다 강력
- 번거로운 작성 방법

##### `.` <sub>(점)</sub> 표기법 vs `[…]` <sub>(대괄호)</sub> 표기법

|표기법|프로퍼티명|
|:---:|---|
|`.` <sub>(점)</sub>|단순 · 확정|
|`[…]` <sub>(대괄호)</sub>|복잡|

### 단축 프로퍼티

##### 프로퍼티 값 단축 구문
- 프로퍼티 이름 · 값 동일 설정
```javascript
function makeUser(name, age) {
  return {
    name,
 // name: name,
    age,
 // age: age,
  };
}

let user = makeUser("John", 30);

alert(user.name); // John
```
- 일반 프로퍼티 함께 사용 가능
```javascript
let user = {
  name,
//name: name
  age: 30,
};
```

### 프로퍼티명 제약사항

##### 특별한 제약 無
- 예약어 사용 가능
```javascript
let obj = {
  // 예약어 프로퍼티 키
  for: 1,
  let: 2,
  return: 3
};

let sum = obj.for + obj.let + obj.return;

alert( sum );  // 6
```

##### `string` · `symbol` 외 자료형
- 문자열 자동 형 변환
```javascript
let obj = {
    0 : "test" // 0 (숫자) → "0" (문자열)
// "0": "test"
};

alert( obj["0"] ); // test

// 0 (숫자) → "0" (문자열)
alert( obj[ 0 ] ); // test
```

##### `__proto__` <sub>(프로퍼티)</sub>
- 특별 대우 프로퍼티명 <sub>(역사적인 이유)</sub>
```javascript
let obj = {};

obj.__proto__ = 5;    // 숫자 할당

// 숫자 할당 무시
alert(obj.__proto__); // [object Object]
```

### `in` <sub>(연산자)</sub>

##### JS 객체 내 부재 프로퍼티
- 접근 시 `undefined` 반환 <sub>(에러 X)</sub>
  - `undefined` 비교 시 확인 가능
```javascript
let user = {};

// true
alert( user.noSuchProperty === undefined );
```

#### `in` <sub>(연산자)</sub>

##### 객체 내 특정 프로퍼티 유무 여부 확인
- 연산자 좌측 프로퍼티명 필수
  - 보통 따옴표 감싼 문자열
  - 표현식 사용 가능
```javascript
"key" in object
```
```javascript
let user = {
  name: "John",
  age: 30
};

alert( "age"    in user ); // true
alert( "blabla" in user ); // false
```
- 따옴표 생략 시 실수 <sub>(타 변수 조사)</sub> 발생 가능
```javascript
let user = { name: "John" };
let key = "age";

// key (변수) 값 ("age") 사용
//     "age" in user
alert(  key  in user ); // false
```

##### 연산자 존재 이유
- `undefined` 일치 비교 <sub>(`=== undefined`)</sub>
  - 실패 가능성 有
```javascript
let obj = {
  // undefined 값 저장
  test: undefined
};

// undefined 일체 비교
// - 프로퍼티 유무 여부 확인 불가능
alert( obj.test );      // undefined

// in (연산자)
// - 프로퍼티 유무 여부 확인
alert( "test" in obj ); // true
```

### `for…in` <sub>(반복문)</sub>

##### 객체 내 모든 키 순회
```javascript
for (key in object) {
  …본문… // 각 프로퍼티 키 (key) 사용
}
```

##### `user` <sub>(객체)</sub> 내 모든 프로퍼티 출력
- 반복 변수 선언 <sub>(`let key`)</sub>
  - 타 변수명 사용 가능 <sub>(`let prop` 등)</sub>
```javascript
let user = {
  name: "John",
  age: 30,
  isAdmin: true
};

for (let key in user) {

  // 프로퍼티 키
  alert( key );       // name, age, isAdmin

  // 프로퍼티 값
  alert( user[key] ); // John, 30 , true
}
```

### 객체 정렬 방식

##### 정수 프로퍼티
- 오름차순 <sub>(자동 정렬)</sub>
```javascript
let codes = {
  "49": "독일",
  "41": "스위스",
  "44": "영국",
  "1": "미국",
};

// 오름차순 (자동 정렬)
for (let code in codes) {
  alert(code); // 1, 41, 44, 49
}
```

<img src="../../images/commons/icons/circle-exclamation-solid.svg" /> **정수 프로퍼티**

##### 변형 없는 정수 형 변환 가능 문자열

|문자열|정수|변형|정수 프로퍼티 여부|
|:---:|:---:|:---:|:---:|
|`"49"`|`49`|X|O|
|`+49`|`49`|O|X|
|`1.2`|`1`|O|X|

```javascript
// Math.trunc (함수)
// - 소수점 아래 버리고 정수부만 반환

//  '49' == 49 (정수 프로퍼티)
alert( String(Math.trunc(Number("49"))) );

// '+49' != 49 (정수 프로퍼티 X)
alert( String(Math.trunc(Number("+49"))) );

// '1.2' != 1  (정수 프로퍼티 X)
alert( String(Math.trunc(Number("1.2"))) );
```

<br />

##### 非정수 프로퍼티
- 객체 내 추가 순서
```javascript
let user = {
  name: "John",
  surname: "Smith"
};

user.age = 25; // 프로퍼티 추가

// 객체 내 추가 순서
for (let prop in user) {
  alert( prop ); // name, surname, age
}
```

##### 정수 프로퍼티 자동 정렬 방지 트릭
- `+` <sub>(단항 덧셈 연산자)</sub> 사용
```javascript
// 非정수 프로퍼티 추가
let codes = {
  "+49": "독일",
  "+41": "스위스",
  "+44": "영국",
  "+1" : "미국",
};

// 객체 내 추가 순서
for (let code in codes) {

  // `+` (단항 덧셈 연산자)
  // - 출력 시 문자열 → 숫자 변환
  alert( +code ); // 49, 41, 44, 1
}
```

### 객체야 안녕?

4. 프로퍼티 수정
   - `name` <sub>(키)</sub>

5. 프로퍼티 삭제
   - `name` <sub>(키)</sub>

### 빈 객체 여부 확인하기

##### `isEmpty(obj)` <sub>(함수)</sub> 작성
- 객체 내 프로퍼티 유무 여부 확인
  - 有 → `true` 반환
  - 無 → `false` 반환
```javascript
let schedule = {};

alert( isEmpty(schedule) ); // true

schedule["8:30"] = "get up";

alert( isEmpty(schedule) ); // false
```

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

##### 객체 내 프로퍼티 대상 반복문 실행
- 프로퍼티 1개 이상 존재 시
  - 즉시 `false` 반환
```javascript
function isEmpty(obj) {

  // 객체 내 프로퍼티 존재 시 반복문 실행
  for (let key in obj) {
    return false;
  }

  return true;
}
```

### 프로퍼티 합계 구하기

##### `salaries` <sub>(객체)</sub>
- 모든 팀원 월급 정보 저장
```javascript
let salaries = {
  John: 100,
  Ann: 160,
  Pete: 130
}
```

##### 코드 작성
1. 모든 팀원 월급 총합 계산
2. 총합 값 <sub>(`390`)</sub> → `sum` <sub>(변수)</sub> 저장

##### 주의
- `salaries` <sub>(객체)</sub> 내 프로퍼티 부재 시
  - `sum` <sub>(변수)</sub> `0` 저장

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

```javascript
let salaries = {
  John: 100,
  Ann: 160,
  Pete: 130
};

let sum = 0;

for (let key in salaries) {
  sum += salaries[key];
}

alert(sum); // 390
```

### 프로퍼티 값 2배 부풀리기

##### `multiplyNumeric(obj)` <sub>(함수)</sub> 작성
- `obj` <sub>(객체)</sub> 내 숫자 프로퍼티
  - 값 2배 ↑
```javascript
// 함수 호출 전
let menu = {
  width: 200,
  height: 300,
  title: "My menu"
};

multiplyNumeric(menu);

// 함수 호출 후
menu = {
  width: 400,
  height: 600,
  title: "My menu"
};
```

##### 참고
- 반환 값 불필요
- 객체 자체 수정

##### `typeof` <sub>(연산자)</sub>
- 프로퍼티 자료형 확인

<br />

<img src="../../images/commons/icons/circle-answer.svg" />

```javascript
function multiplyNumeric(obj) {
  for (let key in obj) {
    if (typeof obj[key] == 'number') {
      obj[key] *= 2;
    }
  }
}
```
