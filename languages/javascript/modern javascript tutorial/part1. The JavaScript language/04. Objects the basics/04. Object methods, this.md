### 메서드 작성

##### 메서드
- 객체 내 프로퍼티 할당 함수
  - 함수형 프로퍼티

##### `user` <sub>(객체)</sub> 내 함수 할당 <sub>(행동 부여)</sub>
- 함수 표현식
```javascript
let user = { … };

// 함수 생성 (함수 표현식)
// - user (객체) 내 프로퍼티 저장
user.sayHi = function() {
  "안녕하세요!";
};

// 할당 함수 (메서드) 호출
user.sayHi(); // 안녕하세요!
```
- 기존 함수
```javascript
let user = { … };

// 함수 선언
function sayHi() {
  "안녕하세요!";
};

// 기존 선언 함수
// - user (객체) 내 프로퍼티 저장
user.sayHi = sayHi;

user.sayHi(); // 안녕하세요!
```

### 메서드 · `this` <sub>(키워드)</sub>

##### 대다수 메서드
- 객체 내 프로퍼티 값 활용

##### 메서드 내부 `this`
- 객체 <sub>(내 저장 정보)</sub> 접근
```javascript
let user = {
  name: "John",
  age: 30,

  sayHi() {
    this.name; // this → 현재 객체 참조
  },
};

// 메서드 실행 중 this
// - user (객체) 참조
user.sayHi(); // John
```

##### 외부 <sub>(객체 할당)</sub> 변수 사용한 객체 참조 가능 <sub>(`this` 미사용)</sub>
- 예상치 못한 에러 가능성 有
  - ex\) 참조 값 복사 후 타 값 할당 시
    - 메서드 내 `null` 참조
```javascript
let user = {
  name: "John",
  age: 30,

  sayHi() {
    user.name; // user (외부 객체 할당 변수) 사용
  },
};

// 객체 참조 값 복사
let admin = user;

// user (변수) → null 할당
user = null;

// sayHi (메서드) 내 엉뚱한 참조 (null.name) 발생
admin.sayHi(); // Error: Cannot read property 'name' of null
```

### 자유로운 `this`

##### JS 내 모든 함수
- `this` 사용 가능
```javascript
function sayHi() {
  this.name; // 문법 에러 X
}
```

##### `this` 값
- 런타임 내 결정
  - 컨텍스트 따라 상이

##### 상이 객체 내 동일 함수 호출 시
- `this` 참조 값 상이
```javascript
let user  = {name: "John"};
let admin = {name: "Admin"};

function sayHi() {
  this.name;
}

// 두 독립 객체 동일 함수 사용
user.f  = sayHi;
admin.f = sayHi;

// 함수 호출 시 this
// - `.` (점) 앞 객체 참조 <sub>(참조 값 상이)</sub>
user.f();  // John  (this == user  객체)
admin.f(); // Admin (this == admin 객체)

// `.` (점) · `[…]` (대괄호) 표기법 동일 동작
user['f']();  // John  (this == user  객체)
admin['f'](); // Admin (this == admin 객체)
```

<img src="../../images/commons/icons/circle-exclamation-solid.svg" /> **객체 없는 호출 시 `this` == `undefined`**

##### 객체 없이 함수 호출 가능
- `this` 참조 값 無 <sub>(`undefined`)</sub>
```javascript
function sayHi() {
  this;
}

sayHi(); // undefined
```

##### 엄격 모드
- `this` → `undefined` 할당
  - `this.name` 접근 시 에러

##### 非엄격 모드
- `this` → 전역 객체 <sub>(`window` 등)</sub> 참조
  - `"use strict"` <sub>(지시자)</sub> 도입 배경

##### 객체 없는 호출 코드
- 보통 작성 실수

### 화살표 함수 <sub>(`this` 無)</sub>

##### 고유 `this` 無 <sub>(별개 `this` 미생성)</sub>
- `this` 참조 시 외부 컨텍스트 내 `this` 이용
  - 일반 외부 함수 내 `this` 값
```javascript
let user = {
  firstName: "보라",

  sayHi() {

    // 화살표 함수 내 this 접근 시
    //  - 일반 외부 함수 (user.sayHi 메서드) 내 this (user) 사용
    let arrow = () => this.firstName;

    arrow();
  }
};

user.sayHi(); // 보라
```

요약
====

##### 메서드
- 객체 내 프로퍼티 할당 함수
- 객체 '행동' 표현
  - `object.doSomthing()`

##### `this` <sub>(키워드)</sub>
- 메서드 내 현재 객체 참조

##### 런타임 내 `this` 값 결정
- 함수 선언 시 `this` 사용 가능
  - 호출 전 값 할당 X <sub>(호출 후 값 할당)</sub>
- 함수 복사 후 객체 간 전달 가능
- 객체 내 프로퍼티 저장 후 <sub>(메서드 형태)</sub> 호출 시
  - `this` → 호출 <sub>(`.` 점 앞)</sub> 객체 참조
```javascript
let object = {
  name: "John"
};

// 객체 내 프로퍼티 함수 할당 (메서드)
object.method = function() {

  // 메서드 호출 시 this
  // - object (호출 객체 · 호출 시 `.` 점 앞 객체) 참조
  this.name;
};

// method (메서드) 내 this → object (객체) 참조 값 할당
object.method(); // John
```

##### 화살표 함수
- 자체 고유 `this` 無
  - 외부 `this` 값 사용

### `{…}` <sub>(객체 리터럴)</sub> 내 `this` 사용하기

##### `makeUser` <sub>(함수)</sub>
- 객체 <sub>(`this` 포함)</sub> 반환

##### 반환 객체 내 `ref` <sub>(프로퍼티)</sub> 접근 시 결과 예측 · 이유 설명
```javascript
function makeUser() {
  return {
    name: "John",
    ref: this
  };
};

let user = makeUser();

user.ref.name; // ?
```

<img src="../../images/commons/icons/circle-answer.svg" />

#### 에러

##### `this` 값
- 정의 시점 無
- 호출 시점 결정

##### `makeUser` <sub>(함수)</sub> 내 `this` 값
- `undefined` <sub>(객체 無)</sub>
  - `ref: undefined`

##### 에러 수정
- `ref` <sub>(프로퍼티)</sub> 함수 할당 <sub>(메서드)</sub>
  - `this` 반환
```javascript
function makeUser() {
  return {
    name: "John",

    // 메서드 내 this
    // - 호출 (`.` 점 앞) 객체 참조
    ref() {
      return this;
    }
  };
};

let user = makeUser();

user.ref().name; // John
```

let calculator = {

calculator.read();

### 체이닝

##### `ladder` <sub>(객체)</sub>
- 사다리 오르내리기 표현
- 메서드
  - `up` <sub>(올라가기)</sub>
  - `down` <sub>(내려가기)</sub>
  - `showStep` <sub>(단 출력)</sub>
```javascript
let ladder = {
  step: 0,

  up() {
    this.step++;
  },

  down() {
    this.step--;
  },

  showStep: function() {
    this.step;
  },
};
```

##### 메서드 연속 호출
```javascript
ladder.up();
ladder.up();
ladder.down();
ladder.showStep(); // 1
```

##### 모든 메서드 수정
- 호출 체이닝 가능
```javascript
// 1
ladder.up().up().down().showStep();
```

##### 메서드 호출 시 객체 자신 반환
```javascript
let ladder = {
  step: 0,

  up() {
    this.step++;
    return this;
  },

  down() {
    this.step--;
    return this;
  },

  showStep() {
    this.step;
    return this;
  },
};

// 1
ladder.up().up().down().up().down().showStep();
```

##### 체이닝 길이 ↑
- 별도 줄 작성 권장 <sub>(가독성 ↑)</sub>
```javascript
ladder
  .up()
  .up()
  .down()
  .up()
  .down()
  .showStep(); // 1
```
