클래스 확인
====

### `instanceof` <sub>(연산자)</sub>

##### 클래스 · 상속 관계 확인
```javascript
obj instanceof Class
```
- `obj` <sub>(객체)</sub> 클래스 일치 여부 확인
  - `Class`
  - `Class` 하위 클래스
- 보통 프로토타입 체인 탐색

##### 클래스 확인
```javascript
class Rabbit {}

let rabbit = new Rabbit();

// rabbit (객체) 클래스 확인
rabbit instanceof Rabbit; // true
```

##### 생성자 확인
```javascript
function Rabbit() {}

let rabbit = new Rabbit();

// rabbit (객체) 생성자 확인
rabbit instanceof Rabbit; // true
```

##### 내장 클래스 확인
```javascript
let array = [1, 2, 3];

// Array → Object (프로토타입) 상속
array instanceof Array;  // true
array instanceof Object; // true
```

##### `[Symbol.hasInstance]` <sub>(정적 메서드)</sub>
- 직접 확인 동작 설정
- 기본 프로토타입 체인 탐색

#### `obj instanceof Class` 알고리즘

클래스에 정적 메서드 `Symbol.hasInstance`가 구현되어 있으면, `obj instanceof Class`문이 실행될 때, `Class[Symbol.hasInstance](obj)`가 호출됩니다. 호출 결과는 `true`나 `false`이어야 합니다. 이런 규칙을 기반으로 `instanceof`의 동작을 커스터마이징 할 수 있습니다.

##### 1. `[Symbol.hasInstance]` 구현
- `Class[Symbol.hasInstance](obj)` 호출
  - `true` · `false` 반환
- `instanceof` 동작 커스터마이징
```javascript
// instanceOf 로직 직접 설정
// - canEat (프로퍼티) 존재 시 animal 이라고 판단
class Animal {
  static [Symbol.hasInstance](obj) {
    if (obj.canEat) return true;
  }
}

let obj = {canEat: true};

// true
// - Animal[Symbol.hasInstance](obj) 호출
alert(obj instanceof Animal);
```

그런데, 대부분의 클래스엔 `Symbol.hasInstance`가 구현되어있지 않습니다. 이럴 땐 일반적인 로직이 사용됩니다. `obj instanceOf Class`는 `Class.prototype`이 `obj` 프로토타입 체인 상의 프로토타입 중 하나와 일치하는지 확인합니다.

비교는 차례 차례 진행됩니다.

##### 2. `[Symbol.hasInstance]` 미구현
- 일반적인 로직 사용
  - 대다수 클래스 미구현
- `Class.prototype` 일치 여부 확인
  - `obj` 프로토타입 체인 상 프로토타입 중 하나인지
  - 차례 차례 비교 진행
```javascript
obj.__proto__ === Class.prototype?
obj.__proto__.__proto__ === Class.prototype?
obj.__proto__.__proto__.__proto__ === Class.prototype?
...
/*
 이 중 하나라도 true
 - true 반환
 체인 끝 도달
 - false 반환
 */
```

##### 상속 클래스
- 2번째 단계에서 일치 여부 확인
```javascript
class Animal {}
class Rabbit extends Animal {}

let rabbit = new Rabbit();

// true
alert(rabbit instanceof Animal);

// rabbit.__proto__ === Rabbit.prototype
// rabbit.__proto__.__proto__ === Animal.prototype (일치)
```

#### `isPrototypeOf` <sub>(메서드)</sub>
```javascript
proto.isPrototypeOf(obj)
```

##### `proto` <sub>(프로토타입)</sub>
- `obj` <sub>(객체)</sub> 프로토타입 체인 상 어딘가 존재 시
  - `true` 반환
```javascript
// 동일
obj instanceof Class;
Class.prototype.isPrototypeOf(obj);
```

##### 검사 시 Class 생성자 제외
- 2가지만 고려
  - 프로토타입 체인
  - `Class.prototype`
- 객체 생성 후 `prototype` <sub>(프로퍼티)</sub> 변경 시
  - 특이한 결과 초래
```javascript
function Rabbit() {}
let rabbit = new Rabbit();

// 프로토타입 변경
Rabbit.prototype = {};

// false
// - 더 이상 Rabbit 아님
alert( rabbit instanceof Rabbit );
```

### 보너스: 타입 확인 위한 `Object.prototype.toString`

##### 일반 객체 → 문자열 변환 시
- `"[object Object]"`
```javascript
let obj = {};

// "[object Object]"
alert(obj);

// "[object Object]"
alert(obj.toString());
```

##### `toString` 숨겨진 기능 사용 시
- `typeof` 확장
- `instanceof` 대안

##### `toString` <sub>(내장 메서드)</sub> 추출 가능
- 명세서 정의
- 모든 값 대상 실행 가능
- 호출 결과
  - 값 따라 상이

|타입|문자열 변환|
|---|---|
|`number`|`"[object Number]"`|
|`boolean`|`"[object Boolean]"`|
|`null`|`"[object Null]"`|
|`undefined`|`"[object Undefined]"`|
|배열|`"[object Array]"`|
|이 외|커스터마이징 가능|

```javascript
// toString (내장 메서드)
// - 편의 위해 변수에 복사
let objectToString = Object.prototype.toString;

let arr = [];

// "[object Array]"
alert( objectToString.call(arr) );
```

##### `toString` 알고리즘
- 내부적 `this` 검사 후
  - 상응 결과 반환
```javascript
let s = Object.prototype.toString;

// "[object Number]"
alert( s.call(123) );

// "[object Null]"
alert( s.call(null) );

// "[object Function]"
alert( s.call(alert) );
```

#### `[Symbol.toStringTag]` <sub>(프로퍼티)</sub>
- `toString` 동작 커스터마이징
```javascript
let user = {
  [Symbol.toStringTag]: "User"
};

// "[object User]"
alert( {}.toString.call(user) );
```

##### 대다수 호스트 환경
- 자체 객체
  - 유사 프로퍼티 구현
```javascript
// toStringTag
// - 특정 호스트 환경의 객체 · 클래스에 구현

// "Window"
alert( window[Symbol.toStringTag]);

// "XMLHttpRequest"
alert( XMLHttpRequest.prototype[Symbol.toStringTag] );

// "[object Window]"
alert( {}.toString.call(window) );

// "[object XMLHttpRequest]"
alert( {}.toString.call(new XMLHttpRequest()) );
```

##### `toString` · `toStringTag`
- `typeof` <sub>(연산자)</sub> 의 강력한 변형들
- 내장 객체
  - 사용 · 커스터마이징 가능
- 타입 정보 문자열 필요 시
  - `instanceof` X
  - `{}.toString.call`

### 이상한 `instanceof` <sub>(연산자)</sub>

##### `instanceof` <sub>(연산자)</sub> : `true` 반환 이유
- `a` <sub>(객체)</sub>
  - `B()` <sub>(생성자)</sub> 생성 X
```javascript
function A() {}
function B() {}

A.prototype = B.prototype = {};

let a = new A();

// 객체 instanceof 생성자
alert( a instanceof B ); // true
```
