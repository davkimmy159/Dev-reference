[믹스인](https://en.wikipedia.org/wiki/Mixin)
====

##### JS 단일 상속만 허용
- 객체 당 `[[Prototype]]` 1개 보유
- 클래스 당 단일 클래스 상속 가능

##### 믹스인 <sub>(mixin)</sub>
- 특정 동작 실행 메서드 제공 개체
  - 단독 사용 X
  - 타 클래스 내 동작 추가

### 예시

##### 단일 객체 내 다수 유용한 메서드 구현 <sub>(가장 간단한 방법)</sub>
- 클래스 프로토타입 병합 용이
```javascript
// 믹스인
let mixin = {
  sayHi() {
    alert(`Hello ${this.name}`);
  },

  sayBye() {
    alert(`Bye ${this.name}`);
  }
};

class Person {}

// 상속 (하위) 클래스
class User extends Person {
  constructor(name) {
    this.name = name;
  }
}

// 하위 클래스 프로토타입 내 믹스인 메서드 복사Object.assign(User.prototype, mixin);

new User("Dude").sayHi(); // Hello Dude!
```

##### 믹스인 상속 활용
```javascript
let supMixin = {
  say(phrase) {
    alert(phrase);
  }
};

// 상속 (하위) 믹스인
let subMixin = {

  // supMixin 상속
  // - Object.create (메서드) 사용 가능
  __proto__: supMixin,

  sayHi() {

    // 프로토타입 (supMixin) 내 부모 메서드 호출
    super.say(`Hello ${this.name}`);
  },

  sayBye() {

    // 프로토타입 (supMixin) 내 부모 메서드 호출
    super.say(`Bye ${this.name}`);
  }
};

class Person {}

// 상속 (하위) 클래스
class User extends Person {
  constructor(name) {
    this.name = name;
  }
}

// 하위 클래스 프로토타입 내 믹스인 메서드 복사
Object.assign(User.prototype, subMixin);

new User("Dude").sayHi(); // Hello Dude!
```

### 이벤트 믹스인

#### 이벤트 메서드

##### `.trigger(name[, ...data])` <sub>(이벤트 생성)</sub>
- 생성 이벤트 처리 리스너 그룹 전체 실행
- `name` <sub>(문자열)</sub>
  - 이벤트명
- `[...data]` <sub>(배열)</sub>
  - 이벤트 데이터 정보

##### `.on(name, handler)` <sub>(리스너 추가)</sub>
- `name` <sub>(문자열)</sub>
  - 이벤트명
- `handler` <sub>(함수)</sub>
  - 이벤트 <sub>(`name`)</sub> 처리 리스너
    - 리스너 배열 내 저장
  - `.trigger` <sub>(메서드)</sub>
    - 리스너 함수 인수 전달

##### `.off(name, handler)` <sub>(리스너 제거)</sub>
- `name` <sub>(문자열)</sub>
  - 이벤트명
- `handler` <sub>(함수)</sub>
  - 이벤트 <sub>(`name`)</sub> 처리 리스너
```javascript
let eventMixin = {

  // 이벤트 구독
  // - menu.on('select', function(item) { … }
  on(eventName, handler) {
    if (!this._eventHandlers) {
      this._eventHandlers = {};
    }

    if (!this._eventHandlers[eventName]) {

      // 이벤트 리스너 그룹 배열
      this._eventHandlers[eventName] = [];
    }

    this._eventHandlers[eventName].push(handler);
  },

  // 이벤트 구독 취소
  // - menu.off('select', handler)
  off(eventName, handler) {
    let handlers = this._eventHandlers?.[eventName];

    // 이벤트 처리 리스너 無
    if (!handlers) {
      return;
    }

    // 이벤트 리스너 그룹 내 제거
    for (let i = 0; i < handlers.length; i++) {
      if (handlers[i] === handler) {
        handlers.splice(i--, 1);
      }
    }
  },

  // 이벤트 생성
  // - this.trigger('select', data1, data2);
  trigger(eventName, ...args) {

    // 해당 이벤트 처리 리스너 無
    if (!this._eventHandlers?.[eventName]) {
      return;
    }

    // 해당 이벤트 리스너 그룹 전체 호출
    this._eventHandlers[eventName].forEach(handler => handler.apply(this, args));
  }
};
```

#### 믹스인 적용

##### 사용자 로그인
- `user` <sub>(객체)</sub>
  - `login` <sub>(이벤트)</sub> 생성
- `calendar` <sub>(객체)</sub>
  - `login` <sub>(이벤트)</sub> 발생 시
    - 적절한 달력 표시

##### 메뉴 항목 선택
- `menu` <sub>(객체)</sub>
  - `select` <sub>(이벤트)</sub> 생성
- 타 객체
  - `select` <sub>(이벤트)</sub> 처리 리스너 할당

##### 사용법
```javascript
class Menu {
  choose(value) {
    this.trigger("select", value);
  }
}
// 클래스 프로토타입 내 믹스인 메서드 복사
// - 다수 이벤트 관련 메서드
Object.assign(Menu.prototype, eventMixin);

let menu = new Menu();

// 메뉴 항목 선택 시 이벤트 리스너 추가
menu.on("select", value => alert(`선택된 값: ${value}`));

// select (이벤트) 발생
// - 이벤트 해당 리스너 실행
//   - 선택 값 (123) 표시
menu.choose("123");
```

요약
====

#### 믹스인

##### 객체 지향 언어 내 범용 용어
- 타 클래스 내 메서드 조합 구현

##### JS 다중 상속 미지원
- 믹스인 내 메서드 복사
  - 프로토타입 내 메서드 구현 가능

##### 클래스 확장 가능
- 특정 행동 추가
  - ex\) 이벤트 처리 등

mixin이 실수로 기존 클래스 메서드를 덮어쓰면 충돌이 발생할 수 있습니다. 따라서 mixin을 만들 땐 충돌이 발생하지 않도록 메서드 이름을 신중하게 정하셔야 합니다.

##### 기존 클래스 내 메서드 덮어쓰기 시 충돌 발생 가능
- 충돌 발생
  - 메서드명 신중히 작성
