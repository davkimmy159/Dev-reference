[커링](https://en.wikipedia.org/wiki/Currying)
====

##### 함수 변환 <sub>(호출 X)</sub>
- `f(a, b, c)` → `f(a)(b)(c)`

##### `curry(f)` <sub>(헬퍼 함수)</sub>
- `f(a, b)` → `f(a)(b)`
  - f <sub>(함수)</sub> 인수 <sub>(2개)</sub> 커링
```javascript
function curry(f) {
  return function(a) {
    return function(b) {
      return f(a, b);
    };
  };
}

function sum(a, b) {
  return a + b;
}

let cSum = curry(sum);

cSum(1)(2); // 3
```

##### 과정
1. `curry(f)`
   - `function(a)` <sub>(함수)</sub> 반환
     - 렉시컬 환경 내 `a` <sub>(인수)</sub> 저장
2. `function(a)` <sub>(함수)</sub>
   - `function(b)` <sub>(함수)</sub> 반환
     - 렉시컬 환경 내 `b` <sub>(인수)</sub> 저장
     - `f` <sub>(기존 함수)</sub> 호출 결과 반환

##### `_.curry` <sub>(`lodash` 라이브러리 내 함수)</sub>
- 적용 후 자유로운 사용 가능
  - 일반 · 커링 호출
```javascript
function sum(a, b) {
  return a + b;
}

// lodash (라이브러리) 사용
let cSum = _.curry(sum);

cSum(1, 2); // 3 (일반 호출)
cSum(1)(2); // 3 (커링 호출)
```

##### 다양한 부분 함수 생성
- 기존 기능 수행
- 특화 기능 수행 <sub>(기존 기능 기반)</sub>

##### `log(date, importance, message)` <sub>(함수)</sub>
- 정보 형식화 후 출력
```javascript
function log(date, importance, message) {
  `[${date.getHours()}:${date.getMinutes()}] [${importance}] ${message}`;
}
```

##### 커링 적용 <sub>(`lodash` 라이브러리 내 함수)</sub>
```javascript
log = _.curry(log);
```
- 일반 호출 정상 동작
```javascript
// log(a, b, c)
log(new Date(), "DEBUG", "some debug");
```
- 커링 호출 정상 동작
```javascript
// log(a)(b)(c)
log(new Date())("DEBUG")("some debug");
```
- `logNow` <sub>(부분 함수)</sub> 생성
  - 기존 함수 1번째 인수 고정
  - 현재 시간 로그 출력 용도
```javascript
// 1번째 인수 고정
let logNow = log(new Date());

// log(new Date(), "INFO", "message")
logNow("INFO", "message"); // [HH:mm] INFO message
```
- `debugNow` <sub>(부분 함수)</sub> 생성
  - 기존 함수 1 · 2번째 인수 고정
  - 디버깅 로그 용도
```javascript
// log (함수) 1 · 2번째 인수 고정
// - logNow (함수) 1번째 인수 고정
let debugNow = logNow("DEBUG");

//    log(new Date(), "DEBUG", "message")
// logNow(            "DEBUG", "message")
debugNow("message"); // [HH:mm] DEBUG 메세지
```

### 고급 구현

##### 다중 인수 허용 커링
```javascript
function curry(f) {
  return function curried(...args) {
    if (args.length >= f.length) {
      return f.apply(this, args);
    } else {
      return function(...args2) {
        return curried.apply(this, args.concat(args2));
      }
    }
  };
}

function sum(a, b, c) {
  return a + b + c;
}

let cSum = curry(sum);

cSum(1, 2, 3); // 6
cSum(1)(2,3);  // 6
cSum(1)(2)(3); // 6
```

##### `curried` <sub>(래퍼 함수)</sub>
- `curry(f)` 반환 값
```javascript
function curried(...args) {

  /* if 분기 (2가지) */

  // ⓐ 기존 함수 호출
  // 래퍼 함수 인수 개수 ≥ 기존 함수 매개변수 개수
  if (args.length >= f.length) {
    return f.apply(this, args);

  /* ⓑ 부분 적용
   pass (래퍼 함수) 반환
   - curried (래퍼 함수) 재적용
     1. 이전 인수 + 새 인수
     2. 새 if 분기 진행 (인수 개수 비교)
   */
  } else {
    return function pass(...args2) {
      return curried.apply(this, args.concat(args2));
    }
  }
};
```

#### `curried(1)(2)(3)` 호출 과정

##### `sum(a, b, c)` <sub>(기존 함수)</sub>
- `sum.length` == `3`

##### 1. `curried(1)`
- `1` <sub>(1번째 인수)</sub>
  - 렉시컬 환경 내 기억
- `pass` <sub>(래퍼 함수)</sub> 반환

##### 2. `pass(2)` <sub>(래퍼)</sub>
1. `1` <sub>(이전 인수)</sub> · `2` <sub>(현재 인수)</sub> 연결
2. `curried(1, 2)` 호출
3. 인수 개수 <sub>(`2`)</sub> \< `sum.length` <sub>(`3`)</sub>
   - `pass` <sub>(래퍼 함수)</sub> 반환

##### 3. `pass(3)` <sub>(래퍼)</sub>
1. `1` · `2` <sub>(이전 인수)</sub> `3` <sub>(현재 인수)</sub> 연결
2. `curried(1, 2, 3)` 호출
3. 인수 개수 <sub>(`3`)</sub> == `sum.length` <sub>(`3`)</sub>
   - 기존 함수 <sub>(`f`)</sub> 호출 결과 반환

#### 커링

- 평소처럼 호출 가능
- 인수 불충분 시
  - 부분 함수 <sub>('partial')</sub> 반환

##### 부분 함수 <sub>('partial')</sub> 적용 용이 경우
- 로그 예시
```javascript
// 범용 함수 (인수 3개)
log(date, importance, message)

// 커링 후 인수 1개 형태
log(date)

// 커링 후 인수 2개 형태
log(date, importance)
```
