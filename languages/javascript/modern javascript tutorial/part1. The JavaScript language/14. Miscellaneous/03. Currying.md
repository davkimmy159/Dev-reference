[커링](https://en.wikipedia.org/wiki/Currying)
====

##### 함수 변환 <sub>(호출 X)</sub>
- `f(a, b, c)` → `f(a)(b)(c)`

##### `curry(f)` <sub>(헬퍼 함수)</sub>
- `f(a, b)` → `f(a)(b)`
  - f <sub>(함수)</sub> 인수 <sub>(2개)</sub> 커링
```javascript
function curry(f) {
  return function(a) {
    return function(b) {
      return f(a, b);
    };
  };
}

function sum(a, b) {
  return a + b;
}

let curriedSum = curry(sum);

curriedSum(1)(2); // 3
```

##### 과정
1. `curry(f)`
   - `function(a)` <sub>(함수)</sub> 반환
     - 렉시컬 환경 내 `a` <sub>(인수)</sub> 저장
2. `function(a)` <sub>(함수)</sub>
   - `function(b)` <sub>(함수)</sub> 반환
     - 렉시컬 환경 내 `b` <sub>(인수)</sub> 저장
     - `f` <sub>(기존 함수)</sub> 호출 결과 반환

##### `_.curry` <sub>(`lodash` 라이브러리 내 함수)</sub>
- 적용 후 자유로운 사용 가능
  - 일반 · 커링 호출
```javascript
function sum(a, b) {
  return a + b;
}

// lodash (라이브러리) 사용
let carriedSum = _.curry(sum);

carriedSum(1, 2); // 3 (일반 호출)
carriedSum(1)(2); // 3 (커링 호출)
```

### 용도

##### 다양한 부분 함수 생성
- 기존 기능 수행
- 특화 기능 수행 <sub>(기존 기능 기반)</sub>

##### `log(date, importance, message)` <sub>(함수)</sub>
- 정보 형식화 후 출력
```javascript
function log(date, importance, message) {
  `[${date.getHours()}:${date.getMinutes()}] [${importance}] ${message}`;
}
```

##### 커링 적용 <sub>(`lodash` 라이브러리 내 함수)</sub>
```javascript
log = _.curry(log);
```
- 일반 호출 정상 동작
```javascript
// log(a, b, c)
log(new Date(), "DEBUG", "some debug");
```
- 커링 호출 정상 동작
```javascript
// log(a)(b)(c)
log(new Date())("DEBUG")("some debug");
```
- `logNow` <sub>(부분 함수)</sub> 생성
  - 기존 함수 1번째 인수 고정
  - 현재 시간 로그 출력 용도
```javascript
// 1번째 인수 고정
let logNow = log(new Date());

// log(new Date(), "INFO", "message")
logNow("INFO", "message"); // [HH:mm] INFO message
```
- `debugNow` <sub>(부분 함수)</sub> 생성
  - 기존 함수 1 · 2번째 인수 고정
  - 디버깅 로그 용도
```javascript
// log (함수) 1 · 2번째 인수 고정
// - logNow (함수) 1번째 인수 고정
let debugNow = logNow("DEBUG");

//    log(new Date(), "DEBUG", "message")
// logNow(            "DEBUG", "message")
debugNow("message"); // [HH:mm] DEBUG 메세지
```

### 고급 구현

##### 다중 인수 허용
```javascript
function curry(func) {

  return function curried(...args) {
    if (args.length >= func.length) {
      return func.apply(this, args);
    } else {
      return function(...args2) {
        return curried.apply(this, args.concat(args2));
      }
    }
  };

}
```

##### 사용법
```javascript
function sum(a, b, c) {
  return a + b + c;
}

let curriedSum = curry(sum);

// 보통때 처럼 호출 (단일 callable 형식)
alert( curriedSum(1, 2, 3) ); // 6

// 1번째 인수 커링
alert( curriedSum(1)(2,3) );  // 6

// 모두 커링
alert( curriedSum(1)(2)(3) ); // 6
```

##### `curry(func)` 반환 값
- `curried` <sub>(래퍼 함수)</sub>
```javascript
// func (함수)
// - 원본 함수
function curried(...args) {

  // ①
  if (args.length >= func.length) {
    return func.apply(this, args);

  // ②
  } else {
    return function pass(...args2) {
      return curried.apply(this, args.concat(args2));
    }
  }
};
```

#### `if` 분기점 <sub>(2개)</sub>

##### ⓐ 함수 호출
- `args` 개수 ≥ `func.length` <sub>(기존 함수 개수)</sub>
  - 그대로 `func` <sub>(기존 함수)</sub> 호출에 전달

##### ⓑ 부분적 적용
- 아직 `func` <sub>(기존 함수)</sub> 호출 X
- `pass` <sub>(래퍼)</sub> 대신 반환
  - `curried` 다시 적용
    - 이전 함수 + 새 인수
- 새 `curried` 호출
  - 다시 새 부분적 함수 반환 <sub>(인수 불충분 시)</sub>
  - 최종 `func` <sub>(기존 함수)</sub> 결과 반환

#### `curried(1)(2)(3)` 호출 과정
- `sum(a, b, c)` <sub>(기존 함수)</sub>
  - `sum.length == 3`

##### 1. 1번째 `curried(1)`  호출 시
- `1` <sub>(1번째 인수)</sub>
  - 렉시컬 환경에 기억
- `curried(1)`
  - `pass` <sub>(래퍼)</sub> 반환

##### 2. `pass` <sub>(래퍼)</sub>
- `2` <sub>(2번째 인수)</sub> 함께 호출
- `1` <sub>(이전 인수)</sub> · `2` 연결
  - `curried (1, 2)` 함께 호출
- 인수 개수 \< 3
  - `pass` <sub>(래퍼)</sub> 반환

##### 3. `pass` <sub>(래퍼)</sub>
- `3` <sub>(3번째 인수)</sub> 함께 호출
- `pass(3)` <sub>(래퍼 호출)</sub>
  - `(1, 2)` <sub>(이전 인수들)</sub> · `3` 추가
  - `curried(1, 2, 3)` 호출
  - `3` <sub>(마지막 인수)</sub>
    - 기존 함수 <sub>(`func`)</sub> 에 전달
