유니코드, 문자열 내부
=========================

<img class="icon" src="../../images/commons/icons/triangle-exclamation-solid.svg" /> **심화**

- 문자열 내부 학습
- 유용한 상황
   - 이모지
   - 수학 기호
   - 상형 문자
   - 기타 등등

#### JS 문자열

##### 유니코드 기반
- 각 글자 : 일련의 byte 표현
  - 1 - 4 byte

##### 16진수 유니코드 표현
- 문자열 내 문자 삽입 가능
  - `\xXX`
  - `\uXXXX XXXX`
  - `\u{X…XXXXXX}`

#### `\xXX`

##### 구성
- 16진수 숫자 2개
  - `00` ~ `FF`

##### 첫 256개 유니코드 문자 표현
- 라틴 알파벳
- 기본 문법 글자
- 기타 등등
- ex&#41; `"\x7A"` == `"z"` <sub>(유니코드 `U+007A`)</sub>

```JavaScript
alert( "\x7A" ); // z
alert( "\xA9" ); // ©, 저작권 문자
```

#### `\uXXXX XXXX`

##### 구성
- 16진수 숫자 4개
  - `0000` ~ `FFFF`

##### `U+FFFF` 초과 유니코드 문자 표현 시
- 서로게이트 쌍 필요

```JavaScript
alert( "\u00A9" ); // ©, 저작권 문자
alert( "\u044F" ); // я, 키릴 문자
alert( "\u2191" ); // ↑, 위화살표 문자
```

#### `\u{X…XXXXXX}`

##### 구성
- 16진수 숫자 <sub>(1 ~ 6 bytes)</sub>
  - `0` ~ `10FFFF`

##### 모든 유니코드 문자 표현

```JavaScript
alert( "\u{20331}" ); // 佫, 드문 한자 문자 (긴 유니코드)
alert( "\u{1F60D}" ); // 😍, 웃는 얼굴 문자 (긴 유니코드)
```
