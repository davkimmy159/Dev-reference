##### 복잡한 상황에서 메서드 호출 시
- `this` 값 잃어버리는 경우 발생
```javascript
let user = {
  name: "John",
  hi() { alert(this.name); },
  bye() { alert("Bye"); }
};

user.hi(); // John (간단한 호출)

// name 따라 user.hi · user.bye 호출
// - TypeError: Cannot read property 'name' of undefined
// - this == undefined
(user.name == "John" ? user.hi : user.bye)();
```
