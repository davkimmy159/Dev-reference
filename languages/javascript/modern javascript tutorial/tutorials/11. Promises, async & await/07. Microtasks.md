마이크로태스크
=============

##### 프라미스 핸들러 <sub>(`.then` / `.catch` / `.finally`)</sub>
- 항상 비동기적 실행

##### 프라미스 즉시 이행 시
- 프라미스 핸들러 아래 코드
  - 핸들러들 실행되기 전 실행
```javascript
let promise = Promise.resolve();

promise.then(() => alert("프라미스 성공!"));

alert("코드 종료");

// 메시지 출력 순서
// 1. "코드 종료"
// 2. "프라미스 성공!"
```
