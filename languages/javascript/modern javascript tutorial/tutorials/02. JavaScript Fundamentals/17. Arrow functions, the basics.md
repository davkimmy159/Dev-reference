화살표 함수
==========

- 함수 표현식보다 단순 · 간결
```javascript
let func = function(arg1, arg2, ...argN) {
  return expression;
};

↓↓↓

let func = (arg1, arg2, ...argN) => expression;
```
```javascript
let sum = function(a, b) {
  return a + b;
};

↓↓↓

let sum = (a, b) => a + b;

alert( sum(1, 2) ); // 3
```
- 인수 1개
  - 인수 괄호 생략 가능
```javascript
let double = function(n) { return n * 2 }

↓↓↓

let double = n => n * 2;

alert( double(3) ); // 6
```
- 인수 X
  - 빈 괄호 <sub>(괄호 생략 X)</sub>
```javascript
let sayHi = () => alert("안녕하세요!");
sayHi();
```
- 함수 동적 생성
```javascript
let age = prompt("나이를 알려주세요.", 18);
let welcome = (age < 18) ?
  () => alert('안녕') :
  () => alert("안녕하세요!");

welcome();
```
- 함수 본문 한 줄
  - 코드량 ↓

### 본문 여러 줄 화살표 함수
- 명시적으로 결과값 반환 필수
  - `{ … }` <sub>(중괄호)</sub> · `return`
```javascript
let sum = (a, b) => { // {} (중괄호)
  let result = a + b;
  return result;      // return (결과값 반환)
};
alert( sum(1, 2) );   // 3
```
