엄격모드
============

- ES5 변경사항 적용 <sub>(모던 JS)</sub>

### `"use strict"`
- 단순 문자열 모양
- 스크립트 최상단 위치
  - 스크립트 전체 '모던한' 방식 <sub>(엄격 모드)</sub> 동작
- 함수 본문 맨 앞 위치
  - 해당 함수만 엄격 모드

- 다른 위치
  - 엄격 모드 활성화 X
- `use strict` 위
  - 주석만 사용 가능
