구조 분해 할당
=============

- 객체 · 배열 → 변수 분해
- 사용처
  - 함수 매개변수 多
  - 매개변수 기본값 필요
  - 기타 등등

### 배열 분해하기
```javascript
let arr = ["Bora", "Lee"]

// 구조 분해 할당
// firstName ← arr[0]
// surname   ← arr[1]
let [firstName, surname] = arr;

alert(firstName); // Bora
alert(surname);   // Lee

// 배열 메서드 활용
let [firstName, surname] = "Bora Lee".split(' ');
```

<img class="icon" src="../../images/commons/icons/circle-exclamation-solid.svg" /> **분해 <sub>(destructuring)</sub> ≠ 파괴 <sub>(destructive)</sub>**

- 분해 대상 수정 · 파괴 X
- 직접 변수 할당보다 코드 양 ↓
```javascript
// 구조 분해 할당
let [firstName1, surname1] = arr;

// 직접 할당
let firstName2 = arr[0];
let surname2 = arr[1];
```

<img class="icon" src="../../images/commons/icons/circle-exclamation-solid.svg" /> **쉼표 사용하여 요소 무시**

- 쉼표 사용하여 불필요 배열 요소 버림
```javascript
// 2번째 요소 불필요
let [firstName, , title] = ["Julius", "Caesar", "Consul", "of the Roman Republic"];

alert( title ); // Consul, 3번째 요소
```

<img class="icon" src="../../images/commons/icons/circle-exclamation-solid.svg" /> **할당 연산자 우측 : 모든 이터러블 <sub>(반복 가능)</sub> 객체 가능**

```javascript
let [a, b, c] = "abc"; // ["a", "b", "c"]
let [one, two, three] = new Set([1, 2, 3]);
```

<img class="icon" src="../../images/commons/icons/circle-exclamation-solid.svg" /> **할당 연산자 좌측 : 뭐든지 가능**

- '할당할 수 있는' 것 전부 가능
- 객체 프로퍼티 가능
```javascript
let user = {};
[user.name, user.surname] = "Bora Lee".split(' ');

alert(user.name); // Bora
```

<img class="icon" src="../../images/commons/icons/circle-exclamation-solid.svg" /> **`Object.entries(obj)` 반복**

- 구조 분해 조합 시
  - 객체 키 · 값 순회해 변수로 분해 할당
```javascript
// 객체
let user1 = {
  name: "John",
  age: 30
};

// 객체 키 · 값 순회
for (let [key, value] of Object.entries(user1)) {
  alert(`${key}:${value}`); // name: John, age: 30
}

// 맵
let user2 = new Map();
user2.set("name", "John");
user2.set("age", "30");

// 맵 키 · 값 순회
for (let [key, value] of user2) {
  alert(`${key}:${value}`); // name: John, age: 30
}
```

<img class="icon" src="../../images/commons/icons/circle-exclamation-solid.svg" /> **변수 교환 트릭**

- 두 변수 값 교환
```javascript
let guest = "Jane";
let admin = "Pete";

[guest, admin] = [admin, guest];

alert(`${guest} ${admin}`); // Pete Jane
```

#### `...` : 나머지 요소 가져오기
- 나머지 요소 → 배열 변수
```javascript
let [name1, name2, ...rest] = ["Julius", "Caesar", "Consul", "of the Roman Republic"];

alert(name1);   // Julius
alert(name2);   // Caesar

// rest : 배열
alert(rest[0]); // Consul
alert(rest[1]); // of the Roman Republic
alert(rest.length); // 2
```
- 가장 마지막 위치 필수

#### 기본값
- 할당 변수 개수 > 분해 대상 배열 길이
  - 에러 발생 X
  - 할당 값 부재 시 `undefined` 취급
```javascript
let [firstName, surname] = [];

alert(firstName); // undefined
alert(surname);   // undefined
```
- 할당 값 부재 시 기본값 설정
```javascript
// 기본값
let [name = "Guest", surname = "Anonymous"] = ["Julius"];

alert(name);    // Julius   , 배열 값
alert(surname); // Anonymous, 기본값
```
- 표현식 · 함수 호출 할당 가능
```javascript
// 값 미제공 때만 함수 호출
// name 의 prompt 한 번만 호출
let [surname = prompt('성을 입력하세요.'), name = prompt('이름을 입력하세요.')] = ["김"];

alert(surname); // 김 (배열 값)
alert(name);    // prompt 값
```

### 객체 분해하기
- 우측
  - 분해 대상 객체
- 좌측
  - 상응 객체 프로퍼티 패턴
```javascript
let {var1, var2} = {var1:…, var2:…}
```
```javascript
/* 객체 프로퍼티 키 목록 패턴 */
let options = {
  title: "Menu",
  width: 100,
  height: 200
};

let {title, width, height} = options;

alert(title);  // Menu
alert(width);  // 100
alert(height); // 200
```
- 순서 미중요
```javascript
// { … } (중괄호) 안 순서 바뀌어도 동일 동작
let {height, width, title} = { title: "Menu", height: 200, width: 100 }
```
- 더 복잡한 패턴 가능
  - 객체 프로퍼티 · 변수 연결 조정
  - `:` <sub>(콜론)</sub>
    - `prop: var`
```javascript
// 객체 프로퍼티 · 다른 이름 변수 연결
// - ':' (콜론)
let options = {
  title: "Menu",
  width: 100,
  height: 200
};

// { 객체 프로퍼티: 목표 변수 }
let {width: w, height: h, title} = options;

/*
 width  → w
 height → h
 title  → title
 */
alert(title);  // Menu
alert(w);      // 100
alert(h);      // 200
```
- 프로퍼티 부재 시 기본값 설정
```javascript
let options = {
  title: "Menu"
};

let {width = 100, height = 200, title} = options;

alert(title);  // Menu
alert(width);  // 100
alert(height); // 200
```
- 표현식 · 함수 호출 할당 가능
```javascript
let options = {
  title: "Menu"
};

let {width = prompt("width?"), title = prompt("title?")} = options;

alert(title);  // Menu
alert(width);  // prompt 창 입력 값
```
- `:` <sub>(콜론)</sub> · `=` <sub>(할당 연산자)</sub> 동시 사용 가능
  - 변수 연결 · 기본값
```javascript
let options = {
  title: "Menu"
};

let {width: w = 100, height: h = 200, title} = options;

alert(title);  // Menu
alert(w);      // 100
alert(h);      // 200
```
- 복잡한 객체 특정 정보 뽑아오기
```javascript
let options = {
  title: "Menu",
  width: 100,
  height: 200
};

// title 만 변수로 뽑아내기
let { title } = options;

alert(title); // Menu
```

#### `...` 나머지 패턴
- 나머지 프로퍼티 할당
- 모던 브라우저 지원
- 몇몇 구식 브라우저 <sub>(IE 등)</sub> 미지원
  - 바벨 <sub>(Babel)</sub> 이용
```javascript
let options = {
  title: "Menu",
  height: 200,
  width: 100
};

// rest : 나머지 프로퍼티들
let {title, ...rest} = options;

// rest → { height: 200, width: 100 } 할당
alert(rest.height); // 200
alert(rest.width);  // 100
```

<img class="icon" src="../../images/commons/icons/circle-exclamation-solid.svg" /> **`let` 없이 사용**

- 기존 변수 사용
```javascript
// 잘못된 코드
let title, width, height;

// SyntaxError: Unexpected token '='
// 코드 블록으로 인식
{title, width, height} = {title: "Menu", width: 200, height: 100};
```
- `{ … }` <sub>(중괄호)</sub>
  - 표현식 안 X · 주요 코드 흐름 상 존재 시
    - 코드 블록 인식
    - 코드 블록 본래 용도
      - 문 <sub>(statement)</sub> 묶기
  ```javascript
  {
    // 코드 블록
    let message = "Hello";
    // …
    alert( message );
  }
  ```
- 할당문 `( … )` <sub>(소괄호)</sub> 감싸기
  - `{ … }` <sub>(중괄호)</sub>
    - 코드 블록 X
    - 표현식 해석
```javascript
let title, width, height;

({title, width, height} = {title: "Menu", width: 200, height: 100});

alert( title ); // Menu
```

### 중첩 구조 분해
- 중첩 객체 · 배열 존재 시
  - 더 복잡한 패턴 사용
```javascript
let options = {
  size: { // 객체
    width: 100,
    height: 200
  },
  items: ["Cake", "Donut"], // 배열
  extra: true
};

/*
 대입 연산자 좌측 패턴
 - 객체 options 동일 구조
 코드 여러 줄 걸쳐 작성해 의도하는 바 명확히 드러냄
 */
let {
  size: { // size 객체 프로퍼티
    width,
    height
  },
  items: [item1, item2], // items 배열 프로퍼티
  title = "Menu" // title 프로퍼티 미존재 시 기본값 사용
} = options;

alert(title);   // Menu
alert(width);   // 100
alert(height);  // 200
alert(item1);   // Cake
alert(item2);   // Donut
// alert(extra) // extra 미할당

// 중첩 객체 · 배열 전용 변수 X (할당 X)
// - 중첩 객체 · 배열 내 정보 할당
```

### 똑똑한 함수 매개변수
- 종종 함수 매개변수 선택적으로 사용
```javascript
function showMenu(title = "Untitled", width = 200, height = 100, items = []) {
  // …
}
```
- 문제 발생
  - 인수 전달 순서 틀림
  - 다수 매개변수 기본값 설정 후 전달 불필요
```javascript
// 기본값
// - undefined 여러 개 전달
// - 가독성 ↓
showMenu("My Menu", undefined, undefined, ["Item1", "Item2"])
```
- 객체로 모아 전달
- 함수 객체 분해하여 변수 할당
```javascript
// 함수에 전달할 객체
let options = {
  title: "My menu",
  items: ["Item1", "Item2"]
};

// 전달받은 객체 분해 후 변수 즉시 할당
function showMenu({title = "Untitled", width = 200, height = 100, items = []}) {
  // title, items  → 객체 options
  // width, height → 기본값
  alert( `${title} ${width} ${height}` ); // My Menu 200 100
  alert( items ); // Item1, Item2
}

showMenu(options);
```
- 중첩 객체 · `:` <sub>(콜론)</sub> 조합
  - 더 복잡한 구조 분해
```javascript
let options = {
  title: "My menu",
  items: ["Item1", "Item2"]
};

function showMenu({
  title = "Untitled",
  width: w = 100,                // width  → w
  height: h = 200,               // height → h
  items: [item1, item2]          // items 1번째 요소 → item1
}) {                             // items 2번째 요소 → item2
  alert( `${title} ${w} ${h}` ); // My Menu 100 200
  alert( item1 ); // Item1
  alert( item2 ); // Item2
}

showMenu(options);
```
- 똑똑한 함수 매개변수 문법
  - 구조 분해 할당 문법 동일
```javascript
// incomingProperty → varName
// - 값 부재 시 defaultValue 기본값 사용
function({
  incomingProperty: varName = defaultValue
  …
})
```
- 매개변수 구조 분해 시
  - 모든 인수 기본값 할당
    - 빈 객체 명시적으로 전달
```javascript
showMenu({}); // 모든 인수 기본값 할당
showMenu();   // 에러 발생 가능
```
- 인수 전체 기본값
  - `{}` <sub>(빈 객체 )</sub> 설정
- 어떤 경우든 분해할 것 존재
  - 인수 전부 미전달 시 에러 발생 X
```javascript
function showMenu({ title = "Menu", width = 100, height = 200 } = {}) {
  alert( `${title} ${width} ${height}` );
}

showMenu(); // Menu 100 200
```
