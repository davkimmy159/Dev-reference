async 이터레이터와 제너레이터
===========================

### `async` 이터레이터
- 일반 이터레이터 유사
- 약간의 문법적인 차이

##### 일반 이터러블 == 객체
```javascript
/* 일반 이터러블 (객체) */
let range = {
  from: 1,
  to: 5,

  [Symbol.iterator]() {
    return {
      current: this.from,
      last: this.to,

      next() {
        if (this.current <= this.last) {
          return { done: false, value: this.current++ };
        } else {
          return { done: true };
        }
      }
    };
  }
};

for(let value of range) {
  alert(value); // 1, 2, 3, 4, 5
}
```

##### 비동기적 이터러블 (객체)
1. `[Symbol.iterator]`
    - → `[Symbol.asyncIterator]`
2. `next()` <sub>(메서드)</sub>
    - 프라미스 반환
3. `for (let i of it)`
    - → `for await (let i of it)`

익숙한 예시인 이터러블 객체 range를 토대로, 일초마다 비동기적으로 값을 반환하는 이터러블 객체를 만들어보겠습니다.
